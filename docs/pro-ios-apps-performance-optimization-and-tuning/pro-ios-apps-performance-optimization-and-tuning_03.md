# 前言

本书旨在帮助你精进 iOS 开发在特定领域——性能优化——的技能。本书面向已具备 iOS 开发基础技能，并希望为用户打造最佳应用的开发者。

受应用程序性能之道的启发，我花费了大量时间在 Web 和智能手机等不同平台上进行实践、学习和分享性能优化相关的知识。我热衷于与他人探讨这一话题。在花时间浏览诸如 Stack Overflow 和 Apple Developer Forum 等论坛及 iOS 社区后，我很快意识到，大多数 iOS 开发者对于如何提升应用程序性能都存在着相同的疑问。我认为，将所有这些常见问题汇编成一本结构清晰、行文流畅的书籍会很有帮助，这样人们就能轻松掌握 iOS 性能优化问题的全貌。这促使我撰写本书，并尽力涵盖了开发者遇到的最常见的问题和错误。

此外，我还观察并记录了 iOS、Android 和 Windows Phone 之间存在的许多相似问题，并将其收录在自己的笔记中。最后一章便是基于这些笔记撰写而成，我认为这一章对于任何想要在这三个平台工作或从一个平台转向另一个平台的开发者来说都极具价值。

在应对性能瓶颈时，多角度审视问题，并在应用性能与解决方案的实现难度之间取得平衡，不失为良策。有些问题很微妙，除非开发者事先了解解决方案，否则很容易犯错。我希望本书能帮助你避免这些错误，将时间投入到优化应用上，从而为你的用户创造更佳的体验。
