# 引言

欢迎阅读《*Android Studio IDE 快速参考手册*》。我编写这本书的目的是为 Android Studio 的显著功能提供一个便捷的参考。

本书共包含 13 个短章节，每章简要介绍 Android Studio 的一些功能。虽然本书侧重于 IDE，但许多章节实际上也涉及 Android 编程，因此你将会看到一些代码示例。

## 本书面向的读者

本书面向那些需要快速了解如何在 Android Studio IDE 中执行常见任务的经验丰富的程序员。代码示例使用 Java 编写，因此适合用 Java 构建 Android 应用的开发者。本书也可能适合那些虽然对 Android Studio 不熟悉，但并非 Android 开发新手的人——比如想要尝试 Android Studio 的 Eclipse 用户。
