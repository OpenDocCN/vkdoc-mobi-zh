# 第 7 章：无障碍测试

本章揭示了如何使用 Espresso for Android 测试应用程序无障碍功能这一主题。它提高了人们对无障碍测试重要性的认识，并概述了可用于测试应用程序无障碍功能的手动工具。

### 第 8 章：Espresso 与 UI Automator：完美搭档

本章解释了 Android 最强大的测试自动化设置之一，它结合了 Espresso 测试框架和 UI Automator 测试工具。示例展示了如何在执行 Espresso 测试时测试通知或操作第三方应用。

### 第 9 章：处理运行时系统操作和权限

本章解释了处理权限请求对话框等系统操作的不同方法，并描述了以编程方式更改 Android 模拟器系统语言的可行解决方案。
