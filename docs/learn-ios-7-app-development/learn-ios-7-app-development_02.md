# 本书适合哪些读者？

本书适合所有希望学习基本工具与技术，为搭载 iOS 操作系统的 Apple 产品（包括 iPad、iPhone 和 iPod Touch）打造精彩动态应用的人。在撰写本书时，这些设备均涵盖在内。

本书假定你是 iOS 应用开发的新手，且编程经验有限。如果你正在学习 `Objective‑C`，那再好不过。如果你熟悉 `C`、`Java`、`C#` 或 `C++`，理解本书内容也不会有太大困难，此外还有一章 `Objective‑C` 入门指南供你阅读。如果你对计算机编程毫无经验，我建议先阅读一本基础的 `Objective‑C` 编程书籍——比如 Gary Bennett、Mitchell Fisher 和 Brad Lees 合著的 `《Objective-C 零基础入门》`——作为起步或同步学习。所有 iOS 应用开发均使用 `Objective‑C` 语言。

本书将阐述 iOS 应用设计、构建和部署的基本原理。你将养成一些良好的设计习惯，掌握核心编程技能，并熟悉用于创建应用的开发工具。

本书并非针对某一技术的深度专著。它的设计初衷是通过让你抢先一步构建应用来激发想象力，这些应用将利用多种设备功能，例如在地图上定位、使用加速计、通过内置摄像头拍照、与其他设备实时通信、接入社交网络，以及将信息存储于云端。在此基础上，你可以超越这些示例，去创造下一款伟大的 iOS 应用！
