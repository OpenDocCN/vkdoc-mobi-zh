# 如何使用本书

本书秉持“边探索边学习”的理念。有些章节会带你一步步创建一个使用摄像头或播放音乐的 iOS 应用。这些章节可能会略过许多细节。在章节之间，你会看到关于基础软件开发技能的章节。其中有关于良好软件设计、内存管理和 `Objective-C` 编程语言的章节。

因此，与传统的一开始就学习所有基础技能，然后再用这些技能构建应用的顺序不同，本书一开始就着手构建应用，然后再探讨其中的具体细节。

你可以按任意顺序阅读各章节，根据需要跳过或回头重读。如果你在早期章节中想深入了解对象，可以跳到后面阅读关于对象的章节。如果你已经了解了 `Objective‑C` 的内存管理，遇到相关章节时直接跳过即可。将本书视为一个可供学习的技能集合，而非必须按顺序完成的系列课程。

以下是各章节的预览：

- **工欲善其事**：指导你如何下载和安装 Xcode 开发工具。你需要它们。
- **应用初登场**：带你走一遍创建 iOS 应用的核心步骤——无需编程。
- **网页旋涡**：创建一个利用 iOS 内置网页浏览器强大功能的应用。
- **事件来临**：讨论事件（触摸、手势、运动）如何从设备进入你的应用，以及如何利用它们使应用对用户做出响应。
- **表格礼仪**：向你展示数据如何在应用中显示以及如何被编辑。
- **对象一课**：直截了当地讲解对象与面向对象编程。
- **笑一个！**：教你如何将摄像头和照片库集成到你的应用中。
- **模范公民**：解释被软件工程师称为“模型-视图-控制器”的神奇咒语。
- **甜美音乐**：通过教你如何向应用添加音乐和 iTunes 来为你的应用增色。
- **再谈工具**：简要介绍 Cocoa Touch 框架中可用的工具（视图、控件和对象）。这样当你需要某个工具时，就知道如何找到它。
- **画幅画给我**：教你如何创建自定义视图，释放绘制 iOS 应用中几乎任何内容的能力。
- **去而复返**：阐述应用导航的基本原理：用户如何从一个屏幕切换到另一个屏幕，以及如何返回。
- **社交网络**：让你的应用接入 Facebook、Twitter、电子邮件等。
- **极客网络**：演示如何使用 Game Kit 在多个 iOS 设备之间进行实时通信。
- **你若打造**：解释 Interface Builder 背后的部分魔法。
- **动感应用**：利用加速计和指南针让你的应用动起来。
- **你在哪里？**：字面意思——为你绘制一张地图。
- **记住我吗？**：展示如何设置和保存用户偏好，以及如何通过 iCloud 与其他 iOS 设备共享。
- **文档储存**：解释应用文档如何存储、交换和同步。
- **Objective 速成**：`Objective-C` 编程语言的速成课程。
- **房间里的大象**：探讨内存管理的技术、技巧和陷阱。
- **您会多国语言吗？**：教你如何创建支持多种语言的应用。
- **再快一点！**：讲解性能分析与优化的基础知识，让你的应用跑得像兔子一样快。
- **好事成双**：通过多任务处理提升你的应用能力，展示你的应用如何同时做两件、三件甚至四件事。
