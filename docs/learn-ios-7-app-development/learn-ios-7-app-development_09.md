# 获取项目文件

现在正是下载本书项目文件的好时机。本书中使用了大量项目。许多项目都可以通过遵循每章的步骤重新创建，我鼓励你尽可能这样做，这样你就能体会到从头开始构建应用的感觉。然而，有些项目并未解释每一个细节，还有一些项目包含了无法在印刷品中重现的二进制资源（图片和声音文件）。

前往本书在[`http://www.apress.com`](http://www.apress.com/)上的页面（你可以通过书名、ISBN 或作者姓名进行搜索）。在书的描述下方，你会看到一些文件夹标签，其中一个标有“源代码/下载”。点击该标签。现在找到下载本书项目的链接。点击该链接，一个名为`Learn iOS Development Projects.zip`的文件将下载到你的硬盘上。

在你的“下载”文件夹（或浏览器保存的位置）中找到`Learn iOS Development Projects.zip`文件。双击该文件解压内容，你会得到一个名为`Learn iOS Development Projects`的文件夹。将这个文件夹移动到任何你喜欢的位置。
