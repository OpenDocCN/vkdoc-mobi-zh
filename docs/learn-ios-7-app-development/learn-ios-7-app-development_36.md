# 10. 有视图了？

## 摘要

你现在已经积累了丰富经验：向设计添加视图对象、排列它们、将它们连接到输出口和操作、以及自定义它们。你已经使用按钮、标签、图像视图、一些文本字段和偶尔出现的工具栏创建了 iOS 应用。虽然你可能还没有渴望其他类型的视图对象，但还有更多可用对象。`Cocoa Touch` 框架提供了各种开关、切换器、滑块、专用按钮、选择器、指示器、小零件和小装置，你可以用来构建应用。如果这还不够，许多这些对象还可以通过你尚未探索的方式进行自定义。在本章中，你将学习：

- 下载示例项目
- 按钮视图
- 开关、滑块和步进器
- 指示器
- 标签、文本字段和文本视图
- 选择器
- 分组表格视图
- 滚动视图

任何花时间搭建乐高模型、拼装玩具或制造实验飞机的人都会知道一件事：你想象可能性的能力，直接与你对可用零件的了解程度相关。为此，我邀请你参加一次 iOS 视图对象的导览之旅。
