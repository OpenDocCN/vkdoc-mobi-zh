# 在搜索窗口中添加谓词编辑器

现在，我们已让 FoundQuotes 控制器基于由应用委托“拥有”的谓词内容来获取值。下一步是在搜索窗口中添加一个 `NSPredicateEditor`，并配置它以允许用户编辑谓词。

在 Xcode 中，调出引用查找窗口。将整个窗口拉高一些，然后将现有的表格视图和文本视图拖到底部。接着，在对象库中找到 `NSPredicateEditor`，将其拖入窗口顶部的空白区域，并调整大小使其填满可用空间。图 10-7 展示了这一构思。
