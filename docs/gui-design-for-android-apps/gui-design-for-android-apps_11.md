# 索引

## 关于首席项目编辑

![9781484203835_unFM-01.jpg](img/9781484203835_unFM-01.jpg)

**瑞安·科恩**是负责领导创建此英特尔学习资源的国际内容贡献者团队的责任编辑；他也是一名 Android 爱好者，毕业于波特兰州立大学。自 2011 年从苹果 iOS 平台转向 Android 以来，他一直关注 Android 的发展。在不撰写 Android 相关内容时，他会花时间研究 Android 世界中的一切新事物。

## 关于首席贡献作者

![9781484203835_unFM-02.jpg](img/9781484203835_unFM-02.jpg)

**王涛**于 1993 年作为博士生来到美国，就读于俄勒冈州立大学。自 2002 年起，他一直是英特尔公司的软件工程师。王涛自 2008 年开始撰写关于 Android 的博客和文章；自 2011 年起，他担任英特尔 Android 开发者专区（英特尔所有 Android 相关内容的开发者资源）的技术资料经理。在业余时间，王涛还经营着一家名为 E-k12 的移动应用/客户端教育初创公司。他密切关注应用程序开发的最新进展，以及针对移动设备和 x86 平台上的 Android 进行测试/调试/性能优化。王涛精通多种平台，包括 Android SDK 和 NDK；英特尔 Android 工具；游戏引擎如 Cocos2D-x、AndEngine 和 libgdx；OpenGL ES；RenderScript 以及 Android Runtime。他的其他兴趣领域包括移动互联网技术，如在线内容管理、基于云的移动技术、嵌入式设备、机器人技术以及移动学习。

## 关于技术审校

![9781484203835_unFM-03.jpg](img/9781484203835_unFM-03.jpg)

**泽维尔·阿拉德**是法国巴黎英特尔软件与服务集团的首席开发者布道师。自 2012 年基于英特尔平台的首款 Android 智能手机公开发布以来，他一直致力于帮助 Android 开发者们更好地支持由英特尔制造或支持的新硬件和技术。

## 引言

这本迷你书汇集了从《基于英特尔平台的 Android 应用程序开发》一书中抽取的四个章节，旨在向开发者介绍如何为其 Android 应用程序创建出色的用户界面。这些章节涵盖的主题包括：为桌面系统和嵌入式系统开发用户界面的差异，以及如何为触摸屏优化应用程序的用户界面。
