# 文件传输

iPad 早已具备文件传输功能，如今其他 iOS 设备也能在 iTunes 中实现电脑与 iOS 设备之间的文件传输。

## 小结

`iOS 4`中蕴藏着大量新机遇，Facebook 和 Twitter 各自的 API 也不例外。用户群体极其庞大：5 亿 Facebook 用户和 1.3 亿 Twitter 用户——而且两者都在持续增长。无论你的 iOS 应用具备何种功能，加入社交元素后，它都可能变得更实用、更具吸引力。

本章中你能带走的最关键建议是：在完成原型设计前，花足够的时间去使用这些服务。这两项服务——尤其是 Facebook——包含大量对象、属性和交互，其功能容易令人困惑。了解用户期望这些资源以何种方式被使用，将帮助你设计出运行可靠且一致的应用。

读完本书后，你将清楚知道该为应用添加什么以及如何构建。现在，请翻页，开始行动吧！
