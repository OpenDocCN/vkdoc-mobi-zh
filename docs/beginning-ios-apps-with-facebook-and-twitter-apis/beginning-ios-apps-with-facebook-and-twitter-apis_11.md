# 前言

Facebook 和 Twitter 或许是对我们的日常沟通至关重要的少数平台，以至于对一些用户而言，它们可以取代苹果自家的通信应用、短信应用和电话。几年后，有些人可能会在 iOS 体验的大部分时间里都沉浸在这些平台之中。

对我们这些第三方开发者来说幸运的是，这两家公司都发展得如此迅速，以至于它们几乎无暇探索和优化这些平台的每一种可能用途。（它们还要为盈利这个小问题而烦恼。）

当 Twitter 和 Facebook 的员工忙于完善他们的产品、隐私政策、API 和商业计划时，那些更小、更灵活的开发者正迎来巨大机遇——走出去，看看人们*下一步*想从在线社交体验中获得什么。作为一名独立开发者，你有能力在 Facebook 和 Twitter 的用户中找到自己的利基市场——也许是一个非常大的利基市场——并打造一款既新颖实用，又熟悉直观的工具。

我们希望这本书能助你实现这一点。
