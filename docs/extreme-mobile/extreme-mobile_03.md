# 引言

本书旨在演示设计并开发可复用、可配置、可品牌化且可本地化的 iOS 移动软件的实际技术。这将通过利用 iOS 框架高效、系统地构建商业级移动应用来实现。

设计和开发此类可复用、模块化的解决方案具有诸多优势。整个工程组织使用的通用功能可在多个应用中复用，并按需扩展，从而减少开发时间并提高一致性。通过定价层级和应用内购买提供功能集，可增强营收机会。以最小的投入即可动态添加新功能。集中化的资源与样式让品牌重塑变得简单直接。此外，无需进行大量的重新工程即可集成国际化与本地化，使应用能够快速覆盖全球用户。

本书面向具备 `Xcode`、`iOS SDK`、`Swift`、`Storyboards` 及 `SwiftUI` 基础知识的**中级软件开发者**。

本书首先介绍使用 iOS 框架开发可复用软件的核心概念，然后以清晰、循序渐进的方式逐步构建复杂性。读者将探索 `Swift 协议`、`模型-视图-视图模型 (MVVM)` 设计模式、本地与远程服务请求、`依赖注入`、`Combine`、`SwiftData`、`Swift 测试`、`Storyboards` 及 `SwiftUI`。书中还涵盖了将 iOS 框架动态集成到应用中的技术。阅读本书后，读者将使用相同的框架开发出两个不同的应用程序，从而展示无需修改底层代码即可实现的强大且灵活的可复用、模块化 iOS 软件。
