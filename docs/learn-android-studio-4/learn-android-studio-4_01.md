# 引言

欢迎阅读 *《学习 Android Studio 4》* 。本书将帮助你开启与绿色小机器人（Android）的编程之旅。你已经购买了这本书（非常感谢你），因此无需再向你说明移动平台编程为软件开发人员提供了许多机会。

本书面向初级的 Android 程序员，但并非完全不懂编程的新手。本书假设你之前有使用 CFOL（C 语言家族，例如 C、C++、Java、C#、JavaScript）中任何一种语言的编程经验。理想情况下，你已经是 Java 程序员，正准备涉足 Android 领域；如果不是，也无需担心。附录中涵盖了 Java 编程的基础知识，在你尝试熟悉该语言时可以随时参考。

本书涵盖两个方面：Android 编程的基础知识和 Android Studio 4 的使用。Android 编程概念和 IDE 的使用将通过图形和代码走读相结合的方式进行讲解，书中包含大量的此类内容。

## 章节概览

**第 1 章：Android 概述** — 本章介绍 Android。内容涉及一些 Android 的历史及其操作系统的技术构成。

**第 2 章：Android Studio** — 如果你尚未设置好 Android 环境，请不要跳过本章；无论你使用的是 macOS、Windows 还是 Linux，本章都会引导你完成 Android Studio 的设置。本章还介绍了 IDE 的基本组成部分。

**第 3 章：项目基础** — 本章介绍 Android 项目的概念和机制。它带你逐步完成创建项目以及在 AVD（Android 虚拟设备）中运行项目的过程。

**第 4 章：Android Studio IDE** — Android Studio 是一个功能完备的 IDE；它拥有众多特性和部件。本章向你介绍 Android Studio 中最常用的工具和窗口。

**第 5 章：Android 应用概述** — 一个 Android 项目由什么组成？什么是组件？什么是 Intent？这些是本章将要解答的一些问题。你将了解 Android 应用与桌面应用有多么不同。

**第 6 章：Activity 与布局** — 我们开始进入 UI 构建的基础知识。Activity 是用户看到你应用的主要方式。我们将学习如何构建这些常用 UI 元素及其他元素。

**第 7 章：事件处理** — 处理用户操作是 Android 编程中非常常见的任务。本章将带你了解监听器对象的基础知识，如何创建它们，以及如何将它们绑定到 View 元素（如按钮）上。

**第 8 章：Intent** — Intent 是 Android 独有的特性。了解这种消息传递机制在 Android 中如何工作，以及它如何将所有其他 Android 组件粘合在一起。

**第 9 章：Fragment** — Fragment 是一种细粒度地构建屏幕的方式。本章讲解 Fragment 的基本概念。

**第 10 章：导航** — 导航组件是相当新的内容。它们是 Jetpack 的一部分。本章向你介绍构建多屏幕应用的更现代化方法。

**第 11 章：在后台运行** — 当你开始构建非平凡的应用时，你需要从 I/O 源读取或写入数据、从网络获取数据等。这些活动需要时间，并且需要在后台运行。本章正是关于这方面的内容。

**第 12 章：调试** — 你经常会犯编码错误。本章向你介绍你可能遇到的错误类型，以及如何使用 Android Studio 的调试功能来解决它们。

**第 13 章：测试** — 在某个时候，你必须在发布应用之前测试你的代码。本章向你介绍可以对应用进行的多种测试。更重要的是，它向你介绍了单元测试和 Espresso 测试。

**第 14 章：文件操作** — 你需要保存到文本文件或从中读取；本章讲解 Android 中文件输入和输出的基础知识。

**第 15 章：BroadcastReceiver** — Android 的基础组件之一是 `BroadcastReceiver`；该组件允许你通过采用发布-订阅模式来构建解耦的应用。

**第 16 章：Jetpack、LiveData、ViewModel 和 Room** — 架构组件中的更多好东西。本章讲解如何构建具有生命周期感知能力的组件，以及如何使用 Room 的基础知识。

**第 17 章：分发应用** — 当你准备好分发应用时，你需要对其进行签名，并将其列在像 Google Play 这样的市场中。本章将带你逐步了解如何操作。

**第 18 章：快速技巧** — 更多 Android Studio 的技巧。

**附录** — 附录快速介绍了 Java 语言。它涉及你开始 Android 编程所需的一些基本语言概念。
