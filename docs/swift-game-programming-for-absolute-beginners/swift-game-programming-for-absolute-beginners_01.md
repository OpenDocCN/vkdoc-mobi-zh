# 引言

Swift，这一取代 Objective C 的新编程语言，由苹果公司于 2014 年 6 月 2 日发布。此后，该语言经历了多次更新。在撰写本书时，其版本为 2.0。Swift 是一种结构非常良好的编程语言，它借鉴了 C#、Haskell 和 Python 等现有现代语言的许多特性，并自身引入了一些新功能。

如果你想为 iOS、watchOS 或 OS X 平台进行开发，就必须学习 Swift。本书将深入教授你 Swift 语言；同时，你还将学习如何为 iOS 平台开发游戏。由于本书以游戏开发为背景，你将了解到许多新特性对于设计游戏软件非常有用。本书基于 SpriteKit 框架，并向你展示如何开发能够准备发布到 App Store、供所有人游玩（并且有望在这个过程中为你赚取丰厚收入！）的应用。

在阅读本书的过程中，你将开发四款游戏。每款游戏都使用了专业设计的素材。每款游戏都建立在开发前一款游戏所获得的知识基础上，从简单的瞄准射击游戏和基于物理的互动游戏，到解谜游戏，最后是一款包含动画、智能敌人等内容的完整平台游戏。正如你将看到的，制作游戏可以与玩游戏一样有趣。

尽管本书在游戏开发的背景下教授 Swift，但阅读本书后，你将能够运用所学技能为苹果设备开发任何类型的应用。

## 本书读者对象

本书适合所有对学习如何创建游戏感兴趣的人。如果你没有（Swift）编程经验，也不必担心。本书将教授你需要了解的所有知识。如果你已经知道如何编程，那么本书仍然会令你感兴趣。我将向你展示如何使用 Swift 2.0 语言和 SpriteKit 框架的最新特性来开发游戏。本书为 iOS 平台开发了四款不同的游戏。这些游戏的代码经过精心开发，考虑了代码组织的恰当方式，使其清晰、健壮且易于扩展。

## 本书结构

本书每一章都有其专属的示例程序集。你可以在本书的网站上找到这些示例。我将基于这些示例解释所有 Swift 编程概念。在本书中，你将找到许多游戏设计和开发技巧，例如如何以及何时使用菜单或教程、与美工团队合作、以及发布和营销方面的建议。

本书大致分为五个部分。以下是各部分内容的概述。

### [第一部分](http://dx.doi.org/10.1007/978-1-4842-0650-8_Part1)

本部分概述了 Swift 编程语言，并介绍了其主要特性。我将介绍最重要的游戏编程结构——游戏循环。你将了解如何使用 `SpriteKit`（苹果为开发 Swift 游戏提供的框架）。你将学习用于表示游戏世界的变量和数据结构，并了解如何将精灵和音效等游戏资源包含到你的程序中。



## [第二部分](http://dx.doi.org/10.1007/978-1-4842-0650-8_Part2)

本部分聚焦于第一个游戏：Painter 游戏。游戏的目标是收集三种不同颜色的颜料：红色、绿色和蓝色。颜料从天空中以罐子形式落下，罐子由气球托浮在空中，你必须确保每个罐子在从屏幕底部落下之前拥有正确的颜色。我将展示如何通过读取鼠标、键盘或触摸输入来响应玩家的操作。我将介绍 `class` 作为对象的蓝图（也可称为该类的实例）。你将学习构造器方法，这些方法负责创建它们所属类的实例。

你将学习如何编写自己的方法、属性和类，以及如何使用这些编程概念来设计不同的游戏对象类。你将看到游戏对象之间应如何交互。作为这种交互的示例，你将了解如何处理游戏对象之间的基本碰撞。你将学习 Swift 中继承的实现方式，以便能够分层构建游戏对象类。你还将了解多态的概念，它允许你自动调用方法的正确版本。你将通过添加一些额外功能来完成 Painter 游戏，例如运动效果、声音、音乐以及维护和显示分数。

## [第三部分](http://dx.doi.org/10.1007/978-1-4842-0650-8_Part3)

你在本书中开发的第二个游戏是 Tut's Tomb：一个需要收集下落宝藏的游戏。这个游戏展示了如何使用 SpriteKit 框架中的物理引擎。你将学习编写更高级的输入处理，例如在屏幕上拖动对象以及同时处理多点触摸。你还将学习如何使用自定义字体让游戏看起来更美观。

## [第四部分](http://dx.doi.org/10.1007/978-1-4842-0650-8_Part4)

本部分介绍 Penguin Pairs 游戏，这是一款益智游戏，目标是将相同颜色的企鹅配对。在游戏的不同关卡中，你将引入新的游戏元素来保持游戏趣味性。例如，有一种特殊的企鹅可以与任何其他企鹅配对，企鹅可能会被困在洞里，你还可以在棋盘上放置食企鹅鲨鱼。

我将向你展示如何处理游戏中的结构和布局，例如游戏网格或一行按钮。你将创建各种用于菜单的有用 GUI 元素，例如开/关按钮和滑块按钮。你将学习如何处理不同游戏状态（如菜单、标题屏幕等）的类设计。你将看到不同状态如何成为游戏循环的一部分，以及如何在它们之间切换。最后，你将学习从文件加载关卡，以及如何存储玩家在游戏中的进度并在游戏再次启动时调用这些信息。

## [第五部分](http://dx.doi.org/10.1007/978-1-4842-0650-8_Part5)

你在本书中开发的最后一个游戏是一个名为 Tick Tick 的平台游戏。我将向你展示如何创建一个基于瓦片的游戏世界，类似于你在开发前一个游戏时所学到的。你将看到如何添加动画，例如奔跑的角色。你将开发专门为平台游戏量身定制的物理引擎，其中包含跳跃、下落以及处理角色与游戏中其他对象碰撞的机制。你还将为游戏中的敌人添加一些基本智能。这样，玩家将拥有不同的游戏玩法选项，并且必须制定不同的策略才能完成每个关卡。

## 工具

为了使用 Swift 开发 iOS 游戏，你需要一台安装了 Xcode 的 Mac。Xcode 是 Apple 提供的开发环境，用于创建 OS X、iOS 或 watchOS 应用程序。为了能够运行示例，你需要一个支持 Swift 2.0 或更高版本的 Xcode 版本（在本书撰写时，该版本为 7）。你需要注册成为 Apple 开发者才能获取 Xcode，不过只有在你想在 iPad 或 iPhone 上运行应用程序并在 App Store 发布应用时，才需要加入付费计划。Xcode 和 iOS 模拟器对于任何 Apple 开发者都是免费使用的。

## 示例游戏

我提供了大量示例，展示了 Swift 游戏编程的各个方面。除了代码外，你还可以获得专业开发的、具有两种不同分辨率的游戏资源。示例合集包含在一个 ZIP 文件中，你可以在本书的信息页面（[`www.apress.com/9781484206515`](http://www.apress.com/9781484206515)）的“源代码/下载”选项卡下找到它。该选项卡位于页面的“相关标题”部分下方。

下载 ZIP 文件后，请将其解压到某处。当你查看解压文件的文件夹时，你会看到许多不同的文件夹。书中的每一章都有自己的文件夹。例如，如果你想运行 Penguin Pairs 游戏的最终版本，请转到第 21 章对应的文件夹，然后打开 `PenguinPairsFinal` Xcode 项目（这是一个以 `.xcodeproj` 结尾的文件）。在 Xcode 打开项目后，按 `Command+R` 或点击窗口左上角的“播放”按钮来运行游戏。

如你所见，这个特定示例有许多不同的相关文件。一个更简单的示例可以查看第 1 章对应的文件夹，你会在那里找到两个非常基础的 Swift 应用程序示例：一个 OS X 控制台应用程序和一个 iOS 游戏应用。
