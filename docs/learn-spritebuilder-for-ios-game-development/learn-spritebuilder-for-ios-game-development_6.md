# 致谢

没有 Cocos2D 社区，本书不可能问世。向你们致敬！请继续创造疯狂的作品！

没有 Apportable 公司辛勤工作的人们（包括像我一样散落世界各地的成员），本书也不可能达到如此质量。在此，我要特别感谢（排名不分先后）维克多·利德霍尔特、科林·杰克逊、克里斯蒂娜·凯利、尼基·韦伯、斯科特·莱姆克、马丁·沃尔什、本杰明·恩茨、拉尔斯·比尔克莫斯和塞耶·安德鲁斯，感谢他们的合作与支持，并容忍我在翻译中屡屡迷失方向。

当然，我不会忘记那些与我直接合作完成本书的人。衷心感谢凯文·沃尔特和马克·鲍尔斯让我保持信息更新且忙碌，感谢彼得·阿劳和米歇尔·洛曼签约我，以及利兰·朗和马丁·沃尔什提供技术反馈。

还有马库斯，我与他在同一间办公室合作：感谢他为我提供了亟需的视角转换，一张出人意料地设计精良、适合键盘鼠标和方向盘操控游戏的扶手椅式赌桌，一个虚拟现实头盔（自制的？我们也希望！），以及偶尔的合作游戏和电影。

我想感谢所有我可能在此忘记提及的人。请放心，你们未在此被点名只是我长期记忆力的缺失。此刻，我或许正后悔未能向你们单独表达谢意。
