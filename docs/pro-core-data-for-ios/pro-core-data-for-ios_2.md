## 引言

为苹果 iOS 平台开发应用的兴趣持续高涨，每天都有更多优秀应用出现在苹果的 App Store 中。当像你一样的人们加入应用创作的行列时，他们通常会发现自己的应用必须将数据存储在 iOS 设备上才能发挥作用。因此，《iOS 专业核心数据》应运而生，这本书专为那些已掌握 iOS 开发基础知识、准备深入探索数据存储相关主题、从而将应用从“还不错”提升至“卓越”的开发者而写。苹果用于数据存储与检索的技术——Core Data，既容易上手又难以精通。本书涵盖了从入门到高级的方方面面，带你从简单内容起步，逐步深入。阅读每个主题，理解其含义，并将其融入到你的 Core Data 应用中。

### 为何推出第二版？

自《iOS 专业核心数据》第一版出版以来，苹果发布了 Xcode 4，这是对其编程工具的一次重大革新。所有内容都发生了变化或调整，因此本书第一版（基于 Xcode 3）中的描述和教程已不再适用。所有描述和截图都已更新为新的界面。

然而，我们并未止步于仅为 Xcode 4 更新本书。我们将对`NSFetchedResultsController`的讨论单独拆分为一章，对其进行了更深入的阐述和更全面的覆盖。我们对数据迁移这一棘手主题进行了更深入的探讨。根据 Brian Kohl 的反馈，我们采用了新的方法来编写数据加密部分。我们也回应了通过书评和电子邮件收到的反馈。我们相信，无论是新读者还是已经阅读过第一版的读者，都能从这一版中获益。

### 你需要准备什么

要跟随本书学习，你需要一台运行 Snow Leopard 或 Lion 系统的 Intel Mac 电脑，以及 Xcode 4（可从 Mac App Store 或`developer.apple.com`获取，后者需注册苹果开发者账号）。如果你对 Objective-C、Cocoa Touch 和 iOS 开发有至少基本的了解，学习效果会更好。

### 你将学到什么

本书首先为理解 Core Data 是什么及其工作原理奠定清晰的基础，然后逐步引导你如何从这个强大的框架中获得所需的结果。你将了解 Core Data 的各个组件及其交互方式、如何设计数据模型、如何过滤结果、如何调优性能、如何跨数据模型版本迁移数据，以及许多其他相关主题，这些都将使你的应用脱颖而出。

本书结合理论与实践来传授知识。虽然你可以把书拿到躺椅上从头读到尾，但如果你坐在电脑前，边输入代码边理解书中讲解的内容，学习效果会更佳。我们也希望，在你阅读本书并实践其中的代码后，能把它当作一本随时可查阅的参考书，经常翻阅以寻找答案和澄清疑惑。

### 本书的组织结构

我们尽量将材料按照从入门主题到高级主题的顺序编排，至少从整体上而言，随着书的推进，这种递进关系是成立的。各主题之间往往相互关联、层层递进，因此从头到尾通读本书，比跳跃式阅读会让你收获更多。如果你在寻找某个特定主题的指导——比如版本管理与数据迁移，或者性能与内存使用调优——可以直接跳到对应的章节。大多数章节都聚焦于某个单一主题，主题由该章节的标题指明。最后一章则涵盖了一系列在其他地方不便归类的进阶主题。
