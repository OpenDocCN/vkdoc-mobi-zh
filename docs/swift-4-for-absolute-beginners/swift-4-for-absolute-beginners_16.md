# 5. 使用 Swift 进行面向对象编程

在过去的 16 年里，编程界一直聚焦于面向对象编程（OOP）的开发范式。大多数现代开发环境和语言都实现了 OOP。简单来说，OOP 构成了你今天所开发的一切的基础。

你可能会问，既然 OOP 是当今主要的开发风格，为什么我们等到第 5 章才介绍如何使用 Swift 进行 OOP？答案很简单：对于新开发者来说，它不是一个容易掌握的概念。本章将详细探讨 OOP 的不同方面及其对你开发工作的影响。

将 OOP 正确地应用到你的应用程序中需要一些前期规划，但在整个项目生命周期中，你会节省大量时间。OOP 已经改变了开发的方式。在本章中，你将学习什么是 OOP。本书第一章曾初步讨论过 OOP，但本章将更深入地讲解。你将重新审视什么是对象，以及它们如何与现实世界中的物理对象相关联。你将探究什么是类，以及它们与对象的关系。你还将学习在规划类时需要采取的步骤，以及可以用来完成这些步骤的一些可视化工具。当你阅读完本章并完成了练习后，你将更好地理解什么是 OOP，以及为什么它对你作为开发者是必要的。

起初，对象和面向对象编程可能看起来难以理解，但希望随着你对本章内容的深入学习，它们会开始变得有意义。
