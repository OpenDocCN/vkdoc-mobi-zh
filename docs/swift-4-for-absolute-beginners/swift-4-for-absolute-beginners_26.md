# 使用数组

`Array`类与其他集合类型一样，允许程序员管理一组对象。数组是一种有序集合，这意味着对象会按照特定顺序存入数组，并按照相同顺序取出。

**注** – 有一些用于处理数组的方法，可以让你更改对象的顺序，或在数组的特定位置添加对象。

`Array`类允许通过对象在数组中的索引来检索该对象。索引是对象在数组中所处的数值位置。例如，如果数组中有三个元素，则可以使用索引 0 到 2 来引用这些对象。与 Swift 及其他编程语言中的大多数情况一样，索引从 0 开始，而不是从 1 开始。请参见代码清单 8-1。

```
1    var myArray: [String] = ["One", "Two", "Three"]
2    print(myArray[0])
3    print(myArray[1])
4    print(myArray[2])
代码清单 8-1.
访问数组中的对象
```

如你所见，数组中的对象可以通过其索引来检索。索引从 0 开始，且不能超过数组大小减 1。你可以通过访问 `Array` 对象的 `count` 属性轻松计算出数组的大小，如下所示：

```
var entries = myArray.count
```

实际上，所有集合类型（包括 `Array` 和 `Dictionary`）都包含一个 `count` 属性。

向数组末尾添加元素很简单。你只需在数组上调用 `append` 方法即可。请参见代码清单 8-2。

```
1    var myArray: [String] = ["One", "Two", "Three"]
2    myArray.append("Four")
3    myArray.append("Five")
4    myArray.append("Six")
代码清单 8-2.
向数组添加对象
```

Swift 提供了多种不同的方法来向数组添加元素。如果你想向数组添加多个对象，可以使用标准的 `+=`（通常称为加等于）运算符。代码清单 8-3 创建了一个数组，然后在第 2 行向该数组添加了三个 `String` 对象。注意新值是用方括号括起来的，而不是圆括号。

```
1    var myArray: [String] = ["One", "Two", "Three"]
2    myArray += ["Four", "Five", "Six"]
代码清单 8-3.
向数组添加多个对象
```

如前所述，数组实际上是有序的。数组中对象的顺序很重要。有时你可能需要在数组的某个特定位置添加元素。你可以使用 `insert(at:)` 方法来实现这一点，如代码清单 8-4 所示。

```
1    var myArray: [String] = ["Two", "Three"]
2    myArray.insert("One", at: 0)
代码清单 8-4.
在数组开头添加一个字符串
```

现在数组包含 One, Two, Three。

访问数组中的元素很简单。你可以使用标准的中括号来访问特定位置的对象。例如，`myArray[0]` 会返回数组中的第一个对象。如果你想遍历数组中的每一项，可以使用所谓的快速枚举或 For-In 循环。代码清单 8-5 是一个快速枚举的例子。

```
1    var myArray: [String] = ["One", "Two", "Three"]
2    for myString in myArray {
3         print(myString)
4    }
代码清单 8-5.
快速枚举
```

神奇之处在于代码清单 8-5 的第 2 行。你告诉 Swift 将 `myArray` 中的每个值赋给一个名为 `myString` 的新常量。然后你就可以对 `myString` 进行任意操作。在这个例子中，你只是打印它。它会遍历数组中的所有对象，而无需知道对象的总数。这是一种从数组中提取元素的快速有效的方法。

从数组中删除对象也很简单。你可以使用 `remove(at:)` 方法，如代码清单 8-6 所示。

```
1    var myArray: [String] = ["One", "Two", "Three"]
2    myArray.remove(at: 1)
3    for myString in myArray {
4         print(myString)
5    }
代码清单 8-6.
删除一个对象
```

代码清单 8-6 的输出将是 `One, Three`。这是因为你删除了索引为 1 的对象。记住，这是数组中的第二个对象，因为数组索引总是从 0 开始。

你已经看到了 Swift 在与数组交互方面是多么灵活。数组是你作为程序员将经常使用的强大集合。本节涵盖了数组的基础知识，但数组还能做更多的事情。
