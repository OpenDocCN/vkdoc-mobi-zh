# 摘要

警告框会在屏幕中央显示信息（`.alert`），而操作表则在屏幕底部显示信息（`.actionSheet`）。警告框和操作表都可以显示标题和消息，并至少包含一个用于关闭警告框/操作表的按钮。但警告框最多可显示两个按钮，而操作表则最多可显示三个。

警告框/操作表上的按钮只是用来关闭警告框/操作表，但你可以附加代码，使其在用户选择按钮时执行。一种方法是将代码直接写入 `.alert`/`.actionSheet` 修饰符中，但更好的方式是调用一个函数，并将该函数中的代码与 `.alert`/`.actionSheet` 修饰符分离开来。

当你需要向用户显示重要信息并立即获得反馈时，请使用警告框，例如确认用户是否要删除数据。当需要在屏幕底部显示信息时，请使用操作表。当你想将多个选项隐藏在用户界面之外，但又在用户需要时可供查看，请使用上下文菜单。
