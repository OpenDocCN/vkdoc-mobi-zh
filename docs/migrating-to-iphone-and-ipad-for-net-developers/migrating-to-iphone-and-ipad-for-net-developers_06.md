# 引言

自从 1981 年我偶然发现并开始阅读第一本计算机书籍以来，我就知道信息技术（IT）将成为我生命中重要的一部分。能够将自己的热爱发展成事业，我感到无比荣幸；而 IT 领域几乎每月都在变化，这意味着它永远不会变得乏味。至今我仍对移动电话在如此短的时间内取得的飞速进步感到惊叹；当 iPhone 问世时，我便意识到智能手机将在我们个人生活和职场生活的方方面面扮演关键角色。它们不仅是管理联系人列表或日历的有用工具，更已成为一种能完成各种事务的机制，比如购买电影票、通过社交媒体与朋友保持联系，以及玩游戏。

然而，那些不了解 Objective-C 或 Xcode 的人，为何就无法享受这个精彩的世界呢？尤其是当你已经熟悉并热爱微软的.NET 和 C#，却不想局限于微软移动智能手机平台的时候。事情并非如此绝对——有大量选择摆在你面前，包括：通过本书了解 C#与 Objective-C、Visual Studio 与 Xcode 4、.NET 框架与 iOS SDK 之间的异同；以及如何利用像 Mono 和 MonoTouch 这样的第三方解决方案。

本书正是为此而生。它将为你提供完成转型所需的一切信息，让你能够运用自身经验，自信地创建世界级的 iOS 应用程序。我甚至还会带你一步步完成将应用发布到苹果 App Store 的全过程。

希望你喜欢这本书，并从中获得机遇。感谢你的阅读。
