## 关于作者

![image](img/9781430246626_unFM-02.jpg) **杰扬特·瓦尔马**是 OZ Apps 公司的执行总监。该公司是一家澳大利亚本土的咨询公司，专注于协助本地及国际企业制定移动产品增强策略，并提供移动开发与培训方面的专业服务。他已在 Mac App Store 上推出了多款应用。

杰扬特拥有工商管理硕士（信息科技方向）学位，是一位拥有超过 20 年丰富且多元经验的 IT 资深人士，曾在多个国家工作，涉及的行业包括汽车与高等教育。秉承对教学的热情，他曾在詹姆斯·库克大学任教并编写课程大纲。

他的移动开发之旅始于 2000 年初的微软 Pocket PC（康柏 iPaq），并一直延续到今天的苹果 iOS。他对科技和游戏的痴迷，可以追溯到他在第一台电脑 ZX Spectrum 上编写游戏的岁月。

杰扬特是澳大利亚计算机协会（ACS）的活跃成员，并担任本地分会执行委员会成员。他是本地首个青年信息技术（YIT）分会的创始主席。他曾为苹果大学联盟（AUC）举办 iOS 开发研讨会，并热衷于指导和培养年轻开发者。他还通过博客分享开发相关主题，帮助其他开发者。

您可以在 Twitter 上关注他（@ozapps），或访问博客 [`howto.oz-apps.com`](http://howto.oz-apps.com) 和 [`reviewme.oz-apps.com`](http://reviewme.oz-apps.com)，以及 Lua 相关博客 [`dev.oz-apps.com`](http://dev.oz-apps.com) 和 [@LearnLua](http://@LearnLua)。

## 关于技术审校

**梅卡·什雷斯塔** 与 **尤詹·什雷斯塔** 是软件开发公司 M.Y. Developers 的联合创始人，该公司专注于为软件和游戏开发提供先进工具。

尤詹·什雷斯塔在德克萨斯大学奥斯汀分校完成了生物医学工程的学习，梅卡·什雷斯塔则在贝勒大学攻读神经科学。在参与大学里的多个项目后，他们决定合作开展一些创新项目，用于德克萨斯大学休斯顿医学院进行的自闭症谱系障碍研究。研究之后，他们开始制作简单的策略游戏。他们的第一个游戏项目是《太空征服》，这是一款具有真实物理引擎的冒险/动作游戏，能让玩家体验策略性游戏玩法。在游戏开发过程中，他们意识到，无论是独立开发者还是专业开发者，都需要能提升游戏开发质量的工具，这促使他们为 Corona SDK 制作并设计了多种工具——即 Corona Profiler 和 Corona Ultimote。他们还开发了一款广受好评的、基于 Lua 的集成开发环境，名为 Lua Glider，支持多种平台，包括 Corona SDK、Moai SDK、Gideros SDK 和 LÖVE 2D。

作为一个团队，他们的目标是开发和改进能提升软件和游戏开发体验的工具。

**阿蒂利姆·切廷：** 1996 年至 2001 年就读于中东技术大学计算机工程系。他曾先后在 Makrosam Software、Tübitak UEKAE 和 Meteksan System 等公司担任软件专家。2006 年至 2009 年，他在 Koç 集团旗下的 KaTron Defense, Space and Simulation Technologies 公司参与多个模拟项目，先后担任专家软件工程师、技术组长及三维软件组长。2010 年，他创立了 Gideros Mobile，并担任 Gideros Studio 的设计架构师和技术负责人。他是《计算机图形学》（ISBN: 975 347 104 0）一书的作者，也是土耳其信息学百科全书中“三维计算机图形学”和“动画”两个词条的合著者。
