# 第 1 章：Lua 入门

苹果公司一直在向开发者支付报酬，2012 年的数据显示，迄今为止总金额已达 50 亿美元。在过去，为开发者带来滚滚财源的是基于微软产品的桌面端，比如 Visual Basic，或是更早的数据库产品如 dBase 和 FoxPro。虽然这笔收入的主要份额流向了 EA、Chillingo、Gameloft、Rovio 甚至是迪士尼这样的大型公司，但许多独立开发者和较小的公司也在觊觎这块大蛋糕。谁知道哪个创意会为开发者摇身一变，成为印钞机呢？14 岁的罗伯特·内伊在学习编程时制作了游戏《泡泡球》，该游戏迅速走红，下载量超过 800 万次。谁也不知道下一个爆款游戏会是什么。
