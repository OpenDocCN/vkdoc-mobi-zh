# 讨论的主题

《Objective-C 程序员参考手册》提供了在 Objective-C 中构建可行解决方案所需的所有工具。从 C 语言的基本构建块出发，我将讨论如何利用面向对象编程以及结构化编程的知识，创建正确且高效的应用程序。

我广泛运用了精通 C++、Java、Perl 和 Python 等其他语言的开发者所掌握的概念。在编写本书时，我的目标是让这类读者能通过一条快节奏的路径来精通 Objective-C。

另一方面，本书以逻辑清晰且结构化的方式介绍 Objective-C，即使是没有编程经验的初学者也能轻松吸收使用该语言开发应用所需的最重要的主题和能力。

我们身处一个行业中，典型的书籍动辄数千页，而其中大部分篇幅都被那些专业开发者不甚关心的入门主题所占据。在这本书中，我力求提供一份更加简洁且重点突出的语言参考，让读者的阅读投入获得更高的价值。本书做到了以下几点：

- 快速带你了解 Objective-C 中的基本概念，例如接口和类的实现。
- 为 Foundation 框架提供一份简洁且便于查阅的参考，该框架在进行 Objective-C 编程时至关重要。
- 突出 Objective-C 与 Java 或 Python 等其他主流语言之间的关键差异。
- 展示如何使用高级概念（如容器类、选择器和键值接口）创建完整的应用程序。
- 提供编写 Cocoa 和 Cocoa Touch 应用程序的基础知识，它们是 Mac OS X 和 iOS 开发的标准框架。

## 你将学到什么

以下是你在本书中将学习的一些主题：

- Objective-C 语言的基本语法。
- 如何在 Objective-C 中创建类。
- 方法和消息传递机制如何工作，以简化你的代码并避免深层类层次结构。
- 用于存储和访问动态数据的键值系统，以及它如何与 Objective-C 接口结合以简化应用程序编程。
- Objective-C 中容器类（如数组和字典）的有效使用，包括其不可变和可变版本。
- Cocoa 编程的基本主题，可用于在 Mac OS X 和 iOS 环境中创建简单的应用程序。

以下是每章所述主题的快速总结。

### 第 1 章：Objective-C 中的 C 语言元素

Objective-C 语言是作为 C 语言的扩展而创建的，这意味着任何用 C 编写的程序也是一个有效的 Objective-C 程序。因此，Objective-C 中的许多基本表达式与其在 C 中的对应表达式完全相同。要成为一名优秀的 Objective-C 程序员，你需要对 C 语言有基本的了解。在本章中，我将快速介绍在 Objective-C 中使用的 C 语言最重要的概念。

### 第 2 章：类

在 Objective-C 中，类是应用程序的构建块。它们是用于定义方法和变量的语法结构，这些方法和变量构成了对象的组件。本章介绍 Objective-C 类的概念以及如何定义它们。类的主要特性之一是它们可以放在不同的文件中进行描述，一个用于接口，另一个用于实现。这种清晰的分离使得只共享类的重要方面成为可能，同时隐藏客户端不需要的任何实现信息。

### 第 3 章：字符串和容器类

Objective-C 程序的主要特征之一是不断使用动态对象来简化内存和其他资源的管理。Objective-C 运行时中最基础的类包括用于处理字符串、数字和对象集合的类。在本章中，我将展示如何使用这些基本类及其方法。

### 第 4 章：协议和类别

Objective-C 提倡实现与接口的清晰分离。用于鼓励这种分离的技术之一是协议的概念。协议允许类通过仅描述它们响应的方法子集来进行通信。另一种促进这种关注点分离的方式是通过类别来扩展类：程序员不必让所有人都知道某个特定类中可用的方法，而是可以根据需要添加类别。这会产生一个更加清晰的实现，从而减少对应用程序其他部分的依赖。

### 第 5 章：继承

面向对象编程是催生 Objective-C 的主要思想。OOP 的一个重要方面是能够创建继承现有类的方法、变量和属性的新类。然而，正确使用继承需要在设计阶段和派生类的正确实现中都要格外小心。在本章中，我将介绍你在创建类层次结构时需要熟悉的主要思想。
