# 引言

欢迎阅读《Objective-C 程序员参考手册》。本书面向那些先前有其他语言（如 Java、Python、Ruby 或 C++）开发经验，并希望将这些知识运用到 Objective-C 领域的开发者。

也许您曾玩过 iPhone，并希望开发一款期待在 AppStore 上架的应用程序。或者您想在 Mac OS X 上创建一个新的桌面应用程序。再或者，您可能只是因为公司需要支持 iOS 设备而必须用 Objective-C 编程。无论出于何种原因，本书都将帮助您从掌握其他语言的编程概念，过渡到学习如何（并希望您能享受）使用 Objective-C 编程。

本书的写作目标也在于帮助那些已经了解 Objective-C，但希望获得该语言最常用技术的简明参考的读者。您将能够理解在使用 Cocoa 和 Cocoa Touch（Objective-C 中两个最大且最重要的框架）时真正重要的主题。您还将学习到在任何需要 Objective-C 的编程场景中都能应用的基本主题。

您可能没有时间投入于那些更传统的书籍——它们通常花上数百页来解释那些对于在职开发者和高级编程学生来说已是标准词汇的简单概念。因此，本书力求仅通过简洁的示例传达最重要的主题。我们还收录了最常用类的快速参考，这样您就不必花太多时间在线查阅这些文档。

《Objective-C 程序员参考手册》对于那些希望有效运用知识，同时学习现代应用程序和移动开发中最具市场热度技能之一的开发者来说，是一个绝佳的选择。
