# 关于作者

![A978-1-4302-5906-0_BookFrontmatter_Fig1_HTML.jpg](img/A978-1-4302-5906-0_BookFrontmatter_Fig1_HTML.jpg)

卡洛斯·奥利维拉是一名供职于金融行业的软件工程师，拥有超过 10 年的 Objective-C 和 C++编程经验。在他的职业生涯中，卡洛斯为大小公司开发了多个大型应用程序。卡洛斯·奥利维拉在佛罗里达大学获得系统工程与运筹学博士学位，在巴西联邦大学（UFC）获得计算机科学硕士学位，并在巴西塞阿拉州立大学（UECE）获得计算机科学学士学位。卡洛斯还在组合优化领域进行了学术研究，其应用涉及金融、电信、计算生物学和物流等多个领域。他撰写了 20 多篇关于这些主题优化方面的论文，并合著了《网络路由优化的数学基础》（Springer，2010 年）。

卡洛斯·奥利维拉目前工作在纽约市，与妻子雅娜伊娜和儿子卢卡斯居住在新泽西州。您可通过他的网站[`http://coliveira.net`](http://coliveira.net)联系卡洛斯·奥利维拉。
