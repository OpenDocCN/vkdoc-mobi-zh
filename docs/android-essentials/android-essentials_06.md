# Android 基础

Chris Haseman

*那么，你想成为一名 Android 开发者？很好，你来对地方了。*

*无论你是浅尝辄止者、专业人士、爱好者还是编码狂热者，我希望你拿起这本书是出于想了解移动开发这块有些混乱的领域。具体来说，我希望你有兴趣接受开放手机联盟 Android 团队所发出的挑战。*

*我利用这个机会写下这本书，是因为很明显，Android 有潜力以前所未有的方式打开移动空间，让我们这些疲惫的移动程序员熬夜工作。过去的平台错误、以及这个行业（至少在美国）最大的成就不过是卖出几百万首彩铃的短视行为，足以写出好几本书来批判。你和你的同僚可以为这个行业带来期盼已久的改变，或许能为一个多年来似乎停滞不前的环境带来急需的新鲜空气。请原谅我的热情；我们已经等待了太久。*
