# Android 要点

亲爱的读者：

本书涵盖了构建 Android 应用程序所需的基本信息。它将帮助任何对 Android 开发感兴趣的开发者，无论是业余爱好者、专业人士，还是浅尝辄止者。在本书中，我将介绍您开始创建自己创新应用所需的要点。我之所以承担这个项目，是因为在移动软件行业工作五年后，我清楚地意识到我们需要帮助。随着移动设备变得越来越普及、功能越来越强大，甚至成为必需品，我们需要新鲜血液来克服困扰我们行业的停滞不前。作为一个行业，我们需要从制造千篇一律的铃声、壁纸和电子邮件应用中毕业。我希望，在您读完本书后，您能够开始向所有朋友夸下海口要制作的那款杀手级移动产品。
