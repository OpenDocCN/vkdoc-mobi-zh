# Android 要点

亲爱的读者：

本书涵盖了构建 Android 应用程序所需的基本信息。它将帮助任何对 Android 开发感兴趣的开发者，无论是业余爱好者、专业人士，还是浅尝辄止者。

在本书中，我将介绍您开始创建自己创新应用所需的要点。我之所以承担这个项目，是因为在移动软件行业工作五年后，我清楚地意识到我们需要帮助。随着移动设备变得越来越普及、功能越来越强大，甚至成为必需品，我们需要新鲜血液来克服困扰我们行业的停滞不前。作为一个行业，我们需要从制造千篇一律的铃声、壁纸和电子邮件应用中毕业。我希望，在您读完本书后，您能够开始向所有朋友夸下海口要制作的那款杀手级移动产品。

在 *Android 要点* 中，我将详细介绍如何安装和使用 Android SDK、创建和渲染用户界面工具、利用位置追踪和 Google 地图，以及通过 XML 解析器和网络客户端将这些内容整合在一起。这些工具和技巧应该足以帮助您跨越空白的编辑器——开发者世界那张令人生畏的空白画布——并踏上创建自己创新移动应用的道路。Android 可能代表着我们摆脱移动开发受限世界的最佳希望。用它来创造一些令人惊叹的东西吧。我迫不及待想看到您的作品。

此致，

Chris Haseman

Gravity Mobile 首席软件工程师
