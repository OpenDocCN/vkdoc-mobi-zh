# 第 1 章

## 必须热爱 Cocoa

欢迎！你来到这里，一定是因为你想为你的 Mac 编写程序。那么，你确实来对地方了。（来找普拉提的？右边第三个门。）当你读完本书时，你将了解创建快速、高效、外观精美的 Mac OS X 应用程序所需的一切知识。

创建现代 Mac 应用程序的关键是 Cocoa。根据苹果公司的说法，Cocoa 是一组面向对象的框架，为 Mac OS X 应用程序提供运行时环境。当你阅读本书时，你将学习关于 Cocoa 框架和运行时环境的所有知识。现在，请将 Cocoa 视为一个程序员助手，它负责处理 Mac 开发中涉及的许多日常维护工作。几乎 Mac 应用程序执行的每一个常见任务，从绘制窗口到在文本字段中闪烁光标，在使用 Cocoa 编写程序时都会为你自动处理，让你能够专注于使你的应用程序独一无二的代码。

Cocoa 几乎为你的每一个开发需求都提供了类。从窗口到菜单，从滚动条到按钮，从图像到图标，Mac OS X 用户界面的每个部分都有对应的 Cocoa 类。如果你能想到一个想添加到你自己应用程序中的用户界面元素，那么这个元素很可能已经作为 Cocoa 类实现了。

使用 Cocoa 的另一个好处是它与 Mac OS X 紧密集成。使用 Cocoa 构建你的应用程序，你的应用程序将能与其他程序良好兼容，并与 Mac OS X 的元素（如 Finder 和 Dock）无缝连接。

## 何时 25 年不等于 25 年？

Cocoa 自 1986 年以来以某种形式存在。我们称之为 Cocoa 的技术是从 NeXTStep AppKit 演变而来的，后者是为 NeXT 平台开发的应用程序构建工具。当苹果公司在 1996 年收购 NeXT 时，他们开始构建一个新版本的 Mac OS，也就是我们现在所知的 OS X，新操作系统的很大一部分基于从 NeXT 收购的技术。

随着 OS X 的发展，Cocoa 也在发展。苹果公司加入了来自经典 Mac OS 的技术，如 QuickTime，以及全新的技术，如支持 OS X 所有花哨视觉效果和动画的 Quartz 渲染系统。他们还确保在此过程中保持了 Mac 著名的易用性。

要点：Cocoa 在不断发展和扩展。你将在本书中学习的开发工具、库和框架是超过 25 年实验和改进的成果。

## 获取 Mac 并下载工具

在你开始使用 Cocoa 创建应用程序之前，你需要一台 Mac。它不必是最新或最强大的 Mac；事实上，过去四五年左右生产的几乎任何 Mac 都能很好地完成本书中的练习。当你对编写软件变得更加认真时，你可能会发现投资一台更新或更快的机器是值得的，但现在，几乎任何基于 Intel 的 Mac 都能很好地满足你学习所需的知识。最新的 Mac 目前最高支持 OS X 10.8，而最古老的基于 Intel 的 Mac 无法运行任何高于 10.7 的 OS X 版本。但本书将使用的版本 `Xcode 4.6` 在 10.7 和 10.8 上都能正常工作。

你还需要拥有一个 Apple ID（可能与你已在 Mac 或 iOS App Store 或其他 iTunes 购买中使用的 ID 相同），并且强烈建议你加入苹果开发者连接（ADC），这是苹果的开发者关系组织。有付费会员，但基本会员是免费的。要加入 ADC，请打开你的网页浏览器，导航至 `http://developer.apple.com/mac/`。如果你已有 Apple ID，请按下 `Log in` 按钮并输入你的信息。如果你是苹果在线服务的新用户，请点击 `register` 链接并创建你的账户。

无论你是注册成为开发者，还是仅使用你的 Apple ID，获取苹果的免费开发者工具集 `Xcode` 最简单最好的方法是从你的 Mac 上的 App Store 下载。只需启动 App Store，搜索 `Xcode`，然后点击安装。与 App Store 上的大多数内容一样，安装过程基本不言自明。`Xcode` 的安装可能需要比大多数其他应用程序更长的时间，因为它是一个相当大的下载文件。

**注意** 如果你已经为 iOS 开发安装了 `Xcode`，那么你已经万事俱备了。`Xcode` 包含创建 OS X 应用程序和 iOS 应用程序所需的所有工具和框架。虽然框架确实存在重大差异，但工具基本相同，所以如果你做过任何 iOS 开发，创建 Cocoa 程序时应该会感到得心应手。

## 下载源代码

除了下载和安装 `Xcode`，你还需要下载与本书配套的示例项目。你可以在 Apress 的《在 Mac 上学习 Cocoa》网页找到代码归档，网址为 `www.apress.com/9781430245421`。此页面包含本书的所有元数据，包括代码归档、勘误表等。

## 开始前须知

本书假定您已具备一些基本的编程知识。您应熟练运用循环、变量、指针和链表等概念。同时，本书也假定您理解面向对象编程的基础知识，并熟悉 Objective-C 编程语言。从 Mac OS X Leopard 10.5 开始，该语言推出了一个名为 Objective-C 2.0 的新版本。如果您不熟悉 Objective-C 语言的最新特性，也不必担心。我们会着重突出所使用到的 2.0 语言特性，并解释其工作原理及使用原因。

## 初次接触 Objective-C？

以下资源可帮助您快速入门。

- 编程新手？请查阅我们好友 Dave Mark 编写的 *Learn C on the Mac，第 4 版*（`www.apress.com/9781430218098`）。
- Objective-C 新手？请查阅由 Mac 编程专家 Scott Knaster、Waqar Malik 和 Mark Dalrymple 编写的 *Learn Objective-C on the Mac，第 2 版*，这是一本出色且易于入门的 Objective-C 书籍（`www.apress.com/9781430241881`）。
- 接下来，前往 Apple 的 Mac 开发者库，下载一份 *Programming with Objective-C*，这是一份对该语言非常详细且全面的描述，也是一本绝佳的参考指南（`http://developer.apple.com/library/mac/#documentation/Cocoa/Conceptual/ProgrammingWithObjectiveC`）。

## 准备好了吗？

Mac 是一个卓越的计算平台，而 Cocoa 是构建 Mac 应用的最佳工具。如果您是从其他平台转投 Mac，可能会觉得使用 Cocoa 和 Objective-C 有些令人困惑。即使一开始您可能难以找准方向，但随着您阅读本书的深入，我们相信您很快就会开始理解“Cocoa 之道”。只要稍加坚持，一切都会变得有意义。

请您记住，本书中的练习并非一个简单的清单，完成它们并不能让您立即成为 Cocoa 开发大师。在每一步中，请确保您在进入下一个项目之前理解刚刚完成的操作。不要害怕修改代码。大胆实验并观察结果。这是在 Cocoa 这样的环境中理解编码复杂性的最佳方法之一。

另外，请记住，我们为读者设立了一个讨论本书练习和内容的留言论坛。因此，如果您遇到困难，或者想与我们及其他 Cocoa 开发者讨论任何 Cocoa 编程主题，请前往 `http://learncocoa.org` 注册论坛。
