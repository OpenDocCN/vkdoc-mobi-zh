**图 1-10.** *为 Android 模拟器创建新设备配置文件*

**注意：** 通常，当你想引用自己的机器时，你会输入 `http://127.0.0.1` 或 `http://localhost`。但在模拟器的浏览器中，这些地址访问的是模拟器本身，而不是主机。你应该使用的地址是 `http://10.0.2.2`。在模拟器中输入此地址；你应该会看到测试页面。如果你使用的是 80 以外的端口，请不要忘记添加端口号。

创建一个名为 `ANDROID_HOME` 的环境变量，并将其指向 SDK 安装路径。然后，用 `ANDROID_HOME/tools` 和 `ANDROID_HOME/platform-tools` 更新 `PATH` 变量。Android SDK 有两个包含可执行脚本的文件夹。确保将这两个文件夹都添加到 `PATH`。
