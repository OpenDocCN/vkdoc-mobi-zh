# 第 12 章：本地化与国际化

## 章节摘要

本地化和国际化不仅仅是简单地将应用中的英文文本替换为用户母语文本。它还涉及文化敏感性，即认识到用户希望应用能够给人一种出自其本国开发者之手的感受。如果你在应用中使用了一句习语，比如“她的面孔看起来很眼熟”（原文："her face rang a bell"），不要只将其翻译成另一种语言——当然更不要使用在线翻译工具来处理。同样，对于应用中的颜色、图片和音频，也需要保持文化敏感性。红色在中国是好运的象征，因此你的应用可以检测到中文区域设置并相应地调整颜色的显示方式。

阅读完本章后，你将能够使应用从仅支持一种语言和区域设置，扩展到支持 iOS 所支持的全部语言和区域设置。这是一项关键技能，能使你的应用从单一市场发布走向全球现象级应用。这项功能很容易被忽略，但它可能是你为应用添加的最重要的功能。
