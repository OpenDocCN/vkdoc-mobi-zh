# 致谢



我要感谢 Apress 的所有同仁为这本书付出的辛勤工作——特别是我的技术审阅者斯科特·加德纳，他提出了许多改进作品质量的建议，并督促我始终提供顶级内容。我的开发编辑道格拉斯·普德尼克在引导内容方向方面也做得非常出色，其他编辑米歇尔·洛曼、珍妮弗·布莱克威尔、凯莉·莫里茨和吉尔·巴尔扎诺都为这本书提供了巨大的帮助。我还要感谢底特律实验室的团队成员，特别是联合创始人保罗·格洛姆斯基、亨利·巴拉农、内森·休斯和丹·沃德，感谢他们理解我白天编写应用、晚上撰写本书的艰辛，并为我提供了一个绝佳的工作场所。在家里，我要特别感谢我的妻子阿曼达，她在我撰写本书期间一直包容我，同时她还怀着我们的第一个孩子，这让我对她感激不尽。最后，我要感谢我的父母，是他们帮助我开启了编程之路；在我上幼儿园时，妈妈会从书上抄下`BASIC`程序，让我回家后可以摆弄它们；后来在我上小学时，爸爸给我买了一本`Visual Studio`——我为别人编写的第一款应用是一个退休金计算器，他可以放在办公室的屏幕上，用来倒计时直到退休的秒数。在这本书出版的过程中，我得到了很多人的帮助，所以，无论是否在此提到，我都要对所有人说一声谢谢。
