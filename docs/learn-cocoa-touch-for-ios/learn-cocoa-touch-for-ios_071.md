# 第 12 章：本地化与国际化

**图 12-4.** *三种不同键盘（从左至右）：英文、法文以及手写模式下的繁体中文*

如您所见，英文和法文键盘非常相似，但中文键盘处于手写模式，允许用户用他们的手指绘制字符。当用户有多个可用键盘时，底部行会出现一个地球图标；点击此图标可切换键盘。

如何处理用户提供的文本取决于您，但一条通用规则是：对您使用的任何网络服务采用`UTF`编码，而非`ASCII`。`UTF`设计用于接受多语言输入。尽管这看似一个微不足道的考虑，但如果您的用户名中包含特殊`Unicode`字符，他们不会希望这些字符被您的应用弄得一团糟。

在处理显示给用户的文本时，您还应特别小心。

只要可能，您的应用应以其用户的母语显示内容。
