# 排版后的文本

在本章中，我们介绍了 Objective-C 的基础知识。现在，你应该能够自如地创建类、声明和实现方法、使用属性、遵守协议，以及运用 Xcode 和其他开发者工具来完成这些任务。得益于 ARC（自动引用计数），你甚至无需编写任何内存管理代码，就能写出完美的内存管理方案！第 1 章中的“Hello, World!”示例项目现已包含一个 `Person` 类，你已用它来展示一个示例人物。同时，你也应该开始思考模型-视图-控制器（Model-View-Controller）范式，为学习第 3 章做好准备。
