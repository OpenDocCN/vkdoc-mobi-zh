# 引言

随着 iOS 及其相关硬件产品的每一次迭代发布，苹果公司和全球各地的记者都会用夸张的言辞来描述“革命性”的功能、“极其出色”的设备以及“难以置信”的销量。数字也从未令人失望：数亿台 iOS 设备已被售出，数十亿美元的收入流向了开发者。随着我们进入后 PC 时代，智能手机和平板电脑成为了我们的主要工具。对于这些新设备而言，苹果的 iOS 始终是最用户友好、功能最强大的平台，世界各地的开发者都能通过 App Store 提供产品并从中受益。尽管如此，这个市场仍在持续增长，尤其是当用户只需签订两年合约，就能以极低的预付价格获得一部 iPhone 时。随着智能手机的市场准入门槛降低，用户基数不断扩大，机遇也随之激增。本书将助您抓住这一机遇。我们将使用 Mac OS X 上的 Xcode 启动并运行，在学习 Objective-C（您将用于开发应用的语言）的过程中创建应用程序，并巡览那些使 Cocoa Touch 成为全球最佳开发环境之一的框架。

就像在苹果环境下编程时您会习惯的那样，这里有一些规则。因此，您需要准备一些东西才能通读本书：一台搭载 Intel 处理器、运行 Mac OS X 10.7（Lion）或更新版本、并安装了 Xcode 4.3 或更新版本（可从 Mac App Store 获取）的 Mac 电脑，以及一台运行 iOS 5.1 或更新版本的 iOS 设备（理想情况下）。尽管旧版本的 Mac OS X、Xcode 和 iOS 可能仍在使用，但本书中提供的屏幕截图和逐步说明可能不适用于其他版本。

## 本书读者对象

本书假定读者具备基本的编程知识。您不必是专家，但任何使用 C、C++甚至 Java 的经验都将有助于理解本书早期阶段所解释的概念。您还应熟悉苹果 Mac OS X 和 iOS 操作系统的基本操作，能够在 Mac OS X 的文件系统中自如操作并启动 Xcode，同时能够在 iOS 上启动应用并理解应用的基本行为。

## 本书结构

通常，本书各章会从更抽象的概念开始。当开发框架和库有所演进时，我们会从较旧、较复杂的方式入手，然后过渡到较新的实现方法，以便更好地理解事物为何会发展到当前状态。随着每章的深入，我们将从抽象转向具体，通过示例项目和示例代码来展开。我们将在多个章节中开发两个应用，此外还会提供其他更小的示例。

-   **第 1 章** 让您开始使用 Xcode，并创建一个“Hello, World!”应用。
-   **第 2 章** 详细介绍了 Objective-C 语言，包括内存管理、最佳实践以及该语言的最新特性。
-   **第 3 章** 讨论如何使用视图控制器，这是您在 iOS 开发中将用到的最重要的对象类型之一。
-   **第 4 章** 涵盖了数据处理，从在应用内部移动数据，到从磁盘保存和加载数据。
-   **第 5 章** 详细介绍了处理用户触摸和基本应用流程。
-   **第 6 章** 涵盖网络和 Web 服务，包括解析 JSON 和 XML。
-   **第 7 章** 介绍了代码块，这是苹果对 C 语言的新增特性，用于封装代码。
-   **第 8 章** 进一步解释了 iOS 中的消息分发过程，进而讨论了多线程代码。
-   **第 9 章** 涵盖了应用中的用户界面设计。
-   **第 10 章** 详细介绍了 iOS 设备上可用的众多硬件 API，包括加速度计、陀螺仪和磁力计，以及使用 GPS 的定位服务。
-   **第 11 章** 概述了如何在应用中使用媒体，包括音频和视频。
-   **第 12 章** 涵盖了国际化与本地化流程，这有助于扩大您应用的受众范围。
