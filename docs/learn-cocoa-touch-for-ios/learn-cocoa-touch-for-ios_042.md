# 排版后的文档

我们首先要做的是将视图控制器的当前标题保存到 `title` 变量中。这样我们就能在 `initWithStyle:` 中修改为视图控制器设置的标题，而无需在此方法中再次修改。接下来，我们在授权账户时将标题改为"正在授权..."，在加载推文时改为"正在加载推文..."，最后在推文加载完成后恢复为最初的标题。对于后两种操作，我们使用 `performSelectorOnMainThread:withObject:waitUntilDone:` 来确保标题始终在主线程上被修改，因为在其他线程中调用 UI 方法可能会导致 iOS 出现问题，包括崩溃和意外的用户界面设置。我们将在下一章中更深入地了解线程，所以如果你现在不太明白这意味着什么，不必担心。构建并运行你的应用，你应该能看到标题随着网络活动指示器一起变化。我们的 Twitter 客户端距离完成还有很长一段路要走；在下一章中，我们将在讨论代码中的性能与并发性时为其添加更多功能。
