# 命名对象

`对象名称`属性是一个可选的静态文本标识符，可以在布局模式下手动分配给某个布局对象或对象组。虽然这些标识符有些形式化，但在查看`对象`面板中的元素时很有帮助，并且某些函数和脚本步骤必须依赖它来引用非字段对象：如门户、选项卡控件、滑动控件和网页浏览器。例如，`GetLayoutObjectAttribute`和`Get (ActiveLayoutObjectName)`函数*只能*通过名称属性来引用对象。同样，像`转到对象`、`刷新门户`和`设置网页浏览器`这样的脚本步骤也需要对象名称作为目标引用。

不应将`对象名称`与`字段名称`混淆，因为它们是两回事。字段名称是对特定数据容器的通用定义，无论该字段放置在何处都通用；而对象名称则是每个布局对象实例所独有的属性，字段也不例外。某个字段的实例可以被赋予与其定义名称不同的对象名称，并且同一字段在任何布局上的每个实例都可以拥有不同的对象名称。

要分配对象名称，请进入布局模式并选择一个对象。在`检查器`面板的`位置`选项卡中的`位置`组里，点击`名称`文本区域，然后为对象键入一个名称。名称不区分大小写，并且必须唯一。

**提示：** 如果从其他布局粘贴的对象具有相同的对象名称，则会自动重命名，以确保所有对象名称的唯一性。在粘贴新对象之前，务必先删除旧对象，并/或在粘贴后检查任何重要的名称。
