# Claris International 重生

2019 年，FileMaker 宣布恢复 Claris 品牌名称，并将公司更名为 Claris International Inc.。此次更名旨在更好地反映其扩展产品计划的意图，例如近期推出的 Claris Connect 工作流集成服务。2020 年 5 月，Claris 发布了 FileMaker Pro 19 版本，其显著特性包括：

-   **拖拽安装（macOS）** — macOS 应用程序作为一个包含嵌入式资源的程序包，可以拖入 `Applications` 文件夹进行一键安装。
-   **插件** — 一项革命性功能，允许通过将小部件拖放到布局上，将一组表、关系、布局对象和脚本添加到数据库中。
-   **JavaScript** — Web 查看器和 FileMaker 脚本现在可以直接相互通信。可以使用 JavaScript 库在 Web 查看器中添加高级界面元素。
-   **布局模式下的键盘导航** — 使用键盘或新的选择对话框快速跳转到其他布局。
-   **深色模式支持（macOS）** — 根据系统偏好设置中的设置更改视觉外观。
-   **启动文件** — 偏好设置，用于在应用程序启动时自动打开特定文件。
-   **新增函数和脚本步骤** — 近场通信 (NFC) 功能、转换 FileMaker 格式的路径、访问卡片尺寸、利用 Core ML（机器学习）的强大功能等。
