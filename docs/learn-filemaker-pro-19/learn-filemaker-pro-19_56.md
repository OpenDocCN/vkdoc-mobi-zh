# 17. 介绍布局

*布局*是开发者设计的图形模板，定义了如何渲染记录、字段和对象以产生交互式体验。每个数据库窗口一次显示一个布局。在特定时刻激活的布局字面上称为*当前布局*。当数据库打开时，这是一个根据数据库配置方式选择的默认布局。它可能是在*文件选项*（第[6]章）中选择的布局、由*脚本触发器*（第[27]章）打开的布局，或者在本地计算机上运行时文件关闭时最后打开的布局。与 Claris 入门解决方案中方便的预定义布局不同，自定义数据库从一个空布局开始，开发者必须从中设计界面。本章通过介绍布局基础知识开始探讨界面设计，包括：

- 理解上下文访问
- 剖析布局
- 规划布局
- 使用布局模式
