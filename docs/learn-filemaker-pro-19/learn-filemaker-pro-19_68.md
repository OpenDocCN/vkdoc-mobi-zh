# 设置门户筛选

要设置筛选示例，请按以下步骤操作：

1.  在 `Company` 表中，创建一个名为“公司联系人门户筛选”的文本字段。可选地，让此字段使用全局存储，以避免用户之间的冲突，并保留用户在所有记录中的输入。
2.  将该字段放置在布局上，位于门户上方，并使用一个能清晰说明其功能的标签，例如“联系人筛选”。或者，使用一个占位计算作为提示。
3.  打开“门户设置”对话框。
4.  单击“筛选门户记录”旁边的“指定”按钮。
5.  输入一个公式，用于指示 `Contacts` 表中的相关记录何时应根据用户在筛选字段中输入的内容显示在门户中（请参阅以下示例）。
6.  然后保存公式并关闭“门户设置”对话框。
