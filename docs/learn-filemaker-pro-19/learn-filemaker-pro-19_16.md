# 维护硬件和软件

始终保持稳定且最新的硬件和软件，并密切关注以下细节：

*   不要急于升级任何东西。在升级硬件或软件之前，先在数据库的*安全副本*上进行全面测试。
*   保持主机和用户操作系统为最新版本。
*   保持 FileMaker 应用程序为最新版本，包括所有临时的错误修复版本。
*   在所有计算机上使用足够的不间断电源和电涌保护器，以防断电时发生意外关机，尤其是数据库主机服务器。
*   使用磁盘工具程序定期进行硬件维护，不要让计算机过时到有故障风险的地步。
*   不要让主机计算机的磁盘空间耗尽。
*   使用配备合适线缆的快速硬件和网络设备。
*   定期进行数据库维护。

**提示**  
考虑使用 FileMaker Cloud，它会自动处理大多数硬件和软件维护功能。
