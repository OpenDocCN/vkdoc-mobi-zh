# 使用 `UNION` 子句

添加 `UNION` 子句可以合并两个或多个 `SELECT` 语句的结果，无论这些语句是来自具有相同或不同标准的同一个表，还是来自不同的表，只要每个语句选择的字段数量相同，并且每个字段位置的数据类型在所有语句中都一致。例如，如果第一个 `SELECT` 语句返回三个字段，数据类型分别为*文本*、*数字*和*文本*，那么第二个 `SELECT` 语句也必须返回三个字段，且具有相同顺序的相同数据类型。以下是在两个 `SELECT` 语句之间添加 `UNION` 的简化模式，未显示可以包含在其中任一或两个语句中的其他子句。

```
SELECT  FROM  UNION SELECT  FROM 
```

默认情况下，此子句会自动从合并结果中排除重复条目。使用 `UNION ALL` 可包含所有结果，即使有重复项也不例外。
