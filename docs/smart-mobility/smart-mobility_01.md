# 前言

如今，我们正目睹出行系统和服务领域的若干范式转变。城市正在推动交通领域脱碳，并从以汽车为中心的出行转向多模式出行，从二维街道的受限出行转向三维空间出行，从固定时刻表的出行转向按需出行，以及从分散孤立的出行转向无缝集成的出行。出行公司正在从制造业和贸易经济转向服务经济或服务化，例如出行即服务（`MaaS`），这代表了人员和货物运输的新自由主义化，同时从不可持续的“售车数量”收入模式转向“车辆行驶里程（`VMT`）基础”、基于信息经济学的数据与客户体验货币化以及基于乘客经济的收入模式。物流服务提供商正从传统缓慢、僵化、不透明的末端配送转向快速、灵活、透明的末端配送服务。人们正从拥有权转向使用权，从被动出行转向主动、零影响出行。这些范式转变的背后是各种基础技术、技术赋能因素和出行颠覆性技术。

本书全面概述了智慧出行系统和服务，并阐述了其基础技术、技术赋能因素和颠覆性技术。本书重点介绍了这些技术的市场规模、潜在增长及其生态社会经济影响。此外，还讨论了新冠疫情对消费者行为与偏好的影响，以及出行系统各个方面（尤其是微出行、共享出行、公共交通和非接触式末端配送服务）预期的短期颠覆和长期结构性变化。

本书主题的展开遵循一个逻辑流程，从基础技术到技术赋能因素，再到智慧出行颠覆性技术。以下是按章节顺序对本书内容的简要说明：

- 第 1 章概述了以人为本的智慧城市。
- 第 2 章介绍了智慧出行三要素，即技术、治理和城市规划这三个互补因素。这三个组成部分并非相互独立，而是相互影响。
- 第 3 章介绍了定位、导航与授时（`PNT`）、地理信息系统（`GIS`）、无线通信、移动云计算（`MCC`）、区块链、物联网（`IoT`）、人工智能（`AI`）、机器人技术和电气化作为智慧出行系统和服务的基础技术。
- 第 4 章讨论了智慧出行的若干技术赋能因素，如智能基础设施、网联出行、自动驾驶出行、电动出行、微出行、主动/慢行出行、包容性出行和情境感知系统（`CAS`）。
- 第 5 章重点阐述了潜在的智慧出行颠覆性技术，如颠覆性出行平台（自动驾驶地面车辆、城市空中交通（`UAM`）、水上出租车、自动旅客捷运系统（`APMs`）、超级高铁和城市高铁）、共享出行、出行即服务（`MaaS`）、按需出行（`MOD`）、无缝集成出行系统（`SIMS`）、末端配送、车辆即服务（`VaaS`）、零工经济和乘客经济。
- 最后，第 6 章讨论了新冠疫情对智慧出行系统和服务的影响。

我坚信，未来的出行是以人为本、软件定义、网联化和电动化的。尽管近年来发展迅速，但智慧出行仍处于起步阶段。日益需要利用上述各种基础技术、技术赋能因素和颠覆性技术，来改善客户与出行提供商之间的关系，实现不同出行服务之间可负担、包容且无缝的集成。围绕若干智慧出行技术的法律和监管环境需要充分考虑不同利益相关者的意见和关切而得到良好发展。此外，在城市规划方面应考虑渐进式和革命性的变革，以适应新兴的智慧出行服务。

本书面向在职专业人士、培训中心和学术机构。受众广泛，包括寻求对未来出行技术和服务拥有整体愿景和新思路的大学生、研究人员、出行工程师、技术专家和城市规划者。本书为读者提供了对当前和新兴智慧出行系统全面且易于理解的介绍，重点聚焦将塑造未来出行的基础技术、技术赋能因素和颠覆性技术。

虽然本书描述了许多未来的可能性，但一些令人惊喜的成就已经实现并公开使用。我们每天都在享受并依赖共享出行服务、网联出行、电动出行、主动出行、微出行以及当日达和即时达的末端配送服务。此外，在不远的将来，我们将把安全且有趣的自动驾驶车辆（`SDVs`）作为拥有以消费者为中心的产品和服务的第三生活空间来享受；我们将能够把自动驾驶车辆用作移动汽车旅馆、美容沙龙、移动商店和移动诊所；我们的在线订单将由送货机器人送到家门口；我们将安全便捷地乘坐零排放的超级高铁、自动驾驶空中出租车、空中地铁或自动驾驶船只出行；不同出行模式之间的无缝集成将成为常态；最终，可负担、包容和可持续的出行将得以实现。希望您享受这段通往人员和货物运输未来的激动人心之旅。
