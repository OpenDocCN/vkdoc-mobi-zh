# 智慧出行：基础技术、使能技术与颠覆性技术

以下章节聚焦于智慧出行系统与服务的技术层面，阐述其基础技术、使能技术与颠覆性技术。本书涵盖的基础技术包括`定位、导航与授时（PNT）`、`地理信息系统（GIS）`、`无线通信`、`移动云计算`、`区块链`、`物联网（IoT）`、`人工智能（AI）`、`机器人技术`和`电气化`。`PNT`技术赋予智慧出行系统定位系统内固定与移动资产、引导移动资产导航以及保持精确授时的能力。`GIS`提供必要的抽象化工作环境的地理信息。`无线通信`使智慧出行系统内能够进行高可靠性实时数据的传输与交换。`移动计算`使授权终端用户和移动资产能够随时随地快速访问数据、信息和计算资源。基于分布式计算概念，`区块链`技术能够实现身份验证、访问控制和快速支付等多种服务，并有潜力重塑智慧出行系统与服务。`物联网`技术能够将智慧出行系统内的任何固定或移动资产连接到互联网，并促进数据共享，以实现监控、控制和优化。`人工智能`是多项现有及新兴智慧出行系统与服务的核心技术及驱动力。鉴于多项智慧出行技术始于机器人研究实验室，并基于机械设计、运动系统、感知、规划、控制、导航与协同算法等机器人领域的进步，`机器人技术`被视为一项基础技术。`电气化`是实现脱碳可持续出行的另一项基础技术。关于这些基础技术的更多细节，请参阅`第[3]章(#511013_1_En_3_Chapter.xhtml)`。

在智慧出行背景下，本书涵盖了几项使能技术，例如智能基础设施、网联出行、自动驾驶出行、电动出行、微出行、主动/软出行、包容性出行以及`情境感知系统（CAS）`。支撑性智能基础设施包括智能交通系统、智能路面、智能交叉路口、高品质骑行基础设施（如自行车高速公路，也称自行车快速路或超级高速公路）、智能停车和充电基础设施。网联出行是实现实时导航与路径规划、交通信息、安全预警、事故规避、`高级驾驶辅助系统（ADAS）`和`自动驾驶系统（ADS）`的关键使能技术。自动驾驶能够大幅减少通常由人类驾驶失误导致的伤亡事故。电动出行支持并驱动多种环保出行平台，如电动汽车、电动自行车、电动滑板车、电动垂直起降飞行器（eVTOL）以及电动摆渡车/通勤车。微出行平台实现了可负担且可持续的第一/最后一公里出行。主动、软性或零影响出行，为第一/最后一公里的城市出行提供了健康、廉价、有趣且环保的选择。出行辅助设备为老年人和行动不便者实现了包容性出行。最后但同样重要的是，情境感知是能够根据特定情境信息进行调整的情境感知出行系统与服务的使能技术。关于这些使能技术的更多细节，请参阅`第[4]章(#511013_1_En_4_Chapter.xhtml)`。

本书描述的出行颠覆性技术包括颠覆性出行模式、共享出行、`出行即服务（MaaS）`、`按需出行（MOD）`、`无缝集成出行系统（SIMS）`、最后一公里配送、`车辆即服务（VaaS）`、零工经济与乘客经济。用于人员出行和货物配送的颠覆性出行平台包括：自主地面车辆（例如自动驾驶摆渡车（SDS）、自主轮椅、自主推车和自主机器人代客泊车）、城市空中出行（例如空中出租车、空中地铁和最后一公里配送无人机）以及水运工具（水上出租车、水上巴士和自主船只/舰艇）。自动旅客捷运系统、超级高铁和城市环线高速是颠覆性出行平台的其他例子。车辆共享、点对点共享、`需求响应式公交（DRT）`、拼车、网约车、网约出租车、合乘等服务是共享出行服务的例子。`出行即服务`、按需出行和无缝集成出行系统促进了便捷直达的无缝出行。现代最后一公里配送服务专注于以最快、最灵活、最透明的方式，将货物从仓库或配送中心运至最终目的地。在颠覆性创新时代，出行平台或车辆可被视作一种服务，以实现车辆内未使用的计算和/或网络资源以及海量收集数据的共享与货币化。基于零工经济和乘客经济，尤其是在自动驾驶汽车时代，车辆可被重新用于休闲、商务或医疗保健等不同用途，由此可催生多种智慧出行服务。关于这些颠覆性技术的更多细节，请参阅`第[5]章(#511013_1_En_5_Chapter.xhtml)`。

## 3. 智慧出行：基础技术

基础技术是智慧出行的基石，可用于有效开发和支撑重要的新产品与服务。本章将讨论`定位、导航与授时（PNT）`、`地理信息系统（GIS）`、`无线通信`、`移动云计算`、`区块链`、`物联网（IoT）`、`人工智能（AI）`、`机器人技术`和`电气化`，作为智慧出行系统与服务基础技术的范例。
