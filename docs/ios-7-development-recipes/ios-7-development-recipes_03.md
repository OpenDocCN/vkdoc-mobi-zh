# 引言

软件开发中容易的部分是知道如何用当前所用的编程语言编写代码。更困难的部分是掌握平台的编程接口，并达到能够有效地将想法转化为具有真正价值的工作特性的水平。iOS 7 虽然极其强大且易于使用，但在这方面也不例外。`Objective-C` 被许多人认为是一种有点“奇特”的编程语言，您能很快掌握它，甚至学会欣赏它。然而，您很可能需要花大量时间学习各种 API 和框架。

我们相信，获得必要知识并达到高生产力水平的最佳方法是通过动手实践。我们认为，最好的学习方式是遵循分步步骤，创建小型项目，在其中测试和调整各种特性，在实际项目中实现它们之前先获得感觉。

基于这个想法，我们创作了 *iOS 7 Development Recipes*。本书包含超过 600 页的示例代码，并附有关于如何创建小型测试应用程序的说明，使您能够在 iOS 7 设备或 iOS 模拟器上运行这些代码。我们已尽力基于 iOS 7 的特性覆盖尽可能多的主题。我们希望本书能为您提供所需的基础知识，帮助您将伟大的想法转化为出色的应用程序。
