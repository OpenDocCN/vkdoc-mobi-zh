# 更多音频理论

关于音频理论，还有大量内容需要学习和理解，一本 Android 书籍的篇幅无法涵盖这一切。诸如序列、合成等话题都有成书库的著作，而且在线上也能找到关于这些主题的丰富信息。维基百科是一个不错的起点。至于关于采样、比特率、编码格式等内容的实践和教程，可以查看下一节介绍的工具。

### 选择更多音频工具

回放已创建的音频是对您应用程序的一个很好的补充，无论是音乐、播客还是应用程序的音效。在某个时候，您很可能需要超越使用他人音频资源的层面，想要创建自己的音频，或者混音和调整其他预先录制的音频。音频录制和编辑软件是一个庞大的领域，但作为入门，查阅一些流行的（且免费的！）可选方案是不会错的。

#### 认识 Audacity

在某些软件领域，免费开源软件具有强大影响力甚至占据主导地位，音频编辑就是其中之一。Audacity 十多年来一直是一个受欢迎的音频编辑套件。用 Audacity 网站关于页面的话来说：

*[Audacity] 是一个免费的、易于使用的多轨音频编辑器和录音机，适用于 Windows、macOS、GNU/Linux 和其他操作系统。其界面已被翻译成多种语言。*

Audacity 拥有海量功能，其中最重要的可以总结如下：

1.  录制实时音频——来自您计算机可用的任何输入源
2.  录制计算机回放——捕捉您可能正在运行的其他系统的音频
3.  完整的编辑功能，支持所有关键的音频格式，包括 `MP3`、`WAV`、`AIFF`、`FLAC`、`Ogg Vorbis` 等（显然，这涵盖了几乎所有 Android 支持的格式——一个额外的好处！）
4.  通过丰富的插件库系统编辑其他格式，例如 `M4A` 和 `WMA`
5.  用于复制、剪切、拼接、混音、音高和速度调整以及变速的完整工具
6.  支持 16 位、24 位和 32 位质量以及全频谱采样率
7.  高级功能，如频谱图分析和其他强大选项

我还可以列举更多特性和功能，但您可以自行在 Audacity 自己优秀的文档网站上阅读更多信息，网址是 [`https://manual.audacityteam.org`](https://manual.audacityteam.org)。在撰写本书时，Audacity 的版本是 2.4.2。由于它是开源且免费的，我强烈建议您至少下载并安装它来试用一下。[`https://manual.audacityteam.org/man/tutorials.html`](https://manual.audacityteam.org/man/tutorials.html) 上有非常易学的教程，其中我推荐每位刚接触音频的爱好者都从两个教程开始：编辑音频文件和创建新录音。您很快就能为您的 Android 应用程序创建素材了。

#### 使用 WavePad

另一个强大的视频编辑工具是 WavePad，它在近年来声名鹊起。WavePad 拥有至少与 Audacity 一样强大的功能集，在某些情况下甚至更进一步。WavePad 一个关键的优势领域是，它支持更多生僻和罕见的音频文件格式，例如 `VOX`。WavePad 最引人注目的部分可能是它拥有自身的移动版本，包括一个 Android 版本。这意味着您可以使用自己的移动设备来创建、编辑和优化您正在为同一设备编写的新应用程序所需的音频。

WavePad 采用“免费增值”模式，部分功能免费提供，但其他功能需要付费许可证。您可以在 [`www.nch.com.au/wavepad/index.html`](http://www.nch.com.au/wavepad/index.html) 找到关于 WavePad 的更多信息。

#### 苹果 Mac 用户的选择：GarageBand

如果您拥有 Mac 电脑，还有一个值得考虑的选择。过去十年（或更久）出厂的每一台 Mac 都预装了苹果自己的音乐编辑和创作软件 GarageBand。GarageBand 拥有许多出色的功能，并围绕它形成了一个非常强大的社区。显然，由于 GarageBand 是 Mac 的捆绑软件，我无法为您提供下载链接，如果您有 Mac，GarageBand 已经包含在您购买机器所付的款项中——因此开始使用它无需额外费用。
