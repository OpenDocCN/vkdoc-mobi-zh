# 前言

iPhone（iPod touch 和 iPad）平台配备了一系列日益丰富的内置传感器：GPS、加速度计、磁力计以及最新的陀螺仪。这些设备还拥有几乎无处不在的数据连接，无论是通过本地无线热点还是运营商数据网络，并可通过多种方法（包括 GPS）进行用户定位。它们堪称分布式传感器网络的卓越中心。

然而，直到最近，要将这些原本有趣的设备与 iPhone 连接起来仍然相当困难。苹果专有的底座连接器是一大主要障碍。现在情况已经改变，读完本书后，你将能够将你的手机用作传感器网络的中心，使其成为物联网的一部分。

## 谁应该阅读本书？

本书介绍了如何将你的 iOS 设备连接到现实世界。为此，我们将广泛使用 [Arduino](http://arduino.cc/) 开源电子原型平台。如果你是有一定 iPhone 开发经验的程序员，本书将帮助你连接 iOS 设备与外部硬件。如果你是一位经验丰富的 Mac 程序员，已熟悉 Objective-C，本书将向你介绍 iPhone 编程中与硬件相关的部分。

## 你应该具备哪些知识？

本书假设你已有一些 Objective-C 语言的经验。此外，还假设你对 iPhone 平台有一定了解。如果你是 iPhone 平台的新手，可能对阿拉斯代尔·艾伦的另一本著作 *[学习 iPhone 编程](http://oreilly.com/catalog/9780596806439/)*（O'Reilly）感兴趣。本书不要求或期望读者熟悉 Arduino 平台。本书面向希望学习如何与外部硬件通信的 Objective-C 程序员。不过，如果你完全不熟悉 Arduino 平台，你可能想阅读马西莫·班齐所著的 *[Arduino 入门](http://oreilly.com/catalog/9780596155520)*（O'Reilly）。

#### 警告

本书假设你具备如何构建和部署应用到你的 iPhone、iPod touch 或 iPad 的工作知识。如果你没有 iOS 经验，建议你结合 *[学习 iPhone 编程](http://oreilly.com/catalog/9780596806439/)* 来阅读本书。

## 你将学到什么？

本书将引导你为 iOS 平台开发应用程序，利用 [Redpark 串行电缆](http://www.redpark.com/) 和 External Accessory 框架，将你的 iPhone、iPod touch 或 iPad 连接到任何标准串行（RS-232）设备。除此之外，我们还会探讨一些不那么官方（但可能更有趣）的方法来实现相同目标。

## 本书内容概览

第 1 章

本章面向不熟悉 Arduino 的 Objective-C 程序员。它将向你介绍该平台，并带你完成硬件的“Hello World”范例——闪烁的 LED。我们还将讨论如何使用 Arduino 和你的开发机器之间的串行连接。

第 2 章

本章将介绍 Redpark 串行电缆，并引导你构建两个基本应用。第一个将允许 Arduino 与 iPhone 通信，第二个将允许 iPhone 与 Arduino 通信。

第 3 章

本章将指导你为 iPad 构建一个完整的应用程序，以便通过用户友好的界面实现对 Arduino 的任意控制。

第 4 章

在本章中，我们将把传感器连接到 Arduino，然后使用串行电缆将读数传输到 iPhone。我们还将研究用于绘制图表的 Core Plot 库。

第 5 章

本章介绍 Digi XBee 无线电模块，并指导你将它们连接到 Arduino 和 iOS 平台。

第 6 章

本章探讨将 iPhone 或 iPad 连接到串行设备的其他方法。我们将研究使用 MIDI 协议（iOS SDK 官方支持），以及使用 iOS 设备的耳机插孔作为软件调制解调器。
