## 关于作者

![9781484204375_unFM-01.jpg](img/9781484204375_unFM-01.jpg)

**Sean Liao**（PMP®）于 2000 年在 PalmOS PDA 应用上开发了他的第一个移动应用。他未曾错过任何一次重大的移动技术演进。他曾为 PalmOS、JavaME、Microsoft .NET CF 和 BlackBerry 编写移动代码，并拥有一些诺基亚 Symbian 的开发经验。自 1998 年起，他便是一名经验丰富的 Java 解决方案架构师。

2009 年，Sean 开始使用 iOS 进行编程，同年，他基于多年的移动编程实践经验，采用相同的迁移策略，开始进行 Android 编程。目前，Sean 主要致力于创建 iOS 应用，并将其迁移到 Android 平台作为额外成果。

## 引言

2000 年，我为一个人库存追踪项目开发了我的第一个 PalmOS 移动应用。最初的项目是一个全员参与的团队协作，包括移动开发者、SAP 顾问、供应链领域专家、中间件开发者、QA 测试人员、架构师、业务赞助方等。2002 年，JavaME 强势崛起，随后是 Pocket PC/Windows Mobile。我参与了几个移动项目，在这些项目中，我通过盲目地将 JavaME 代码翻译成 C# .NETCF 移动代码，将移动应用移植到 Pocket PC 平台。这些“翻译”工作延长了整个产品生命周期。由于翻译移动代码的额外成本低得出奇，随着产品生命周期的延长，项目获得了更高的投资回报率。从那时起，我便一直在 JavaME、BlackBerry 和 Windows Mobile 平台之间翻译前端移动应用。
