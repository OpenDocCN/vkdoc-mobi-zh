# 第一部分：准备你的工具

工欲善其事，必先利其器。对于创建软件来说，这也是非常正确的。`Xcode` 是用于编写、编译、调试和构建 iOS 应用程序代码的集成开发环境（`IDE`）。本章的第一部分将指导你完成安装和启动步骤。本书中的所有主题都附带示例代码。你需要使用 `Xcode` 来学习这些示例项目，并且你也会使用 `Xcode` 来创建世界级的 iOS 应用。

对于 iOS 编程，Swift 是为 iOS 8 发布的最新、最强大的编程语言。本部分的章节将为你提供足够的 Swift 知识，以便你能阅读和编写示例代码。你会发现学习 Swift 是对你使用 Java 的一个非常自然的延伸，你一定会很自然地将本书中的代码当作你自己的代码来使用。
