# 前言

Dan Hermes 的这本非凡之作是我见过的关于跨平台移动开发最睿智的著作。他专注于 Xamarin.Forms，同时也为 Xamarin.Android 和 Xamarin.iOS 提供了全面的指南。这使 Xamarin.Forms 置于恰当的背景中，并为那些不仅想知道如何用 Xamarin.Forms 编写代码，还想了解其运作原理及底层机制的人解释了底层的原生代码。我从这本优秀著作的这些部分中学到了很多。

Dan 决定用 C# 编写所有示例。虽然我更喜欢用可扩展应用程序标记语言 (XAML) 来编写大部分 UI，但他的方法确实清楚地说明了实际创建的内容。最重要的是，你可以下载所有 C# 和 XAML 的示例，所有代码示例都是完整的、注释清晰的，并且随时可以运行。

另一个有用的特点是，插图展示了在三个平台（iOS、Android 和 Windows Phone）上运行示例代码的情况。我一直对那些过于复杂、旨在展示作者聪明才智而非将问题阐述得一清二楚的示例感到不满和批评。谢天谢地，Dan 理解这一点，他的示例解释充分且重点突出。

本书从简单入手，但带你了解基础知识，然后进入更高级的主题，例如自定义控件渲染以扩展 Xamarin.Forms 的功能。

代码模式可以帮助你避免重复造轮子，Dan 详细使用并解释了移动开发中一些最重要的模式，包括分层窗口和模态窗口、下钻列表、选项卡、创建导航抽屉以及更多内容。

接着，他处理了移动开发中一个棘手但极其重要的主题：持久化。Dan 涵盖了三种最重要的方法：使用 SQLite、使用 ADO.NET 以及使用带数据访问层 (DAL) 的仓储模式。他深入探讨了数据绑定，并讨论了模型-视图-视图模型 (MVVM)，这可能是移动开发中最重要的模式。

通过对 Xamarin.iOS、Xamarin.Android 和 Xamarin.Forms 的深入覆盖，你将能够很好地决定为你的特定项目使用哪种技术。

Xamarin 新手将从本书中受益匪浅，但也不要低估它对 Xamarin 老手的价值。

—Jesse Liberty
