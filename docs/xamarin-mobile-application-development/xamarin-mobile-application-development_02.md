# 额外前言

我第一次看到 Xamarin.Forms 时，它几乎还难以看出如今这个强大的跨平台框架的影子。我当时正在拜访我们旧金山的办公室，夜深了，Jason Smith（Xamarin.Forms 的创建者）正在忙碌地编写代码，而我正在收拾一些东西准备下班。我突然听到 Jason 发出一声惊呼，脸上露出灿烂的笑容：“想看点真正酷的东西吗？”他问道。“当然想，”我回答。我绕过办公桌去看他想给我展示什么。他的屏幕上是一个 iOS 模拟器，上面显示着一个方框。

我不想打击他对这个显然非常令人印象深刻的壮举的热情——我的意思是，我们说的可是高级东西，一个方框（注：讽刺）——我回答了一些诸如“太棒了。我在看什么？”之类的话。我知道 Jason 的项目；Nat Friedman 和我曾绞尽脑汁试图解决跨平台移动开发的一个基本挑战——UI 层，而 Jason 正在进行一些原型工作，以研究以正确方式实现跨平台 UI 框架的可行性。
