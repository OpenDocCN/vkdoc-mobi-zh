# 摘要

以上就是开始阅读本书所需了解的全部内容：目录、预备知识、免责声明、学习路径、地图和路标。如果你刚接触 Xamarin 平台，欢迎加入社区！如果你对 Xamarin 已有经验，感谢你的阅读，本书同样会为你提供丰富的内容。

我向你保证，阅读本书、理解其中的概念并练习书中的代码技巧，将提升你对 Xamarin 的领悟力，并将你的编程技能推向新的高度。我也希望让你对 Xamarin 平台这一令人惊叹的成就产生更深刻的理解。
