# Firebase 简介

我将向你介绍 Firebase 控制台，以及如何创建一个新项目并将 iOS 应用连接到它。

我们将涵盖 Firebase 是什么，为什么我们选择这个后端而非其他，以及它的优缺点。

我们将一步步指导你直接从 Web 浏览器创建你的第一个 Firebase 项目。我们还将把 iOS 应用连接起来，以便通过其 API 与 Firebase 通信。

在本章结束时，你将能够创建一个连接到 Firebase 后端的新 iOS 应用。如需参考，请访问以下链接的 Firebase 文档：`HTTPS://FIREBASE.GOOGLE.COM/DOCS/IOS/SETUP`。
