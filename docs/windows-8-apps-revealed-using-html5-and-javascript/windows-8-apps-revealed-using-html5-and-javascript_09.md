# 第 1 章 ■ 开始入门

## 本章小结

在本章中，我介绍了本书的结构，并简要预览了后续各章的内容。我还向你展示了一个基本 Windows 8 项目的结构，并创建了一个贯穿全书始终的示例项目。最后，我演示了如何使用模拟器运行 Windows 8 应用程序，并简要介绍了在运行代码时触发 Visual Studio 调试器中断的两种方式。在下一章中，我将开始为示例添加一些功能，并开始使用 WinJS API 的部分特性。首先我会定义一个视图模型，并演示如何将其包含的数据绑定到 Windows 8 应用程序布局上；这是构建易于构建、扩展和维护的健壮应用程序的基本技术。
