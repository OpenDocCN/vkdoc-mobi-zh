## 总结

在本章中，我向你展示了如何使用生命周期事件来响应 Windows 管理应用的方式。我描述了每个关键事件，并向你展示了如何使用应用布局来确保你的应用能够正确地接收和处理这些事件。

当应用被挂起时，必须特别小心地干净地结束后台任务，我向你展示了如何通过请求挂起延迟来获得对此过程的控制权，允许额外几秒钟的时间，以在应用恢复时最小化潜在错误或陈旧数据的风险。

最后，我向你展示了 `activated` 事件如何用于发出不同的请求，包括履行将应用与更广泛的 Windows 平台绑定在一起的契约义务。我演示了搜索契约，但还有其他几种契约，我建议你花时间全面地探索它们。你实现的契约越多，你的应用集成度就越高。

在这本简短的书里，我旨在向你展示核心系统功能，这些功能将帮助你快速起步进行 Windows 8 应用开发。我已经向你展示了如何使用数据绑定、如何使用主要的结构控件、如何处理贴靠和填充布局、如何自定义应用磁贴，以及在本章中，如何控制应用生命周期。以这些技能为基础，你将能够创建功能丰富且富有表现力的应用，并在 Windows 8 开发中抢占先机。

祝愿你在 Windows 8 应用开发项目中取得圆满成功。
