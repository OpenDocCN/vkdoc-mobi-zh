# 使用 HTML5 与 JavaScript 进行 Windows 8 应用开发

## 简介

Windows 8 应用是微软 Windows 8 系统的重要组成部分，为台式机、平板电脑和智能手机提供了统一且一致的编程与交互模型基础。Windows 8 应用的用户体验与以往各代 Windows 应用程序截然不同：它们采用全屏模式，倡导一种简洁、直接且不受干扰的可用性风格。

Windows 8 应用开发彻底颠覆了以往 Windows 版本的开发方式。它引入了全新的 API、新的交互控件，以及一套截然不同的应用生命周期管理方法。

Windows 8 应用可以使用多种语言进行开发，包括 `C#`、`Visual Basic`、`C++`，以及本书的主题——`JavaScript`。Windows 8 是首个接纳 Web 应用开发者技能与知识的 Windows 版本，它让 `JavaScript` 和 `HTML` 成为应用开发中的“一等公民”。

在本书中，我将向您展示如何基于现有的 Web 应用开发知识，使用 HTML 和 JavaScript 来创建 Windows 8 应用。最终开发出的应用将拥有 Windows 体验的浑然一体之感，并能充分利用核心平台功能。

本书将带您快速步入 Windows 8 的世界；通过阅读本书，您将了解如何使用那些定义了核心 Windows 8 体验的控件和特性。

## 关于本书

本书面向经验丰富的 HTML 和 JavaScript 开发者，帮助您利用消费者预览版测试版，抢先一步为 Windows 8 创建应用程序。在 Windows 8 最终版本发布之前，我将为您讲解快速上手所需的概念和技术，助您提升 Windows 8 的开发技巧和知识储备。

### 阅读本书前需要了解什么？

您需要对 `HTML` 和 `JavaScript` 有深入的了解，最好具备开发富 Web 应用的经验。您需要理解 `DOM` API，了解事件的工作机制，并扎实掌握 HTML 元素及其对应的 DOM 对象。

### 需要了解 HTML5 吗？

不需要。在开发 Windows 8 应用时，您可以使用部分 HTML5 JavaScript API，但这并非本书的重点。扎实的 `HTML4` 或 `HTML5` 基础知识，加上丰富的 JavaScript 经验，就足够了。

### 阅读本书需要什么软件？

您需要 Windows 8 消费者预览版和 Visual Studio 11 Express Beta for Windows 8。这两款软件均可从 http://preview.windows.com 下载。

开发 Windows 8 应用程序或运行本书中的示例，无需任何其他工具。

Windows 8 消费者预览版并非最终成品，存在一些稳定性问题。如果您能将 Windows 8 直接安装到配置良好的 PC 上，将会获得最佳体验。如果您尚未准备好升级，也可以在虚拟机上凑合使用。

### 本书结构如何？

本书侧重于构成 Windows 8 应用的关键技术与特性。您已经知道如何编写 HTML 以及如何使用表单元素收集用户输入，我不会浪费您的时间去教授您已经掌握的知识。本书的核心是将您的 Web 应用开发经验迁移到 Windows 8 的世界，这意味着要聚焦于 Windows 8 应用的独特之处。

我采取了一种较为轻松的方式来混合不同主题。除了每章的主线内容外，您还会找到一些关键背景知识，用以解释为何某些特性很重要以及为何应该实现它们。在阅读过程中，我将向您展示编写 Windows 8 JavaScript 应用的惯例，并尽可能多地介绍 Windows 8 的特性。

通过阅读本书，您将了解如何构建一个能够完美集成到 Windows 8 中的应用程序，并呈现与使用其他技术（如 `XAML`/`C#`）编写的 Windows 8 应用相一致的用户体验。

本书是一本入门指南，助您开启 Windows 8 编程之旅。它并非一本全面的教程；因此，我着重介绍了构建 Windows 8 应用的主要基石。有大量信息我无法塞进这本薄薄的册子中。如果您想更全面地了解 Windows 8 开发，Apress 将会在 Windows 8 最终版本发布时，出版我的《Pro Windows 8 Development with HTML5 and JavaScript》一书。

以下各节概括了本书各章的内容。

#### 第 1 章：快速入门

除了对本书进行介绍，我还将向您演示如何为本书通篇使用的示例 Windows 8 应用创建 Visual Studio 项目。我将展示如何在 Visual Studio 中使用 JavaScript 工具，如何在 Visual Studio 模拟器中测试您的 Windows 8 应用，以及如何使用调试器。

#### 第 2 章：数据与绑定

数据是任何 Windows 8 应用的核心。在本章中，我将向您展示如何定义视图模型，以及如何使用 Windows 8 数据绑定将数据引入到您的应用布局中。这些技术对于构建易于扩展、易于测试和维护的 Windows 8 应用至关重要。在此过程中，我还将向您展示如何定义 Windows 8 JavaScript 命名空间、创建可观察数组、使用 JavaScript Promise 以及通过模板生成内容。

#### 第 3 章：应用程序控件

某些用户界面控件是所有 Windows 8 应用的通用控件，无论它们使用何种语言创建。在本章中，我将向您展示如何创建和配置 `AppBar` 和 `Flyout`，它们是这些通用控件中最重要的两个；它们共同构成了与用户交互的主干。我还将展示如何将您的 Windows 8 内容和代码分解成多个部分，以便于管理，并展示如何在运行时将这些部分组合在一起。

#### 第 4 章：布局与磁贴

Windows 8 应用程序的功能延伸到了 Windows 8 开始菜单，后者提供了多种向用户呈现额外信息的方式。在本章中，我将向您展示如何创建和更新动态“开始”磁贴，以及如何为这些磁贴应用徽章。

我还将向您展示如何处理 Windows 8 的*贴靠*和*填充*布局，这两种布局允许 Windows 8 用户并排使用两个 Windows 8 应用。您可以使用 CSS 或 JavaScript 来适应这些布局，我将向您展示这两种方法。

#### 第 5 章：生命周期事件

Windows 对 Windows 8 应用应用了一种非常特定的生命周期模型。在本章中，我将解释该模型的工作原理，向您展示如何接收和响应关键的生命周期事件，并描述如何管理挂起和运行中应用之间的转换。我将演示如何创建和管理异步任务，以及如何在应用程序被挂起时对它们进行*控制*。最后，我将向您展示如何支持 Windows 8 契约，这允许您的应用无缝集成到更广泛的 Windows 8 体验中。
