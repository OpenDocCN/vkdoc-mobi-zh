# 第 1 章 入门

### 需要了解 HTML5 吗？

不需要。在开发 `Windows 应用商店`应用时，您可以使用一些 `HTML5` 的 `JavaScript` 应用编程接口，但这并非本书重点。扎实的 `HTML4` 或 `HTML5` 基础知识，加上丰富的 `JavaScript` 经验，就足够了。

### 本书需要什么软件？

进行 `Windows 8` 应用开发需要两样东西：一台运行 `Windows 8` 的个人电脑和 `Visual Studio 2012`。如果您打算认真对待 `Windows 8` 开发，那么需要购买一份 `Windows 8` 的副本；但如果只是出于好奇，您可以从 `微软` 这里获取 90 天试用版：[`msdn.microsoft.com/en-us/evalcenter`](http://msdn.microsoft.com/en-us/evalcenter)。您只需点击“Windows 与平台开发”部分中的“发布”链接，即可找到所需内容。

您需要创建一个 `微软` 账户才能获得评估版本，但无论如何，要设置为开发者，您也需要这样一个账户。`微软` 账户是免费创建的，如果您还没有，可以按照相关说明进行操作。

**提示** 您可以在虚拟机中运行 `Windows 8`，但为了获得最佳效果，我建议直接将操作系统安装到个人电脑上。

`Visual Studio 2012` 是 `微软` 的开发环境。好消息是，`微软` 提供了一个免费的 `Visual Studio` 基础版本，本书将使用的正是这个版本。它有一个朗朗上口的名字——`Visual Studio 2012 Express for Windows 8`，您可以从 [www.microsoft.com/visualstudio/11/en-us/products/express](http://www.microsoft.com/visualstudio/11/en-us/products/express) 下载安装程序。有多种版本可供选择，每个版本都可用于开发特定类型的应用程序。对于 `Windows 8` 开发，您需要 `Visual Studio Express 2012 for Windows 8`。不同版本的名字极其相似，容易混淆，因此请务必下载正确的版本。

`Visual Studio Express 2012 for Windows 8` 不包含某些付费版 `Visual Studio` 所具备的全部测试和集成功能，但您不需要这些功能来创建 `Windows 应用商店` 应用。在其他所有方面，这个免费版的 `Visual Studio` 功能齐全，能满足您的一切需求。

**提示** 如果您拥有付费版的 `Visual Studio 2012`，请不要担心。您可能用不到您那个版本的所有功能，但本书中的所有说明和示例都能正常运行，无需任何修改。

像安装其他任何应用一样安装 `Visual Studio`。尽管该软件是免费的，但您需要激活它，这需要用到您之前创建的 `微软` 账户。完成这个过程后，您会获得一个可用于激活 `Visual Studio` 的代码。您还需要一个免费的开发者许可证。当您首次启动 `Visual Studio 2012` 时，系统会提示您获取许可证；这只需短短几秒钟，同样需要用到您之前创建的 `微软` 账户。

### 本书的结构是怎样的？

我专注于构成 `Windows 8` 应用的关键技术和特性。您已经知道如何编写 `HTML` 以及如何使用表单元素（例如 `input` 元素）来收集用户数据，我不会浪费您的时间去教您已经知道的东西。

本书旨在将您的 Web 应用开发经验转化为 `Windows 8` 世界中的技能，这意味着要专注于 `Windows 8` 应用的独特之处。

我采用了一种轻松的方式来混合不同的主题。除了每章的主线内容外，您还会找到一些必要的背景知识，用以解释为什么某些功能很重要以及为什么应该实现它们。在此过程中，我将向您展示编写 `JavaScript` `Windows 应用商店` 应用的约定，并尽可能多地介绍 `Windows 8` 功能。到本书结束时，您将理解如何构建一个能够正确集成到 `Windows 8` 中，并提供与使用其他技术（如 `XAML`/`C#`）编写的 `Windows 应用商店` 应用相一致的用户体验的 `Windows 8` 应用。

这是一本入门指南，帮助您开始编写 `Windows 应用商店` 应用。它并非全面的教程；因此，我重点介绍了构成 `Windows 8` 应用主要构建模块的那些主题。有很多信息无法纳入这本薄薄的书中。您可以在我的另一本由 Apress 出版的《Pro Windows 8 Development with HTML5 and JavaScript》中找到更全面的 `Windows 8` 开发介绍。

以下各节总结了本书的各章内容。

#### 第 1 章：入门

除了介绍本书之外，我还将向您展示如何为贯穿本书的示例 `Windows 8` 应用创建 `Visual Studio` 项目。我将展示如何使用 `Visual Studio` 中的 `JavaScript` 工具，如何在 `Visual Studio` 模拟器中测试您的 `Windows 应用商店` 应用，以及如何使用调试器。

#### 第 2 章：数据与绑定

数据是任何 `Windows 8` 应用的核心，在本章中，我将向您展示如何定义视图模型，以及如何使用 `Windows 8` 数据绑定将这些数据引入到您的应用布局中。这些技术对于构建易于扩展、易于测试且易于维护的 `Windows 应用商店` 应用至关重要。在此过程中，我将向您展示如何定义 `Windows 8` `JavaScript` 命名空间、创建可观察数组、使用 `JavaScript` promise 以及使用模板生成内容。

#### 第 3 章：应用程序控件

某些用户界面控件是所有 `Windows 应用商店` 应用通用的，无论使用何种语言创建它们。在本章中，我将向您展示如何创建和配置 `AppBar` 和 `Flyout`，这是这些通用控件中最重要的两个；它们共同构成了您与用户交互的主干。我还将向您展示如何将 `Windows 8` 的内容和代码分解成若干部分，以便于管理应用，以及如何在运行时将这些部分组合在一起。

#### 第 4 章：布局与磁贴

`Windows 8` 应用的功能扩展到了 `Windows 8` 的“开始”菜单，该菜单提供了多种方式向用户展示附加信息。在本章中，我将向您展示如何创建和更新动态的“开始”磁贴，以及如何为这些磁贴添加徽章。
