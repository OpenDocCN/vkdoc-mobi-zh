# Windows Store App 开发入门

`Windows 应用商店`应用可以使用多种语言进行开发，包括 `C#`、`Visual Basic`、`C++`，以及本书的主题——`JavaScript`。`Windows 8` 是第一个拥抱 Web 应用开发者技能与知识的 `Windows` 版本，并让 `JavaScript` 和 `HTML` 在应用开发中成为一等公民。

在本书中，我将向您展示如何利用 Web 应用开发的知识，使用 `HTML` 和 `JavaScript` 创建 `Windows 应用商店`应用。由此开发出的应用，其外观与操作感受将如同 `Windows` 体验中不可或缺的一部分，并能充分利用核心平台功能。

本书将为您开启进入 `Windows 8` 世界的关键起点；阅读完本书后，您将了解如何使用那些定义核心 `Windows 8` 体验的控件和功能。

## 关于本书

本书面向有经验的 `HTML` 和 `JavaScript` 开发者，他们希望抢先一步为 `Windows 8` 创建应用。我将解释您需要快速上手所需的概念与技术，并提升您的 `Windows 8` 开发技巧和知识。

### 阅读本书前需要了解什么？

您需要对 `HTML` 和 `JavaScript` 有很好的理解，最好是通过创建富 Web 应用所积累的经验。您需要了解 `DOM 应用编程接口`，知道事件如何工作，并扎实掌握 `HTML` 元素及其对应的 `DOM` 对象。
