# Objective-C 中的嵌套控制结构

嵌套控制结构是指将一个控制结构放置在另一个控制结构内部，例如将一个循环放在另一个循环内，或在一个 `if` 语句内再放置一个 `if` 语句。这种技术用于处理需要多层次决策的复杂条件和场景。

## 示例——嵌套的 `for` 循环

```objectivec
for (int i = 1; i <= 3; i++) {
    NSLog(@"外层循环迭代 %d", i);
    for (int j = 1; j <= 2; j++) {
        NSLog(@"内层循环迭代 %d", j);
    }
}
```

在此示例中，外层循环 (`i`) 执行了三次，而外层循环的每一次迭代中，内层循环 (`j`) 都会执行两次。因此总共执行了 6 次迭代（3 * 2）。

## 示例——嵌套的 `if-else` 语句

```objectivec
int x = 10;
int y = 20;
if (x > 5) {
    if (y > 15) {
        NSLog(@"两个条件都为真");
    } else {
        NSLog(@"第一个条件为真，第二个条件为假");
    }
} else {
    NSLog(@"第一个条件为假");
}
```

在此示例中，外层的 `if` 语句检查 `x` 是否大于 5。如果为真，则根据 `y` 的值评估内层的 `if-else` 语句。

`break` 和 `continue` 语句提供了在 Objective-C 中改变循环内执行流程的方式。`break` 会终止当前循环或 `switch` 语句，而 `continue` 则跳过当前迭代并继续循环的下一次迭代。嵌套控制结构，如嵌套循环和嵌套的 `if` 语句，允许你有效地处理复杂条件和控制流程场景。掌握这些技术能让你编写更高效、更灵活的 Objective-C 代码，以满足各种编程挑战和需求。
