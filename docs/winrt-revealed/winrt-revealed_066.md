# *独裁者*拥有“`指令`”

最后，是*独裁者*。他的口头禅是：“我就是告诉你事情是怎样的人。”要么听他的，要么滚蛋。没人反驳。没人争论。尽管他的命令在组织中传递时会被微妙地塑造和传达，但他对这些小变化浑然不觉，因为在他看来，他认为指令被逐字逐句地执行了。*独裁者*之所以能存在，是因为没人否认他能出结果这一事实。

*独裁者*的超能力是`指令`，听起来就像这样：*我们现在都要爬上那座山，在所有人都到达山顶之前，我们绝不会停止爬山。开始爬吧。*

然后每个人都开始爬了。没有`辩论`。没有`轻推`。甚至连那些坚决不爬山的人也放下手头的一切，开始愉快地爬山，一边唱着：“我们，爬山的人，多么刺激。”

`指令`的力量是诱人的。它不需要管理`辩论`时那种费力的“赶猫入袋”式工作。它摒弃了构建恰到好处的`轻推`所需的微妙内心对话。`指令`是立即的、无可争议的行动，而如果被滥用，`指令`可能会让你丧命。

当你使用`指令`时，你是在动用自己很大一部分的可信度。你实际上是在说：“闭嘴，去干”，而如果最终的结果出乎所有人的意料，那么你的领导力反而不如从前了。这是一个真实的*独裁者*经常忘记的重要事实。他们相信纯粹的个人魅力——绝对的、毫不动摇的信心——就是你为`指令`提供燃料所需的一切。然而，虽然有些人会因为有人做了决定而高兴，但如果`指令`产生了错误的结果，同样会有准备好干掉你的人。

*独裁者*会被干掉。这是头衔带来的后果。这不是一种血腥的死亡，而是一种职业上的死亡——当你彻底透支了你的可信度时，员工不再听你的，最终离开；或者当你悍然表现出无法胜任时，董事会直接处理掉你。

## 我最喜欢的超能力

这些都是对领导力刻板印象的鲜明例子。虽然你可能会在这些生动的描述中看到自己或你管理方式的某些方面，但更有可能的是，每种特质都已是你的一部分。你有倾向成为一个*绝地大师*，但被逼到墙角时，你又是*机器*。*独裁者*这一面能很好地为你服务，但它会让你的*绝地大师*一面感到……残忍。

解决你的菲尔问题，很可能不是靠单一的权力举动，而是靠所有招数——在适当的时候。当*独裁者*下达一个轻松明确的指令“解雇菲尔”时，*绝地大师*会建议：“菲尔在团队里和谁沟通最好？他们能告诉你关于菲尔的什么？他们如何能促成一次`轻推`？”这引导出我最喜欢的超能力——`辩论`。虽然它不立竿见影，并且对菲尔做最终决定的人是你，但它支持了我的核心信念：一个想法，经过越多人的审视，就会变得越好。
