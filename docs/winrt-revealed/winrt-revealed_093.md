# 避免成为菲兹

菲兹。

菲兹是我手下的一名高级工程师。他是虚构的，但你知道菲兹这种人。他就是那个在 900 万年前写了一段代码，人人都依赖它，但没人确切知道它到底干什么的家伙，因为菲兹懒得多写一行注释……哦，对了，他还是用 Forth 语言写的。

菲兹有自己的办公室，他经常点头。这种点头属于一个相信自己工作固若金汤的人，因为他的技术至关重要。菲兹惹恼了很多人，但当问题爆发时，菲兹总能解决问题，因为他精心隔离了一条关键路径，这条路径只属于他一个人。

每年，菲兹和我坐下来，我会提出他的绩效评估。我会先询问他的抱负，而他则用含糊的点头回应。

- *听起来不错，老板。*
- *好的，老板。*
- *当然，老板。*

菲兹根本没在听我们讨论，因为菲兹连续 12 年都在听这些绩效评估的陈词滥调。他相信自己是不受影响的。

菲兹的做派是一种必然会导致自己慢慢变得无关紧要，更重要的是，最终会失业的万无一失的方法。

## 了解你的位置

在我解释为什么我们每个人心里都住着一个小菲兹之前，让我们先退后几步。健康企业的定义是一个正在成长的企业，所谓成长，是指它每年赚的钱越来越多。公司创造这种增长的方式数不胜数，但你永远不应忘记的一条基本商业物理定律是：“企业成长，其员工亦须成长。”
