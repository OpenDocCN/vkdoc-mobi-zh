# 议程识别

我讨厌开会。

*每个人都*讨厌开会，因为我们经历过太多糟糕透顶的会议，以至于我们现在走进会议室，抱臂坐下，想着：“好吧，还要多久这个会议才算正式浪费我的时间？还要多久*这个*会议才会变得无聊透顶？”然后它就真的无聊了。时间被浪费。产生了一堆空话，每个人都围坐在桌子旁，想知道什么时候会有人来制止这种疯狂。

如果你曾在会议中感到沮丧——如果你坐在那里想知道为什么这些人，这些拿着高薪推动公司前进的管理者，就是不能做或说出那些显而易见的事——那么请继续读下去。

每当你走进一个可能变得糟糕的会议时，你需要一项基本技能。无论你是参与者还是会议主持人，这项技能都很重要。这项技能叫做*议程识别*。

简单来说，议程识别是辨别以下内容的能力：

1.  典型的会议角色以及与会者如何扮演这些角色。
2.  解释这些不同的会议角色想从会议中获得什么。
3.  如何利用这种理解，以最快的速度离开会议。

## 退出会议技巧 #1：识别会议类型

离开会议的第一步是识别这是什么类型的会议。会议议程会有帮助，但由于大多数会议没有议程就开始了，你得靠自己。你很可能参加的要么是信息传达会，要么是冲突解决会。

在信息传达会中（各位，这里有个大惊喜），信息被传递出去。想想你最爱的季度全员大会。员工会议。任何有固定默认议程的会议。这类会议中有两种参与者：发言者和倾听者。

这些会议中的角色和议程很简单。发言者在发言，倾听者在倾听。明白吗？除了信息传递，没有其他需要解决的问题。信息传递得越快，大家就能越早回去工作。
