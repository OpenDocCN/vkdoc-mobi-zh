# 排版结果

当三位手持链锯的伙计出现准备清理土地时，这种态度让我产生了独特的好奇心。他们毫不费力地开始了工作，显然已经掌握了迭代的窍门，但同时也展示了“涓滴理论”最后一个也是最重要的组成部分：混搭操作。

具体过程是这样的：一个人负责砍伐并拖拽灌木到火堆，另一个人负责砍树，第三个人则修剪倒下的树木。这样进行了一段时间后，他们会全员轮换。现在，原来拖拽的人变成了砍伐者，砍伐者变成了搬运木材的人，修剪者则变成了堆叠者。午餐时，我坐下来问道：“你们什么时候换工？”

“当我们觉得无聊的时候。”

太棒了，真是美妙的涓滴理论。这有多酷？如果你正在处理一项极其困难或极其枯燥的任务，并且发现自己因无聊或困惑而思维受阻，那就停下来去做别的事情。停下来的好处是惊人的。

首先，停下来等于给了那个“批评家”当头一棒。要知道，他也是你脑海中那个说“呃，如果我们不在这件事上拼命努力，就完蛋了”的声音。而当你不想做却坐在那里硬啃不可能完成的任务时，时间越长，他的声音就越大。

其次，停下来去做别的事情对你和你的大脑来说都很有趣。它打破了你正在做的任何任务的循环，将你的灰质细胞指向一个全新的问题——而你的大脑热爱新事物；它会充满活力地吸收新东西，这会给你的精神脚步注入活力。

第三，也是最重要的一点，即使你停了下来，你的大脑也足够聪明，会在后台继续处理那个不可能完成的任务。这就是为什么我们会在淋浴时找到那么多灵感；你停下来，让大脑漫步，而你的大脑很聪明。你的大脑知道在两天内重写那个功能有多重要，无论你是否意识到，它都在一直处理那个功能。

“等等，等等，等等。兰兹，让我把话说清楚。你的意思是，当我面临一个迫在眉睫的不可能完成的最后期限时，我应该停止交付我的工作成果？”

我想说的是，当你面对内心的激烈斗争，正与那个不可能完成的任务较劲时，是时候选择另一场战斗了……一场不是战斗的战斗。

## 熵增永远会赢

我的生活似乎是一系列旨在稍微整理我世界的无尽任务。从整体上看，这些任务代表了大量的工作。但与我所在世界的小小角落中实际存在的熵增相比，这些任务代表着徒劳的努力。

事实是，你的世界变化的速度永远快于你能跟上的速度，你可以从两个不同的角度来看待这个事实：

*   我相信我能控制我的世界，通过积极的任务管理、个人目标以及*我能行*的态度，我将成功地完成不可能的事情。我真棒！

或者……

*   我知道无法控制世界，但我会通过不断改变自己来灵活地驾驭熵增。

驾驭熵增需要自信。这并非托尼·罗宾斯式的自信；这是一种通过不断让自己适应不可能之事而赢得的个人自信。
