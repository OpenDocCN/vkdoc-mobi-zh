# 说“不”

在你做管理者的第三年左右的某个时候，管理小精灵会在一股香甜的黑烟中出现在你的办公室里。会有三个小精灵，其中一个会拿着一顶华丽的黑色高顶礼帽。

“你是勒罗伊·麦克管理者吗？”

“我是。”

小精灵们笑了。“恭喜你，你已经成功通过了三年的管理考验，我们是来给你奖励的。但首先，问你一个问题：你看过《蜘蛛侠》吗？”

“第一部还是续集？”

“第一部。”

“看过。”

小精灵们又笑了。“你认为《蜘蛛侠》的主题是什么，勒罗伊·麦克管理者？”

“嗯，呃……生活真他妈艰难？”

奇怪的是，小精灵们没有笑。“不，再猜猜。这很重要。”

“好吧，嗯。嗯……彼得的叔叔说过一句他们一直唠叨的话……*哦，我知道了……*能力越大，责任越大。”

小精灵们欢呼起来，那个拿着礼帽的精灵飞到你身边，把帽子放在你的膝盖上。帽子很柔软，而且出奇地温暖。拿着礼帽的小精灵抬头看着你，咧嘴一笑：“当你想让别人知道你是谁的时候，你就戴上这顶帽子。”

“那我是谁？”你低头看着帽子，注意到前面有巨大的白色粗体字母。上面写着：

`我是老板。`

你的嘴角慢慢咧开，露出了笑容，你发现帽子上有股你妈妈新烤面包的淡淡味道。那种气味总是给你一种奇怪的自信，你知道，每当你戴上那顶帽子，你就会被那种自信所充满。



三个小精灵跳上半空，咯咯笑着。“祝你好运，勒罗伊·麦克经理，好好利用你的帽子吧！”又是一阵笑声。又一股黑烟飘过，它们便消失无踪了。

你缓缓把帽子举到面前，盯着上面白色的方块字母，沉浸在这顶帽子带给你的权力感之中，然后你把它戴上了。

你大步走出办公室，从未想过为什么小精灵们笑得那么厉害，因为，嘿，你是老板。第一个人看见你在自信的氛围中走过，而当你转过拐角后，你听不到他们的窃笑，因为，再说一次，你是老板。

他们在笑，是因为虽然他们知道你是老板，但他们能看到帽子的另一面，上面写着：

……暂时是
