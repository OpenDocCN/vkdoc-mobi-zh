# 第 1 章：别当刺头

为网络写作的美妙之处在于确实没有计划。我有幸在精神上随意摸索任何主题。渐渐地，这些主题聚焦到了工程管理上，随着每篇文章的发表，我收到越来越多的邮件问：“书在哪里？”是啊，我一直想出版一本书，但有个问题——卖点是什么？*做个好经理？呃呃呃呃呃。* 我需要一个引人入胜的真相，它能优雅地将我所有的沉思串联起来。

时光倒流回互联网泡沫破灭中期。我们，一家濒临倒闭的初创公司的快乐团队，正在……大量饮酒。公司总部附近有各种酒吧，每家都有独特的用途。有家简陋酒吧非常适合裁员后的聚会，酒水便宜，如果你想发泄“我其实没那么没用”的怨气，你可以和趴在吧台上那个没牙的水手干一架，或者和刚解雇你的家伙打一架。

街那头是家英式酒吧。啤酒更好，有威士忌可选，还有能吃的食物。我们就是在这里，对我们当前正在经历的、让我们在三年内滑向无足轻重的组织抽搐进行哲学探讨。

我们现在就在那儿。我们喝得酩酊大醉，因为公司刚刚被卖给了一家无名上市公司，这家公司将迅速瓦解那个我们曾为之呕心沥血的公司。每个人都知道我们迟早会落到这步田地，但没人料到自己是最后一个坚守者。更没人料到 CEO 会出现。
