# 你的直觉百分之百错误

与你想要领导的人完全个性断裂，就像华莱士的情况那样，虽然罕见，但为了弄清楚如何解决这个问题，让我们假设这是最坏的情况。

在与员工进行正常的相互了解时，我第一个想回答的问题是：“他们想要什么？”他们当前工作的核心动机是什么？他们是为了晋升而努力吗？他们只是在摸索工作吗？他们是否迷茫？他们是否安于迷茫？

理解核心动机并不能让我完全了解一个人，但它给了我一个起点。当我知道他们想去哪里，我就可以开始琢磨如何帮助他们到达那里。不快乐而迷茫的人想要立刻采取行动。寻求晋升的人热切地寻找机会，并且会为了得到机会而承担一切。然而，这是华莱士，当我轻快地问“什么让你感兴趣，华莱士？”时，华莱士茫然地盯着我。

放弃你能巧妙诱使华莱士说出他的秘密的想法。你所有自然而然的想法，对一个真心相信你是外星人的人来说，都不会奏效。你的首要任务不是理解核心动机，而是基本沟通。

在任何沟通存疑的情况下，我严重依赖澄清。每当我说了一些可能含糊不清的话，我会问：“你听到了什么？”反过来，当我在倾听时，如果主题或意图不够清晰，我会重述：“好的，我听到的是……”

听起来像是这样：

*我：* ……这周末来了新的需求，但詹妮弗仍然希望周四看到一份初步规格说明。华莱士，*你听到了什么？*

*华莱士：* 詹妮弗周四要一份规格说明。

现在，如果这是哈罗德或斯坦，我们就完事了。口头承诺已经做出，我们可以继续了，但这是华莱士，我不能做任何假设。

*我：* 是的，周四的规格说明。那对你的工作意味着什么？

*华莱士：* 我原计划周一完成文档审查，周二周三搭建测试数据。如果詹妮弗周四需要规格说明，我就没时间做这些了。

*我：我听到的是*，你有一堆工作需要重新安排时间，对吗？

*华莱士：* 对。

*我：* 如果我们重新安排，你认为你能完成规格说明吗？

*华莱士：* 也许吧。

我知道这感觉像是消极攻击的奥运会，但我发誓华莱士就是这样思考的。在他的世界里，工作承诺从来不是暗示性的，必须口头说明。我们需要反复沟通，直到他说出：“我将把本周所有计划好的工作重新安排到下周。我会向文档和 QA 团队传达这些变更的影响，我有中等信心，在周四下班前交付修订版规格说明的初稿不会有什么问题。”

这很迂腐、烦人且低效，但当个性断裂时，你不知道如何沟通，所以这就是你的起点。通过练习，你会学到独特的交往规则。你会了解到人们用来描述他们快乐与不快的词汇和想法。你会学会当人们完全不知道你在说什么时，他们使用的视觉和口头线索。随着时间的推移，你会为这个与你截然不同的人构建起一幅心理地图。

与这些“外星人”互动的感觉永远不会自然。它总是显得刻意而陌生，但练习会为你提供一把理解他们思考方式的钥匙，以及一幅或许能让你洞察他们想要什么的地图。
