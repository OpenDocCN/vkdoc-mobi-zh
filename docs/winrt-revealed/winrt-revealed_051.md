# 坠落之感

坠落带来的感受与教训的规模不成比例。你体验到的不仅是失败感，还有巨大的非理性失望——你不再是一个你从未接触过任务的专家——这很愚蠢。关键是——你现在正好有足够的经验，可以理解掌握一项你曾以为能神奇地即兴发挥的任务所需付出的实际努力。

华莱士式的坠落在新晋领导者中很常见，而工程师普遍的内向倾向会加剧这种情况。你会发现不仅你领导的某个人不跟随你，而且你和这个人之间存在着彻底且完全的个性断裂。你所有惯用的社交手段都失效了。他们对你机智的妙语回应以茫然的眼神，而当他们开口说话时，仿佛你说的每一个字他们都没听见。

这是一个令人沮丧的发现，因为你成为领导者的部分原因在于你天生且被证明有效的人际交往能力。但这也是一个令人羡慕的发现，因为在一个充满人的星球上，大多数人跟你不一样，而且你的第一个“华莱士”绝不会是最后一个。
