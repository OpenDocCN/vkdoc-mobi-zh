# 排版后的 Markdown 文档

大公司里百分之九十五的人对暗地里的公司政治斗争一无所知，不清楚自己会受到何种影响，也不知道自己六个月后是否还能保住饭碗。你的团队将根据你与他们的沟通方式来评判你作为经理的表现。这不仅意味着要抽出时间召开季度全员大会，还意味着要理解他们需要听到什么，并能以他们理解的方式表达出来。

## 章节

### 弗雷德讨厌外出会议

管理界有一套自己借用的权力词汇，用来赋予自身一种身份认同感。这份清单无穷无尽且颇为有趣。当这些词被说出时，它们被以一种刻意的方式表达，让你不禁敬畏地琢磨：“那是什么意思？” 但你又不敢问，生怕显得像个傻瓜。

今天的词：*外出会议*。外出会议是一种……会议。它有一些特定的特征，但它实际上就是与一群人开会，而且很可能名副其实，是在别处——也就是“外出”举行。

既然你明白了它是什么，那我们来理解一下为什么你可能会讨厌它。

## 我为何要与弗雷德针锋相对

外出会议存在的原因很简单：你，作为团队的领导者，需要完成某些在楼内常规条件下难以轻易完成的关键工作。这有点可悲。当团队只有 20 人时，我接下来要描述的三种不同的外出会议都会……自然地发生。弗雷德会站在办公室中间说：“好了，我们需要一个新的 UI 框架。我来做，谁想跟我唱反调现在就提出来。”

于是你与弗雷德针锋相对。你们争论、辩论。菲兹和菲尔也加入进来，在至关重要的 17 分钟里，你们彻底改变了产品的 UI 架构。

当组织规模扩大到一定程度（每个团队情况不同）时，过去自然发生、用于完成文化和战略工作、并允许做出重大决策的那种跨领域沟通活动，再也无法进行了。团队再也无法环顾四周了解每个人的状态，因为人太多了。

`时代精神` 变得淡薄。随机的走廊纠错不再发生，因为相关人员碰不到面。这很可悲，尤其是对于那些还清楚地记得站起来与弗雷德争论的人来说。你需要重新创造一个空间和场所，让团队能够凝聚，让战略得以制定，或者让你踏上一次史诗般的旅程。

## 我们是谁，我们需要什么，以及我们的史诗旅程

你发起外出会议的原因会因团队而异。以下是三种我认为需要采用外出会议的具体场景，但并非仅限于此。

### 我们需要了解我们是谁

如果你熟悉我的文章，你会知道我认为在头 90 天内你根本无法真正了解一个团队到底发生了什么。在最初三个月里，你会有一些令人欣慰的清晰时刻，但如果没过上一段时间，你不可能了解所有动态。现在，把这种初期的困惑乘以每个在过去九个月内入职的新员工。

在快速扩张时期，团队成员不一定有时间停下来互相了解，因为他们一来就注意到：“哇，每个人都忙得不可开交，所以我也必须赶紧忙起来。” 他们认识周围人的正常本能被埋没在了“被认作也是个大忙人”的目标之下。

你需要一个外出会议，不是为了解决战略性的产品问题，而是为了让团队成员暂时远离匆忙，去彼此了解。我即将描述的每种外出会议都会促进社交，但团队需要了解自身，这本身就值得为此专门举办一次外出会议。

### 我们需要新的方向和/或减少灾难

某件重要的事情出了大问题。要么是灾难接连发生，而常规的发现和纠正机制失效了；要么是一切看似正常，但我们并未取得成功——无论在这个公司阶段“*成功*”意味着什么。无论哪种情况，现状都对公司构成了真正的威胁。

这种外出会议的目的是深入集思广益。团队的任务是发现和完善想法，提出测试这些想法的实验方案，并最终挺身而出，将这些想法带回到“大本营”（公司）去执行。

### 我们即将开启一段史诗旅程

没出什么问题；只是到了该开始做点什么的时候了。这最后一种外出会议也可以称为启动会议。你大致知道想做什么以及何时去做，但你需要所有负责执行这件事的领导就前进方向和原因达成一致。

这种外出会议是一个对齐会议。战略可能会在这次会议中发现，但这并非主要目标。你们要共同将大家指向正确的方向（*这是我们要去的地方*），并明确行动的紧迫性（*这就是我们要去那里的原因*）。

### 一场具有特定特征的会议

虽然外出会议只是一个长会议，但它具有某些与你日常会议不同的特征。我来逐一解释，并且我要带上弗雷德，因为他能毫不含糊地告诉我们他讨厌什么。

#### 定义上，你无法邀请所有人

记住，你需要外出会议的首要原因就是团队规模已经大到大家都被（希望是必要的）战术性工作所吞噬。他们没时间停下来思考，因为从踏进办公室的那一刻起，就有开不完的会、回不完的电话、邮件和面试。每个人都沉浸在工作中。是的，个人偶尔能挤出一小时喘口气，但作为一个整体，团队没有时间去弄清楚自己在想什么，或者为什么这么拼命工作。

根据你外出会议的目标，你需要挑选那些既有能力又愿意解决眼前难题的人。如果你试图了解团队，你就邀请你想了解的所有人。如果你想减少灾难，你就邀请那些制造灾难的人和有能力解决灾难的人。你需要能够环顾房间，心想：“这些人就是能解决问题的人。”

*弗雷德讨厌这一点。* 他说：“嘿，为什么我们要搞排外？我希望每个人都能对影响公司的重要决策发表意见。我们都是股东，都应该有投票权。”

弗雷德，那种邀请所有人的会议是有名字的。它叫“全员大会”。如果全员大会不是你公司常规会议的一部分，我能理解你为什么生气。但是，你的全员大会有 120 个人，我问你，你上次看到 120 个人高效地提出、辩论然后做出决定是什么时候？外出会议并不是忽视意见的机会；外出会议是一个选择一群最能代表公司去解决当前重大难题的人的机会。是的，筛选过程很艰难。

#### 每人都有展示，或至少发言

一旦你有了初步的名单，问问自己：我会要求每位受邀者做什么合适的演示？我的经验法则是，每个参加外出会议的人都需要提交一项成果，这通常意味着他们需要站出来进行演示。这种做法让我明白两件事。首先，如果面对手头的问题，我想不出想让这位演示者谈些什么，那为什么邀请他？其次，我开始看到重复。*嗯，莎拉和弗兰克都很擅长谈论我们缺乏设计流程的问题。我真的需要他们俩都来吗？*
