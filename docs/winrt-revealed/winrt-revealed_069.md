# 管理者失控

我是认真的。有些管理者完全被权力冲昏了头脑，如果你正在为这样的人工作，我真的很同情你。你是疯狂小镇的居民，这意味着你永远不知道接下来会发生什么莫名其妙的破事，这糟透了。

管理者并非生来就疯狂。这是一种后天习得的特质。本章将探讨一个最佳策略，你可以用来应对你的经理和你自己，以避免踏上疯狂小镇之旅。我们先从一个与你经理有关的故事讲起。

你正在键盘上愉快地打字，努力开发下一个伟大的产品，这时你的老板走进来说：“嘿，你能做个‘吉兹·弗利比特’项目吗？”

“呃，我们不是应该先完成‘弗拉布贾姆’吗？我们才刚刚开始做呢。这还需要不少时间。”

“哦，对对，我们还在做弗拉布贾姆，但我需要你为吉兹·弗利比特做个原型，而且我要在两天内完成，好跟高管们开会。”

“好……吧，你是老板。”

“没错。我是老板。”

两天过去了，你的团队为这个原型项目倾注了全部心血。就像所有探索一样，你们发现每一步发现所需的时间都比预期的要长三倍。最终的原型传达了想法，但达成这个结果的过程让团队筋疲力尽，并且确信完成剩下的工作将花费很长时间。

当你的老板走进你的办公室时，你总结道：“这就是成果。看起来不错，但要完成还需要一段时间，而且我们现在在弗拉布贾姆的工作上已经严重落后了。我们能回去做那个吗？”

她眯起眼睛，指尖滑过她大礼帽的前沿。她点点头，凝视着，*“好的，这太棒了。* 我们要同时做这个 *和* 弗拉布贾姆，*而且*还要按原定时间表完成！加油，我们！”她转身离开了房间，你的办公室里留下了一股淡淡的面包味。

我来总结一下。你的老板刚刚选择了一个工作量最大、成功可能性最小的方案。你完蛋了。但你可能觉得是老板失控了，实际上你也是这场灾难的共犯，因为你没有做一件简单的事：你没有*说“不”*。

## 失控

管理者失控，并非仅仅因为小精灵带来了那顶大礼帽，而是因为与他们共事的人忘记了看帽子的背面。记住：

正面：我是老板。

背面：……暂时是。

管理就像那顶大礼帽一样，是个神话。作为员工，我们相信它的存在，所以我们以不同的方式对待这些管理层。我们假设他们才是能做决定的人。当团队卡在一个难题上时，我们会聚集在经理的办公室里，陈述我们的情况，然后经理点点头说：“走那条路！”很多时候，我们太高兴能绕过这个难题，以至于甚至不去质疑这个决定是否正确。“他戴着大礼帽，所以他肯定是对的！”

不不不不。还有？不。

当管理者做决定时不再受到质疑，他们就失控了。当团队停止质疑权威时，经理就开始慢慢相信自己的决定总是好的。虽然永远正确的感觉很棒，但从统计上看是不可能的。世界上最富有经验的经理也时常会做出糟糕透顶的决定。优秀的经理学会了如何有尊严地从自己的决定中恢复过来，但更重要的是，借助团队的帮助。

说“不”迫使一个想法用事实来为自己辩护。它迫使那位受大礼帽影响的经理停下来思考。是的，我知道那顶大礼帽可能令人生畏，而且没错，他是签支票的人，但每次你任由你的经理带着未经审视的盲目热情向前冲时，你只是在强化他“自己永远不会错”的认知。这简直就是通往疯狂小镇的单程票。

## 恢复控制

一个坚实有力的*“不”*可以指向任何方向。即使这很难做到。

我的团队刚刚在董事会上被高管团队痛批了一顿。我们一直因当前产品的销售而飘飘然，认为自己不会犯错，所以我们对下一代产品的演示准备得很不充分。我们以为既然当前版本表现这么好，高管们会忽略我们对未来的含糊其辞，但他们没有。

问答环节起初还算愉快，但三个问题之后，当高管们清楚地意识到我们是在即兴发挥，并且那单薄的演示背后并没有什么宏伟计划时，他们就开始火力全开了。从这种挫败中抽身的唯一方法就是：“嗯，看来我们得安排一次后续会议了。”

团队进入了紧急模式。我们需要一个产品路线图，一周内就要，还需要重建高管团队对团队的信心。头脑风暴开始时，每个人都惊魂未定。我们从“被选中的团队”变成了“连路线图演示都搞不定的团队”。因为惊魂未定，房间里蹦出的想法都畏首畏尾。这些想法旨在安抚刚刚训斥过我们的人。虽然我的信心也有些动摇，但我知道，是时候再次说“不”了——对他们，也对那个想要快速周转的高管团队。

“不，我们不能接受平庸。不，没人想要我们做‘我也一样’的设计。而且，不，除非这个路线图能激励房间里的每一个人，否则我们就不算完成。”

现在，我站在办公室里就激励人心的产品路线图发表演讲，与一位因被高管痛批而徘徊在疯狂小镇边缘的经理之间，差别其实很微妙，但这正是艺术所在。说“不”就是喊“停”，在一个所有人都热衷于永不停歇的竞争环境中，有能力策略性地选择何时停下来，正是一位经理敢于挑战常规的标志。

## 永远别信小精灵

那顶大礼帽并非看上去那么简单。是的，黑色天鹅绒般的优雅意在给予你自信，但请记住——帽子的背面是一个威胁。它不是在背面，因为小精灵们无法把整条信息都放在正面。它在背面，是因为它们既不想让你活在疯狂小镇，也不想让你因“自己可能因为一个糟糕透顶的决定就丢掉工作”的现实而瘫痪。它们希望你拥抱大礼帽赋予你的自信，因为那会帮助你为自己、为团队、为产品做出伟大的决定。有些伟大的决定是纯粹靠运气，有些则是因为你知道自己在做什么。然而，你也会做出一些糟糕的决定，有些你能挺过去，是因为它们没那么糟，而有些则纯粹是靠你大礼帽带来的那股牛劲儿。



## 然后，你将做出一个重大而艰难的决定，并会想起：“能力越大，责任越大。” 作为管理者，你负责做出伟大的决策，而实现这一目标的最佳方式，就是尽可能让整个团队参与到每一个决策中。

你的团队综合起来比你更聪明，仅仅因为人多。更重要的是，通过让他们参与决策过程，并打造一个让他们敢于说“不”的团队，你正在建立信任。

一个信任你的团队会为你着想。他们绝不会袖手旁观，看着你兴高采烈地冲向“疯狂小镇”，盯着你的后脑勺想：“下一个戴高顶礼帽的会是谁？”
