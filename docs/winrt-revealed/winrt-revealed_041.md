# 兰兹测试

当 Borland 公司首次裁员时，我已在质量保证岗位工作两年。我不记得当时曾困惑裁员是什么，也不记得曾担心是否会失去工作。我记忆犹新的是，应用部门的高级副总裁在楼内巡视一圈，召集产品团队，然后就在走廊里直截了当地向我们说明了裁员的情况：这次裁员是怎么回事，哪些人受到影响，以及何时执行。此前我从未与副总裁罗布打过交道，事后想来，后来也再未真正与他交流过。但我至今对这位老兄心存好感——因为在压力时刻，他选择了坦陈真相，而非含糊其辞。

一个成功的组织是由被管理者粘合在一起的层层人员构建而成。每一层负责一项广义任务，无论是工程、QA 还是市场营销。层与层之间，管理者扮演着翻译角色——在上下层之间双向传译。他了解下属的诉求，也明白上级的期望，并能在这两者出现分歧时成功斡旋。

他之所以能游刃有余，是因为掌握了两个问题的答案。问题一：我从哪里来？能够与下属产生共鸣，源于对其工作的深刻理解，这让你能与他们说同一种语言。问题二：我要去哪里？为上司规划下一步重大行动就是对他的激励。这会让他处于不安的境地，努力分辨公司主要影响者那难以捉摸的政治动机。这或许不是他与生俱来的技能，但他绝不会停止尝试——因为他清楚自己的目标方向。他有一张由自身动力绘制的地图。

你为何要在意老板想成为软件副总裁？因为他的成功就是你的成功。如果他不懂得这个道理，那他可能是个糟糕的上司。
