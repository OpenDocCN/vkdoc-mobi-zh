# 总结

在本章中，我们学习了如何开始为 tvOS 开发应用。通过使用 OpenWeatherMap API，我向你展示了如何开发一个显示所选城市天气状况的应用。你可以轻松扩展此功能以显示天气预报。为此，你需要修改请求的基本地址，将 `weather` 替换为 `forecast`。然后，将 JSON 响应映射到相应的 C# 对象，并在 UI 中显示其属性。为了显示天气预报，你可以考虑使用不同的控件，例如 Table View。这将是你练习应用本章及之前所有章节所学知识的一个好机会。

本章也结束了这本书。我希望你喜欢所有的内容，并且我坚信，所呈现的材料将帮助你快速入门开发适用于 iOS、watchOS 和 tvOS 的移动应用。尽管我没有涵盖每个平台的许多高级功能，但你应当能够在 Apple SDK 文档的帮助下使用这些功能，因为正如我在第 2 章中解释的，相应的 API 已由 Xamarin.iOS 映射到 C# 类和接口。因此，我祝你在开发中好运，并期待在 Apple Store 中看到你的应用。
