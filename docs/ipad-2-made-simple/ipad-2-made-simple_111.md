# 从外部硬盘备份中恢复

从外部硬盘备份中恢复需要多个步骤。首先关闭 `iTunes`，然后按照以下各节所述步骤操作。

## 步骤 1：将资料库复制回电脑

逆向执行前述步骤，将资料库从外部硬盘拖放回电脑。为简便起见，建议将资料库放回原 iTunes 资料库的相同位置。如有需要，可参考前面提到的默认位置。

## 步骤 2：打开 iTunes 并选择资料库

要打开新复制的资料库，需以特殊方式打开 `iTunes`：

1. 在 Windows 上，按住 `Shift` 键的同时打开 `iTunes`。在 Mac 上，按住 `Option` 键。请务必保持按键按下，直至看到右侧的小窗口。如果过早松开，`iTunes` 将正常打开。

   ![images](img/p717-01.jpg)

2. 选择 `选取资料库`，然后找到刚从外部硬盘复制的资料库。
3. 点击 `打开` (Windows) 或 `选取` (Mac) 以打开 iTunes 资料库文件。

按照这些步骤，通过打开从外部硬盘复制的文件，即可恢复 iTunes 资料库。
