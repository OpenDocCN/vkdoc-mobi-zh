# 通过电子传真服务发送传真

如果你不在家或办公室，又想快速打印或发送传真，通过电子传真服务发送文件是个很棒的技巧。

**注意：** 要执行此操作，你首先需要注册一个电子传真服务，例如 [`www.efax.com`](http://www.efax.com)，这样你就可以将传真作为电子邮件附件发送。

拥有电子传真帐户后，你可以按照以下步骤将文档作为电子邮件附件发送到传真机：

1.  在任何应用中，选择`共享` ![images](img/U001.jpg)`电子邮件演示文稿`。
2.  选择`PDF 格式`。
3.  按照电子传真服务要求的格式，在邮件地址中使用传真号码。某些服务的格式如下：`(传真号码)@faxsend.com` 或 `3865551212@faxsend.com`
4.  邮件发送后，你的传真服务会将其转换为传真，并发送到相应的传真机。
