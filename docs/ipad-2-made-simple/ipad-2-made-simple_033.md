# 在 iPad 锁定时显示音乐控制

即使你的 iPad 处于锁定状态，你仍可以控制音乐。只需（在 iPad 锁定时）双击 `Home` 按钮，用于调节音乐的控制选项便会出现在锁定屏幕的顶部。无需解锁屏幕再前往 `iPod` 应用查找控制选项。

![images](img/p242-01.jpg)

请注意，右侧图片中的屏幕仍处于锁定状态——但音乐控制选项已显示在顶部。你可以暂停、跳过、返回上一首歌曲或调节音量，而无需实际解锁 iPad。

#### 家庭共享

今年 iPad 新增的功能是可通过苹果专有的家庭共享功能共享你的家庭 iTunes 资料库。

本质上，家庭共享允许你在 iPad 上浏览和播放家庭资料库（通常位于你的台式电脑或网络驱动器上）中的任何内容。

此功能的一大优势是你不受 iPad 有限存储空间的限制；只要连接到家庭网络，此功能就能让你访问整个资料库。

请按照以下步骤启用家庭共享：

1.  点击 `Home` 页面上的 `设置` 图标。
2.  触摸左侧栏中的 `iPod` 图标。
3.  在右侧的 `家庭共享` 标题下输入你的 `Apple ID` 和 `密码`。请确保输入的网络中已启用家庭共享的电脑所使用的 Apple ID 和密码相同（参见第 29 章）。
4.  触摸键盘上的 `完成` 按钮。
5.  启动 `iPod` 应用，注意左上角出现了一个名为 `资料库` 的新类别。
6.  触摸 `资料库`，查看可供浏览的网络资料库。
7.  选择你的 `网络` 资料库，即可在 iPad 上浏览和播放家庭网络中的任何项目（参见图 9–4）。
