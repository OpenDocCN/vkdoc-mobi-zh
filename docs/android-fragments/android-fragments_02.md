## 引言

移动应用开发如今依然是最热门的话题之一。多年来消费者一直受益于智能手机，现在越来越多的公司也加入其中。有些开发者是全新学习移动开发，而另一些则来自 Web 开发或 PC 编程背景。无论如何，制作运行良好的移动应用需要学习一些新概念和技能。对于 Android 而言，碎片是对 Android 应用至关重要的主题之一，但掌握起来并不容易。一旦您理解了碎片工作的内部细节以及它们的使用方式，您会发现设计和构建 Android 应用要容易得多。

碎片是用户界面和逻辑的概念性容器，可以像构建块一样轻松组合到您的移动应用中。Android 并未放弃`Activity`类，但现在活动通常由碎片组成。这使得构建支持不同设备类型、尺寸和方向的应用变得更加容易。

本书《Android 碎片》是我们关于 Android 的第六本书。在前四本以《Pro Android》名义出版的书中，我们涵盖了许多 Android API，从基本视图到广播接收器、触摸屏、服务和动画。在《Expert Android》中，我们探讨了更高级的 Android API 以及 Android 的高级调试功能。

《Android 碎片》实际上是《Pro Android》书中的一个子集。它专门聚焦于碎片，为您提供其他书中无法找到的细节。它不仅涵盖了基本碎片，还包括对话框碎片、偏好设置碎片和进度对话框碎片。它涵盖了兼容库，以便您可以在最初不支持碎片的旧版 Android 上使用碎片。阅读本书后，您将毫无畏惧地将各种碎片融入您的应用中。

## 本书适合你吗？

本书面向那些对 Android 和 Android 应用基础有良好理解，但需要或想要深入了解碎片的移动开发者。优秀的架构师了解材料。要构建出色的移动应用，优秀的移动开发者需要了解碎片。如果您不了解，或者不确定自己是否充分理解碎片，那么本书适合您。

## 开始前你需要了解什么

《Android 碎片》假设您熟悉 Java 和基本的 Android 知识，并且您已经使用 Eclipse 或其他 IDE 开发过 Android 应用。您不需要是 Android 专家。

基于此，以下是《Android 碎片》各章节的简要概述。

## 本书内容

我们从深入记录`Fragment`类及其生命周期开始《Android 碎片》。我们会介绍应用中的活动与碎片之间精细的集成配合。您将看到如何将碎片融入用户界面，以及如何将功能封装到碎片中。

**第 2 章**帮助您理解在配置变更（例如设备旋转）期间碎片会发生什么。您将看到碎片如何比过去更轻松地处理这一问题。

对话框在应用中非常常见，现在您将使用碎片来显示它们。请在**第 3 章**中了解具体方法。

当碎片出现时，偏好设置也经历了一些重大的用户界面变化。**第 4 章**涵盖了您需要了解的所有内容，涉及如何使用 Android 的偏好设置框架来管理偏好设置，或者仅作为一种快速简便的方式来保存从一次调用到下一次调用的某些应用状态。
