# Android 碎片（Fragment）

面向专业人士的专业书籍

**配套** ® **电子书** **可获取**

《Android 碎片》是一本 100 页的快速入门指南，旨在帮助您快速学习并使用 Android 碎片。您将学习如何为碎片编写代码；利用碎片的力量构建专业级 Android 用户界面，处理配置变更；编写常规对话框与碎片对话框的代码；处理偏好设置和状态保存；使用兼容库；以及处理高级异步任务和进度对话框。

阅读并使用本书后（本书内容源自畅销书《Pro Android》），您将成为 Android 用户界面专家。至少，您的应用的用户界面和事件处理将更具竞争力且性能更优，尤其是针对平板优化的用户界面和事件。

**您将学会：**

- 什么是 Android 碎片
- 如何操作和编写碎片代码
- 如何响应配置变更
- 如何实现常规对话框与碎片对话框
- 如何处理偏好设置和状态保存
- 如何使用兼容库
- 如何处理高级异步任务和进度对话框
