# 13. 出售公司

你已经成功建立了一家稳固的公司。你有客户，有良好的业务流程来交付优质服务，经过 2 年或 20 年后，你已准备好开启新篇章。这个新篇章可能是寻找投资者，以便达到需要现金注入（哪怕只是为了维持现金流）的新增长点。又或者，你实际上想直接卖掉公司，让别人来接手。

在本章中，我们将探讨如何通过确定公司价值来为其定价。我们将了解退出公司时签订的协议类型，以及如果你选择留在公司会是什么情况。

但在开始之前，我们先来谈谈感受。无论你做什么或怎么做，出售部分或全部公司都是一段非常情绪化的时期。而且你拥有公司的时间越长，这种情绪就越强烈。许多人可能会告诉你把情绪抛在脑后。请记住，客户对你超越合同范畴的忠诚是无法定价的。无论你的员工社群多么出色，那也是无法定价的。相反，估值关注的是资产和负债。

你与客户签订的合同是资产。你欠预付款客户的时间以及你的账单，是负债。就这么简单，或者说通常如此。在本章的第一部分，我们将探讨谁会购买这家公司。

## 谁会购买这家公司？

咨询公司通常会被其他希望填补业务空白或寻求非有机增长方式的咨询公司收购。

* **技能收购**：某些收购是为了购买你团队拥有的技能。通常当一家非常大的公司收购一家规模小得多、产品将被停产的组织时，就会出现这种情况。例如，假设你有一家咨询公司，而一位大客户希望在全企业范围内使用苹果设备。他们可能会直接收购你的公司，让其成为新的支持团队，以便更快地完成项目。

* **客户收购**：其他收购则完全围绕客户展开。收购方可能不想要任何数据库或员工。当被一家看似相似的公司吞并时，这种情况很常见，尤其是当它们处于同一地理区域且是长期竞争对手时。

* **非有机增长**：一家传统上以 Windows 为中心的 MSP（管理服务提供商）可能希望在其产品组合中增加苹果业务，而不是从零开始建立。或者另一家苹果业务公司可能想进入你的区域，与其慢慢开始销售，不如收购现有的市场领导者。

* **产品**：另一个组织可能有意收购你的公司，是因为他们想要你开发的产品。服务产品通常是一个预先打包的服务包，能提供可交付成果。除了获得产品的知名度外，收购方通常也在寻找有关如何开发产品的专业知识。请记住，产品对收购方的价值在于领域知识和客户获取。除此之外的任何东西，收购方都可以在内部自行开发，无需收购另一家公司。

* **软件**：软件产品可能是面向客户的，客户为此付费，也可能是内部软件产品，用于提高组织效率，从而更具成本竞争力。如果操作得当（并且打包得当），软件可以是一个巨大的卖点；但请注意，内部系统之间的集成通常已在可能收购你公司的组织中建立，因此可能不会增加你公司的价值。

* **另一个 SKU（库存单位）**：一些组织已经向客户销售软件、打印机服务或其他产品，而你的团队可能恰好提供最佳的新 SKU，可以交叉销售给客户。当这种情况发生时，你可能成为一个有吸引力的收购目标，因为收购方可以大力推销你所做的事情。当你有过剩产能和良好的业务流程，但缺乏强大的销售团队时，这非常有利。

你建立的社群何时被视为公司的资产？通常，当收购组织能够利用它产生超出你资产标准价值的更大影响时。例如，假设另一家服务提供商收购了你的公司，并希望利用你构建的票据管理工具或与常见服务管理工具的集成。或者，当收购方能扩展其平台支持时，这也会发生，如此一来，他们不仅接手了你的苹果客户，还能向其传统客户销售苹果服务。

希望到目前为止，你已成为苹果相关事务的专家。而且做到这一点很时髦。如今，许多大公司都想知道自己的“苹果战略”是什么。即使寻求收购你公司的组织没有明说，这很可能也在他们的考虑之中，并且当你试图为组织争取更高估值时，这应成为谈判的考量因素。



## 估值

估值是对公司价值的一种估算。成熟的组织会聘请专业评估师来确定估值。这些评估通常由麦肯锡、埃森哲或德勤等大型会计师事务所编制——费用不菲，但绝对物有所值！

根据公司规模和出售方式的不同，为估值做准备可能需要花费数年时间。因此，越早开始越好。以下是让公司为估值做好准备的一些步骤。

-   列出所有负债。负债就是你欠别人的。这包括传统债务，如信用卡，也包括任何你已从客户那里收到款项但尚未交付服务的合同，例如年度预聘金。同时，也要审视任何关于房产、互联网接入、保险、法律等方面的未结合同。
-   列出所有资产。资产就是你拥有的。这包括传统的固定资产，如房产，以及折旧资产（即随时间推移价值下降的资产），如电脑。资产还包括合同价值，以及未受合同约束的客户业务价值的一定比例。与客户关系的时长以及客户业务的稳定增长，会使这个比例更高。
-   从资产中减去负债。一旦你整理好了“所有物”减去“所欠债”的清单，这一步就是个简单的计算。请遵循**公认会计准则**，并务必请一位会计师协助你，以免在后续过程中出现任何意外。
-   记录过去 3 年的利润率。健康的利润率可以提高估值。你的收入是多少，运营成本又是多少？记录得越详细越好，例如按客户、按季度等。在电子表格中绘制图表并形成稳固的趋势线，将有助于这一过程。
-   预测未来增长。我喜欢趋势线。`Microsoft Excel`在生成趋势线方面非常出色（`Keynote`和`Numbers`也同样出色）。更多关于趋势线的信息，请参见附录 D。
-   为你的知识产权定价。考虑一下公司博客及其带来的相关 SEO（搜索引擎优化）、你的销售渠道、你的人脉（包括客户）、各项标准操作程序、用于自动化业务的数据库，以及你创建的任何其他公司知识产权。在出售时，这些将随公司一起留下。如果你在离开时希望保留其中任何一项（或将其用于创办新公司），请在出售过程中进行协商。
-   关注 MRR（月度经常性收入）。这是确立公司价值最可预测的方式。理想情况下，最好是有合同约束的。MRR 的增长可以绘制在趋势线上，以证明公司正朝着正确的方向发展。MRR 不仅包括你产品的销售，还包括订阅服务的转售，例如互联网服务提供商的订阅、`Google Apps`订阅、`Office 365`订阅、租赁以及其他佣金。

最终，这家公司值多少钱？无论估值如何，答案都是：有人实际愿意为这家公司支付的价格。对于合适的买家来说，公司的价值可能远超其账面价值，因为他们可能有交叉销售服务的计划，或者他们拥有一个系统，可以将你的客户纳入其中，从而提高利润率等。但在大多数情况下，我见过的公司价值远低于老板自以为的价值，尤其是在那些主要依靠按小时计费的咨询工作且没有长期合同的组织中。

## 乘数

兼并与收购可以有多种不同形式。我见过许多收购案例，买家并非支付现金，而是基本上用与客户合作业务的某个百分比来换取员工和客户名单/合同。那些因缺乏动力、增长乏力或利润率低下而苦苦挣扎的咨询公司，正是此类收购的绝佳目标。如果你正在评估一份此类出价，或准备构建一份此类报价，请记住，收购后，组织中的负责人很少会留下来。

那么，是什么决定了乘数？是有人愿意为一家公司支付多少钱。仅此而已。但有所帮助的因素是：平稳且递增的利润数字。这意味着要将你大部分业务转向我们之前讨论过的 MRR 模式。

## 竞业禁止协议与承诺书

出售你的公司时，很可能附带一份协议，要求你不会与买家竞争。你的竞业禁止协议或承诺书（类似于竞业禁止，但更具约束力）是买家购买的一项资产。

竞业禁止协议在某些州的约束力比其他州更强，但总的来说，法律条文的规定在巨额的诉讼费用面前可能相形见绌。如果你打算以任何身份与前客户合作，我会建议你争取获得相关豁免。

我听过许多咨询师说：“但客户是先给我打电话的”，或者“新管理层对他们不好”。你的公司能成功，原因之一很可能是你对客户很好。你可能与许多客户在工作之外建立了私人关系，看到他们可能无法得到你本会给予的照顾，这确实令人难受。但你在法律上有义务不进行干涉。违反该协议可能会带来严重后果，所以如果你这样做，要意识到你和你前客户所面临的风险。

## 你会留在公司吗？

这是最重要的问题之一。如果你还在，公司很可能会经营得更好，至少在一段时间内是这样。任何成熟的收购方都会理解这一点，并且很可能希望你留下来，至少过渡期是如此。

在经营公司多年并做出所有决策之后，突然要询问他人意见或请求许可，可能会感到别扭。如果你将要留下来一段时间，请务必与公司的新所有者敲定一些细节。

-   在招聘和人力资源方面，你仍然拥有哪些权力？
-   你拥有多大的财务审批权？
-   如果同时负责销售，你拥有多大的折扣权力？
-   你将被如何逐步退出，包括最终的离职日期是什么？

如果你计划离开，请务必与新所有者商定一个可接受的退出策略。这应该包括附带固定日期的里程碑节点。我发现对员工保持透明非常重要，因此，在可能的情况下，最好让团队中的每个人都了解时间表。


## 运用政治资本

从经营自己的企业到为他人工作，这可能会充满挑战。当你拥有自己的企业时，你可以为团队规划路线。当你不拥有时，你就做不到。相反，你有时必须运用政治资本。例如，你帮某人一个忙。因为这样他们就欠了你人情。事情就是这样，对吧？不。

你帮某人的忙，因为这是正确的事。你不应期待任何回报。这叫做做个正派人。也许你会得到回报；也许不会。但最终总会有好结果。我看到很多人试图操纵局面以获得特定结果。别这么做。

如今我努力不去操纵任何人。但我曾经做过。最终，当人们意识到你在操纵他们时，他们将永远不会再信任你。这很糟糕。但这并非避免此类行为的唯一原因。避免它，是因为这本身就是完全错误的。如果你无法用真相赢得人心和思想，那就别费劲了。

突然之间，似乎政治资本这种东西可能并不存在。但它确实存在。它叫做因果报应。而且你没必要去想它。相反，只需做个好人。践行诚实。品德高尚。帮助他人。避免评判他人。帮助那些不如你幸运的人。那么，你就不需要运用那种政治资本了，因为你的行为会指引你，你将以你注定该有的方式获得成功。

这并不意味着你不该有主见。也不意味着你应该试图说服他人，你的观点是对特定情况的正确看法。这也不意味着你应该软弱。你可以坚强，并持有在某个组织中并非人人认同的观点。而且，你可以通过不必永远正确（即使你并不正确时）来获取政治资本。

如果你是一个决策者，你可以扮演守门人的角色，确保你所在的组织部分为整个组织的使命做出贡献。当他人向你提出请求时，做正确的事——不仅仅是为了你的预算或团队，而是为了整个组织的使命。当你需要他人帮助时，相信他们也会这样做——最好不要设定他们需要回报你的期望。你或许能与他人建立信任；但双方都不应带着期望回报的心态离开，除非在给予帮助时就协商好了。但那样的话，它就不是一个帮助了，对吧？

但人非圣贤。因此权力斗争时有发生。远离太多的权力斗争。谨记父母教导我们的那些古老格言，比如“走正道，行高义”。

最终，我们已将政治资本定义为信任，而不是你能为他人做什么。信任建立在价值之上，无法急于求成。你帮助自己所在组织部分（长期来看，是所有部分）的能力，随后便建立在你和你的团队所能构建的政治资本之上，而这需要时间、成功和耐心。记住这一点，并避免那些我们在幼儿园就被教导要避免的错误，我想你会做得很好！

## 结论

你把公司的控制权交给了别人。不要总是事后批评他们，说诸如“换了我，我不会这么干”之类的话。这对任何人都没有好处。这并不意味着你需要不真诚，对新所有者的所有说法都一味附和。相反，要给予信任，私下里表达关切，以便给新所有者最大的成功机会。

我曾听过一个很棒的类比：“老市长不应该总是对新手市长指手画脚。” 这是智者之言。变革的发生是有原因的；尊重界限，但要尽量提供帮助——即使你并不总是认同新的观点。

最终，你是拥有选择的！你可能最终选择不出售公司，而是走一条完全不同的道路。或者你可能会选择保留公司，转为兼职所有者，我们将在下一章讨论这点。无论你做什么，都举办个派对吧。
