# 6. 广交朋友：发展合作伙伴关系

在这个世界上，没有朋友寸步难行。来自你所在行业及相邻行业的朋友能为你提供不同的视角和技能组合，还能让你接触到许多不同的客户。但让人购买某样东西或使用某项特定服务，并不总是值得冒险，即使你并非因将客户推荐给“朋友”而赚取佣金。

第 2 章讨论了构建产品组合，包括要销售的产品和提供的服务等内容。但在本章中，我们将探讨如何利用与其他组织的合作伙伴关系，将你的公司提升到新的水平，从那个常常被遗忘的伙伴——你的客户——开始。

## 吸引客户进行转介绍

你的客户是寻找新客户最具成本效益的途径。事实证明，做好本职工作终有回报。但每个行业都各不相同。你不可能总是毫无防备地向客户请求，让他们把你推荐给其他可能从你服务中获益的组织。以下是几种你可以用来获取转介绍的巧妙策略：

-   **主动请求**：如果你与客户关系良好，那么在项目结束之际或通过电子邮件，直接请求他们将你推荐给其他客户。这样做很难不显得绝望或过于肉麻，但却是可行的。
-   **留下资料**：我曾遇到过客户转交营销材料的情况，但如果你不给他们留下点什么，他们就没法这样做。询问客户是否可以留下几页单页资料，供他们转交给其他人，这是另一种从“遗留资料”中增加业务的方式。
-   **为转介绍提供补偿**：如果涉及某种形式的补偿，最合乎道德的做法是补偿你客户所在的公司。这是一条危险的下坡路，在某些行业，这会立即引发不信任（甚至可能因各种法规而违法）。请谨慎行事。
-   **在客户的网络界面中放置链接**：如果你为客户提供了一种在线查看服务的方式，同时也可以给他们一个小按钮来推荐客户。这是获取转介绍最简单、最被动的方式之一。

接下来，是时候将你的注意力转向你想要接触的外部合作伙伴类型了。在下一节中，我们将探讨如何找到合适的合作伙伴，包括寻找能够提供互补服务、分包业务，或者仅仅是潜在的互相推荐业务（单方面或双向）的人。

## 寻找合适的合作伙伴

我曾收到过成百上千家公司发来的电子邮件，它们希望与我所任职的组织合作。有些对我来说，或者对我当时所代表的组织来说，是非常划算的交易。其他一些则有点牵强。而最初看起来最划算的那些，在我们深入研究其业务后，却发现糟透了。

在选择合适的合作伙伴时，请始终将你的特定客户放在首位。问自己以下几个问题：

-   我的客户能和这个人或这家公司相处融洽吗？
-   我的客户需要哪些商品和服务？
-   当（而不是如果）他们与这个其他供应商发生问题时，我最终会如何卷入其中？
-   该供应商是否能真正提供他们声称能够提供的服务？
-   成本是否匹配？
-   最终会是该供应商还是我，在客户公司里无意中承担了对方的角色？
-   这应该是一种分包安排、双向的正式业务推荐，还是仅仅是我可以指向的人？

对于许多苹果咨询公司来说，最常见的商业安排是帮助客户找人处理他们的 Windows 设备。我经常看到这种情况。也许只是为了那两台财务用电脑，或者是为了大多数基于 Windows 的系统。让我们面对现实吧，传统上，管理 Windows 系统的人与典型的使用 Mac 的客户确实有些不同。找一两个（或三个）你信得过，并且适合你这类客户的人。


### 构建合作伙伴关系

在大多数情况下，如果没有报酬，我建议为每个需要与客户对接的产品或服务寻找三个合作伙伴。然后你可以指向这些合作伙伴（例如，在你网站的页面上）并这样说：“我建议你联系这三家，看看你喜欢与哪家打交道，并认为哪家最符合你的业务需求。”

你不想与合作伙伴竞争。例如，路由器制造商可能会直接销售但不提供任何服务。如果你想提供服务但不出售他们的设备，那么如果他们需要有人能提供劳动力（特别是对于复杂产品），你可能就找到了一个完美的合作伙伴。但如果你也想销售路由器，请务必弄清楚他们是否直接向客户销售，这样你就能最好地了解他们在某个交易中与你竞争时可能发生的情况。

大多数供应商的管理层希望与你建立健康的关系。但负责你所在地区的客户团队可能希望通过销售产品来赚取佣金。在评估合作伙伴时，还要查明：一旦你注册了一个交易，每个合作伙伴是否会直接销售；如果会，那么只要你比他们的内部员工更早注册该交易，你是否能锁定比他们的客户团队所能提供的更优惠的价格。

什么样的人是好的合作伙伴？显然，找到好合作伙伴最重要的方面之一是，你的客户不会因为你的推荐而解雇你，这要么是因为合作伙伴从你这里抢走了客户，要么是因为他们的工作质量太差，以至于客户将你与该工作联系在一起。

如果你选错了合作伙伴，你就有失去客户的风险。客户的信任是日积月累建立起来的。如果你的客户与某个合作伙伴有过糟糕的经历，他们很可能会失去对你的信任。限制由此产生的责任的一种方法是，不收取费用。相反，你可以采用双边推荐协议。这不一定总是奏效，但通常被称为“互惠互利”。

## 互惠互利

当你找到一个新合作伙伴时，如果你大量销售他们的产品，他们就会更有动力向你推荐客户。这种关系可能需要几个月才能成熟，所以如果你想建立良好的关系，那么每一方都应该有所收获：互惠互利。例如，我曾与一家新公司达成合作，转售他们的路由器。恰巧在第一个月内，我给他们介绍了大约十几个客户。他们的回应是给我推荐了一大批客户。这些都是全新的客户，拥有大量未受管理的苹果设备。

基于关系类型的不同，互惠互利可以有不同的表现形式。我曾经有一家公司给我推荐了很多业务，而我没有业务可以回馈给他们。于是我制定了一个佣金方案来补偿他们的推荐。另一次，对方要求一个我无法给予的佣金结构，这实际上结束了这段关系（尽管那些知道我是值得信赖供应商的客户团队，仍会零星地给我带来一些客户线索）。

鉴于所有关系各不相同，要对你能销售多少产品抱有现实的期望，并尽早设定预期。同时也要向他们提出同样的要求。这将使你能够为这段关系设定一些目标并制定计划。在发展业务时，将合作伙伴转化为客户非常重要。然而，务必不要期望过高，也不要仅仅因为随之增加的回报而提高你在关系中的风险。

所有软件、硬件和服务公司，如果与那些依赖自身业务成功才能成功的伙伴合作，都会从中受益。所以要寻找方法让所有这些关系对参与各方都有利。成为合作伙伴能让你保持敏捷；但有时，你会觉得简单的合作伙伴关系潜力太大，而希望转为分包安排。

## 分包与合作

合作很棒，如果做得好，它将帮助你以最优的价格为客户提供最好的服务。但有些时候，你会决定把工作留在自己的“纸面”上。有很多公司都有合同，他们希望将所有费用集中在一份采购订单上，以简化会计操作。你也可能决定分包某些类型的工作，以便为客户提供更全面的服务。

在分包之前，请确保签好合同，并让所有分包商签署该合同。制定公平且价格合理的费率。你应当保持 50%或更高的利润率，以支付佣金、税费、财务处理费等。如果你无法做到这一点却仍然选择分包，那么最终你很可能会将辅助服务作为亏损引流的项目提供，或者通过“为客户提供单一问责点”来获取市场份额。如果能保持那样的利润率，分包商就不能直接与客户合作吗？是的。但他们将不得不去寻找客户、处理付款、纳税、等待 60 到 90 天的付款周期，以及所有其他“有趣”的事务。而你则承担了大量责任。

如果你决定分包工作，请确保至少有两家供应商能够分包某个特定项目。假如你被要求做一个项目，而你有一个可以雇佣来干活的供应商网络。请确保分包协议中至少有两家供应商。否则，如果其中一家倒闭或决定不再分包，你将面临一个可能带来负利润的业务。

一旦你拥有了分包商网络，就该理清物流了。像 `Expensify`、`Ebay`、`Stripe` 等网站可以用来实现自动支付给分包商。如果你能在你的 PSA 或会计系统中实现自动路由来跟踪项目支出，那就更好了！此外，你需要将项目和工单备注输入你的系统，所以提前决定好你希望承包商如何将他们的时间录入你的时间跟踪系统（工作量少的话，可以选择通过电子邮件将工单发给你，然后你粘贴进去）。

分包协议中其他需要涵盖的较小方面：

-   提前决定：你是否希望提供来自你组织的电子邮件地址访问权限？你是否希望分包商穿着品牌服饰？以及你是否希望他们承认他们不是你的直接雇员？
-   当（不是“如果”）发生账单纠纷时，你将如何处理？不要仅仅因为你的客户不喜欢提供的服务就克扣分包商的款项，但要清楚，长期暴露在这种风险下可能会适得其反，特别是当分包商没有使用与你相同的系统时。
-   你将如何处理直接与分包商接洽的请求？分包商是否允许在你的项目结束后与客户合作，还是这需要永远通过你的公司进行？作为分包商完成最后一个项目后，在直接与客户接触之前是否需要等待一段时间？

所有这些关系的属性都应该提前讨论，并形成书面记录（例如，通过电子邮件）。安排越正式，处理文书工作可能就越困难，但从一开始就设定好合理的期望，将引导你走向更成功的长期关系。

如果你本人就是分包商，那么请确保你也理解所有这些关系的属性。



### 处理与合作伙伴之间的问题

在投入大量精力发展合作伙伴关系后，你希望维系这段关系并拓展业务。如果失去客户的信任，这一切都将无从谈起。这意味着在做转介绍业务时，你必须确保自己全力以赴。

无论你多么努力，有时与客户之间仍会出现问题。遇到这种情况时，我通常会选择做正确的事。做正确的事，这样客户就不会向合作伙伴抱怨你的业务。没有什么比愤怒的客户更能快速破坏一段关系。而且，无论你多么优秀，一些看似简单的问题都可能导致客户去找合作伙伴。防范于未然的一个好方法是在潜在客户开始流向你之前，就与客户建立好升级沟通路径。

另一种轻易破坏与合作伙伴关系的方式是争夺业务。合作伙伴通常占据着相邻的业务领域。如果你也涉足其中，那么务必清楚了解自己的哪些服务与你获得转介绍的那些合作伙伴所提供的服务存在重叠，并确保你不会实际抢占合作伙伴的业务，尤其是那些向你推荐业务的合作伙伴。相反，你应该多夸赞他们，并向客户推荐他们。

绝不要说转介绍合作伙伴或其产品的任何坏话。如果该机构生产的软件或硬件存在问题，主动提出帮助他们联系供应商，而不是简单地说他们的问题导致了额外的服务需求。如果客户不信任你帮助修复的产品，他们可能不会继续使用这些产品，最终导致他们不再使用双方的产品，从而引发问题。

## 结论

最好的转介绍来自客户。你的客户是对你服务的真实评级。无论广告、营销、公关，还是花在其他公司转介绍上的金钱，都无法在成交率、成交后客户质量等方面为你提供更有价值的潜在客户生成引擎。但你不能永远依赖转介绍。

一旦达到一定阶段，你需要寻找合作伙伴。如果一个合作伙伴好得令人难以置信，那这种好景不会长久。但当一位优秀的合作伙伴出现，并且双方都能获得价值时，你就找到了金矿。所以，务必照顾好你的合作伙伴，并始终保持对他们坦诚相待，明确你们之间的关系。这不仅因为本书始终贯穿透明和诚实（品性的关键要素）这一主题，还因为如果你不这样做，他们最终会厌恶你，而你肯定不希望其他公司在市场上破坏你的潜在客户。
