# 12. 何时停止增长

这一章内容会很简短。一种盛行的商业理念认为，如果你的收入增长没有快于行业标准，那么你就会被竞争对手超越。一派胡言。无论你公司规模多大，你都应该感到满足。如果你想增长，那很好。如果不想，你也完全不必勉强。应该在你感到舒适的范围内增长；否则你就无法积极地做到这一点。

你的公司能发展到多大，是没有限制的；唯一的限制是你对组织规模的适应程度。我见过许多公司规模停留在两人，然后就这样维持了 30 年。也见过一些公司发展到 50 人左右，那便是最理想的状态。还见过一些组织增长缓慢，却并不想这样。无论何种情况，如同商业中的许多事情一样，清晰、简洁地沟通正在发生的事情至关重要。

本章针对一种非常具体的情况，即公司领导层已明智地做出选择，不希望公司或公司内的某个部门继续增长，至少短期内不打算这样做。通常这是因为他们想专注于其他事情。你通常可以把这类公司称为生活方式型企业。即使只是暂时的。

做出这样的决定后，最大的挑战之一就是留住员工，所以我们先从这里开始讲起。

## 与员工沟通

停止增长是困难的。这里最重要的是，你对所有人公开坦诚，说明你在做什么。尤其是当你一直在增长，而现在是刻意放缓增长时。告诉人们你为什么放缓增长，并清晰地说明组织的策略在不久的将来可能会如何变化。

放缓增长降低了你的风险，但也减少了你能提供给员工的机会。在本章后面，我们会探讨保持团队积极性的方法，但现在，首先要让你雇佣的每个人都感到安全。解释你为什么放缓增长，你将如何做，以及在新模式下每个人拥有什么机会。

预计会有一些人离职。毕竟，每个在职员工的未来在哪里？这通常不会立刻发生，而且只要你清楚地表明了意图，这往往是友好的分手。但如果有人离开，也要预计会有一些客户流失。每当人们感觉工作中机会变少时，这种情况就不可避免。

## 增长放缓后领导（并留住）团队

建立和发展一个组织是很有趣的！管理一个停滞或萎缩的组织则可能糟透了。但这并非必然。员工满意度是一件了不起的事，也是让你的公司成为一个绝佳工作场所的关键。在增长时期，每个人都无一例外地对自己的工作感到满意。员工们看到巨大的成长机会，许多可以指导的优秀新团队成员，社区中也弥漫着围绕扩张的热烈气氛。但最终，每个组织和每个团队的扩张都会放缓。我从未见过有谁能从容应对这种变化，除非是那些刻意放缓增长的人。

领导者该如何应对？如何让一个完成了组织所有要求的、有凝聚力的团队保持投入？如果团队内部无法提供，又如何提供伟大职业生涯的回报？让我们看看在你即将跨入公司生命周期下一阶段时，需要牢记的一些重要概念。

1.  **确保员工感到安全**：员工需要在工作中感到安稳。当员工承担了更多工作，并看到招聘放缓时，他们会做最坏的打算。要完全透明。给予员工足够的尊重，尽可能多地解释情况，即使这意味着他们会因此主动离开。当然，这始于一个健全的策略！

2.  **理解是什么激励着人们**：我们都被不同的事物所激励。务必了解是什么激励着团队成员，或者让他们的主管去弄清楚。然后通过调查员工，并将结果与团队所做的定性分析进行对比，以再次确认。这通常比以前更容易，因为你无需将精力分散在开设新办公室、招聘等事务上。

3.  **提供不同的认可成功的方式**：当一个组织决定转变优先级、限制对某个团队的资助，或者公司增长放缓时，你衡量绩效的关键指标也必须随之成熟。这通常始于从关注营收转向关注利润率，从利润率转向渠道赋能，在不同类型的工作之间切换，或者从这些转向关注创新。

4.  **寻找在组织中提高可见度的方法**：随着团队增长放缓，提高在整个组织或社区内的可见度，能让团队成员对自己在组织中的角色感到积极。允许员工接触其他团队，也能促进人际网络建设，以可能是在高速增长时期无法实现的方式进行更多的创意分享，并在组织内部培养更强的团队合作精神。提高可见度也为每位员工带来了曝光机会。

5.  **如果你爱他，就让他自由**：支持你最优秀的员工；但也要做好替换他们的准备。即使这样做会让你的工作更难，奖励出色的工作将激励下一代做得更好。反之，不识别员工的潜能则会导致员工不满和工作质量低劣。看到我以前的员工在新的岗位上茁壮成长，是我职业生涯中最满意的部分；希望你也如此！如果你的组织不增长，那么组织内的职业发展通常会有天花板。你可以通过利润分享来弥补这一点。

最终，领导一个成长中的组织比领导一个在电子表格上趋势并非向上向右的组织要容易。接手一个停滞的组织其实非常容易，因为别人已经处理好了重塑组织生存方式的问题。但进行这种重塑意味着首先要照顾好员工。要审视团队如何融入新的布局。让团队不仅帮助定义策略，还要制定实现策略的战术。如果还没有这样做，那就授权每个人在各个层面做出贡献。

无论你做什么，都不要退缩到你的团队里。要关注团队的幸福和福祉，但当增长放缓时，要向外寻找机会，为组织和社区的各个领域做出贡献。寻找帮助员工的方法，做正确的事情，确保将他们的需求与组织的需求结合起来。并且要记住，将一个群体的策略与组织的策略保持一致，是整个组织成功的关键。如果每个管理者都只是想着壮大自己的团队，那么他们所做的只会是分裂一个原本有凝聚力的组织。



## 人员裁减

有时，我们会发现自己不再适合当前的发展方向。我们赢得了大合同却无法履约，或者失去了它。于是我们不得不进行缩减，这令人沮丧。在本节中，我们将探讨在裁减人员时应该怎么做，并提供一些技巧，使这一过程尽可能合乎道德，同时又不损害公司。

过去我曾不得不进行不同类型的裁员，我深信坐在谈判桌对面的人同样值得尊重，因此绝不敢轻率对待。归根结底，缩小组织规模首先要设法留住最优秀、最聪明、最忠诚的团队成员。但当你别无选择，不得不让员工离开时，那些日子绝不好受。

对于留下的员工，请务必尽你所能传达你的战略，并在可能的情况下保持透明。告诉他们是否还会有进一步的裁员。如果会，他们也许会自行离开，而你将无法决定谁走谁留。但通过坦诚相待，如果你愿意，你们之间的关系将得以维系，也不会有人四处抱怨你亏待了他们。但最重要的是，你确实不会亏待他们。

对于即将离开组织的员工，请保持尊重。尽可能提供丰厚的遣散费，并在适当的时候为他们提供出色的推荐信。与每位离职员工进行离职面谈。把你最后的支票放在办公桌上，这样他们就能清楚地知道你做出的决定已不可更改。我通常发现，一旦人们意识到决定已经最终敲定，他们就能立即开始制定计划，并找到心理上的终结感。

## 增长停滞时的财务管理

成长型公司往往现金充裕。但当你停止增长时，你的成本往往不会随之停止。税收上涨，通货膨胀加剧。至少，每年需要给员工增加生活成本，才能留住他们。

到了某个阶段，如果你没有增加客户和员工，你就只能在一个看似萎缩的公司里想办法赚更多的钱。你可以这样做的一些方法包括：

*  用真笔在真纸上写下一句话，说明如何削减`总账`中`应付账款`栏目下的每一个条目。用显微镜审视成本，将有助于你弄清楚哪些可以削减而不影响员工绩效，哪些不能。
*  重新谈判每一份合同。你付给你的律师多少钱？少付一些。你付给你的会计师多少钱？也少付一些。如果无法重新谈判，就开始寻找其他同样具备资质的承包商。
*  与你的税务顾问一起审查你当前和未来的计划。在许多情况下，如果你转换成一个不以增长为导向的不同类型的公司，你最终可能缴纳更少的税款。
*  偿还公司的任何债务。债务利息可能会压垮你，尤其是信用卡债务。而且，如果你选择出售公司，负债越少，交易就越容易。
*  将员工转为承包商。这是我最不喜欢的选项，而且可能实际上比雇佣`W2`员工成本更高。根据我的经验，如果你将一名员工转为`1099`形式，他们会立刻开始寻找新工作。不，真的，即使他们告诉你说没关系，他们也会这么做。
*  转向虚拟办公。当你试图说服员工留下来，但个人成长机会（或至少是感知到的机会）匮乏时，这尤其困难。但租金往往和一名员工的薪水差不多。转向虚拟办公最重要的方面是保持社区感；否则，你和员工之间产生的距离会导致他们更有可能带着你的客户（或者说，在他们看来，是他们的客户）跑路。
*  购置房产。等等，我说的是降低成本、偿还债务……商业地产的抵押贷款通常是 15 年期，所以这通常是一笔相当大的投资。不过，通过重新分配租金，你应该能更容易地建立起更多的应急基金或退休基金。而且你的净回报应该高于其他投资。

无论你选择哪种方案，在做任何决定之前，请确保你清楚自己的收入和成本，并确保建立更大的现金储备，或者将利润投资于低风险投资工具。或者，忘掉我所说的一切，去咨询财务顾问吧！

## 结论

在不得不解雇四五个人的日子里，再多的酒精也无法让你感觉好受些。避免这种情况最简单的方法是有意识地增长，而不是作为对业务的被动反应。这意味着不要因为招揽了某些新客户就随意招聘和解雇。相反，它意味着要知道你想要达到的规模，以及如何实现。也意味着要知道何时该放缓脚步，甚至何时应该避免签署一份可能导致你破产的合同。

在你感到舒适的时候成长，而不是因为面子、利润或压力而感到有义务时才成长。我再怎么强调“有意识”这个词也不为过。在正确的时间成长更具可持续性，对你周围的人更好，最终对你的健康也更好。一旦你的组织达到了最大规模，你可能会选择出售公司，这正是我们在下一章要讨论的内容。
