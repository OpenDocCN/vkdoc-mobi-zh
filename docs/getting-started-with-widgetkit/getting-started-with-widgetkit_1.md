# 阅读前准备……

在你开始通过本书探索 WidgetKit 的魅力和强大功能之前，请确保已满足以下先决条件：

*   **一台运行 macOS Catalina（10.15.4 版本）或更高版本的 Mac**：但我们建议使用运行 macOS Big Sur（11 版本）或更高版本的 Mac，因为代码在该版本上测试通过。
*   **Xcode 12 或更高版本**：`Xcode` 是为 Apple 生态系统开发应用的主要工具。你可以从 Apple 开发者网站下载最新版本的 `Xcode`。^(¹) 我们建议使用 `Xcode 12.4` 或更高版本，因为代码在该版本上测试通过。
*   **Swift 5 或更高版本**：由于新版本的 `Xcode` 附带更新版本的 `Swift`，因此你无需为此担心。
*   **安装了 iOS 14 或更高版本的模拟器和设备**，因为用于开发小组件的 `WidgetKit` 框架仅从 iOS 14 开始支持。
*   **一个 Twitter 开发者账户**：在本书的最后一章，你将开发一个使用 Twitter API 的小组件。只有在你拥有 Twitter 开发者账户后，才能访问 Twitter 的 API。你可以通过 Twitter 的开发者账户页面申请开发者账户。^(²)

通常情况下，Twitter 需要一两天甚至更长时间来审核你的申请并予以批准或拒绝。因此，在开始阅读本书之前申请该账户是个好主意，这样当你读到最后一章时，你的 Twitter 开发者账户就可以直接使用了。
