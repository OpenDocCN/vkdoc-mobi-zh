#### 自定义布局

预设布局提供了多种工作区，但幸运的是，你不必完全照搬使用。Unity 提供了灵活性，让你可以随心所欲地完全重新排列编辑器窗口。

##### 调整区域大小

首先，在尝试各种预设布局时，你可能会注意到某些区域太窄了，例如，宽布局（图 2-20）的左侧面板。幸运的是，你可以点击区域的边框并拖动来调整区域大小。

##### 移动视图

更酷的是，你可以移动视图。将视图的标签拖拽到另一个标签区域，即可将视图移动到那里。将标签拖拽到“停靠”区域则会创建一个新的区域。例如，从默认布局开始，将检视面板标签拖拽到层级标签的右侧。现在检视面板视图和层级视图共享同一区域。然后将控制台标签拖拽到层级/检视面板视图的底部区域。这会将区域有效地分成两部分，底部现在包含控制台。结果应该类似于 图 2-21。
