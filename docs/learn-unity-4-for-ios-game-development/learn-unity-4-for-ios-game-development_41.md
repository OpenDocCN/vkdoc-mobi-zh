# 第三章 制作场景

玩 Angry Bots 很有趣，但本书是关于用 Unity 制作游戏，而不是玩 Unity 游戏！终于，是时候开始游戏开发了。从一无所有开始，最终打造出一个可玩的游戏，这是一段漫长的旅程，而现在正是起点。在本章中，您将从 Unity 编辑器中的一个空场景开始，并用 3D 环境的基本构建块来填充它：模型、纹理、灯光、天空盒和摄像机。

游戏开发者（尤其是预算有限的开发者）面临的挑战之一是如何获取这些构建块。Unity 编辑器提供了一些基本的模型，任何图像都可以用作纹理（按照互联网的传统，我将提供一张我的猫的图片作为示例纹理）。并且，正如第 1 章中提到的，Unity 安装包中包含一组标准资源包，您可以随时将其导入到项目中。然而，仅靠这些资源所能达到的效果是有限的。

幸运的是，Unity Technologies 的团队认识到了这一需求，并创建了 `Asset Store`（资源商店），这是一个 Unity 就绪资源的在线市场，其商店界面直接集成在 Unity 编辑器中。事实证明，其中很多资源都是免费的，因此我们将利用这一点，将其中一些免费资源整合到本章创建的 Unity 项目中。
