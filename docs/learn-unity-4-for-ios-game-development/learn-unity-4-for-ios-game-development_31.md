# 关于作者

**菲利普·朱**是一名通用型程序员、半路出家的游戏开发者，以及怀有抱负的作家（他希望自己写过《*雪崩*》这部小说）。自从高中时第一次接触到 Apple II 电脑，菲尔在半导体、国防/政府、计算机图形、互联网、移动和电子游戏行业积累了超过二十年的专业软件开发经验。他获得了麻省理工学院计算机科学与工程学士学位（熟悉内情的人知道是课程 6-3），以及约翰霍普金斯大学计算机科学硕士学位。

菲尔目前以独立开发者身份居住在阳光明媚的加利福尼亚州亨廷顿海滩，通过自己的咨询公司 Technicat, LLC 承接合同工作。项目包括 3D 内容创作工具（`Nendo`）、电子游戏（《Darkwatch》、《Tech Deck: Bare Knuckle Grind》）、虚拟世界（《Blue Mars》、《Playstation@Home》），以及一些零星的 Java 工作。但他仍然怀念在 Lisp 机器上工作的时光。

在个人项目中，菲尔使用 Unity 引擎开发并自助发布了面向 Mac/PC、网页和移动平台的游戏，品牌为 Fugu Games 和 HyperBowl（后者从 Hyper Entertainment 获得授权）。欢迎访问 `http://fugugames.com/` 和 `http://hyperbowl3d.com/` 查看。

菲尔定期撰写博客和发布推文（`http://fugutalk.com/` 和 `@fugugames`），并在 `http://technicat.com/` 上维护游戏开发参考资料和软件开发文章，其中部分内容已被收录为大学课程阅读材料，并发表在 MIT 创业评论、任天堂开发者支持小组网站以及 Nook 和 Kindle 等平台上。菲尔在 `http://github.com/technicat` 上分享开源代码片段，包括本书的 Unity 项目。
