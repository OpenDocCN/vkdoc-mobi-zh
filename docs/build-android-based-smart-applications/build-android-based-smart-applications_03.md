# 第二部分：SmartAppGen：从结构化知识自动生成知识应用

## 6\. SmartAppGen 的设计与实现

SmartAppGen 是一套框架，旨在帮助从结构化知识中自动生成基于知识的应用程序。结构化知识可以以多种格式提供，例如 Excel 表、文本和 XML。

* 从文档中提取问题、规则、信息等内容，并保存为 XML。
* 根据生成的 XML，生成相应的 Android 布局、Android Activity、CLIPS 规则和决策引擎。
* 各种框架，如语音转文本、音频捕获器、照片捕获器、上传管理器等，也会自动嵌入到项目中。
* 生成的代码框架被整合在一起，从一个新的 Android 项目生成一个功能完整的 Android 项目。
