# 未来工作

命名实体识别软件可用于细化“谁/什么”类问题，使其更具针对性。这将有助于用户理解问题，同时提升问题质量。这些数据可用于生成高度专业化的问题，例如：谁、什么、何时、多少等。此外，它还将有助于识别句子的主语。

另一项可改进之处是允许出题者选择用于生成问题的句子，而非使用所有句子。一个关键标准可以是句子中是否存在对问题至关重要的特定词性标注的词。

此外，复合句既可以拆分为两个独立的句子，也可以直接用作问答对。例如：

输入句子：

- “乔去了厨房，因为他饿了。”
- (“乔去了厨房。”) + (“他饿了。”)
- 可能的问题：
- “谁去了厨房？”
- “谁饿了？”
- “乔为什么去厨房？”（答案：“因为他饿了”）

然而，这类问题要求程序对句子有更深的理解。第一步应以能够识别句子主干和修饰部分的方式来解析句子。可以使用`OpenNLP`解析器来确定句子结构。例如：

输入句子：“敏捷的棕色狐狸跳过了那只懒狗。”

输出：

```
(TOP
(NP
(NP
(DT The)
(JJ quick)
(JJ brown)
(NN fox)
(NNS jumps)
)
(PP
(IN over)
(NP
(DT the)
(JJ lazy)
(NN dog)
)
)
(. .)
```

`PP`、`IN`、`JJ`等是来自 Penn 树库标签集[20]的词性标注。以这种方式解析的句子有助于计算机理解句子的结构，并帮助它们生成比简单操作原始句子更具创造性的问题。

此外，以这种方式解析的句子有助于从句子中提取知识。知识可以以规则的形式存储（例如，`CLIPS`[19]是一种规则引擎）。这些规则可以作为谓词，用于推导结论。这些结论将作为新规则存储，也可以输出为问题。

企业需要规划知识管理自动化。“知识应用”可以从现有的结构化知识中自动生成。此类应用可以帮助知识工作者更好地完成日常工作。这将显著减少培训人员所需的时间，并提高知识工作者的效率和准确性。

在培训员工掌握非结构化知识方面，公司每年花费数百万美元；许多培训效果不佳，白白浪费了这些资金。实施 AutoQuiz 可以确保员工真正理解培训或演示中分享的知识。项目经理可以评估培训或演示的效果（例如，哪些培训是重复的，哪位员工学习慢，哪位培训师效果不佳等），并采取相应措施。
