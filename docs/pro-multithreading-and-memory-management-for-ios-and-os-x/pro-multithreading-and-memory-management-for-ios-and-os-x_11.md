# 引言

## 前言

本书探讨的是 iOS 和 OS X 中的 ARC、Block 及 Grand Central Dispatch。它与其他书籍略有不同。简而言之，本书内容相当深入，并解释了以下要点：

- ARC：为 iOS 5 和 OS X Lion 新引入的内存管理机制。
- Block 和 Grand Central Dispatch：为 iOS 4 和 OS X Snow Leopard 引入，主要用于编写多线程应用程序。

这些新技术是开发 iOS 5 和 OS X Lion 应用所必需的。虽然它们看似容易理解，但实际上，如果您未能真正掌握它们，将会遇到各种陷阱。在本书中，我将结合苹果的源代码进行讲解，帮助您获得比仅仅阅读苹果官方参考资料更深入的理解。

## 本书适合的读者

- 了解 C/C++，但对 Objective-C 不太熟悉的读者
- 希望了解 Objective-C 源码如何工作的读者
- 正在开发 iOS 或 Mac 应用，并希望提升知识水平以实现进阶的读者
