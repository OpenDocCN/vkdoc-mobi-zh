# 关于技术审校

**保罗·查普曼**、**马克·马克达德**和**罗斯·沙罗特**是三位不受地域限制的移动开发者，他们共同组成了 Long Weekend 团队（[`www.longweekendmobile.com`](http://www.longweekendmobile.com)）。

保罗拥有 15 年软件行业经验。他曾创立 SaaS 初创公司 cvMail（该公司于 2007 年被汤森路透收购），并曾担任东京一家日本公司的 IT 高级经理。保罗目前是澳大利亚斯威本科技大学移动软件领域的兼职行业研究员，并担任该校 ICT 学院顾问委员会成员。

马克是一位多产的网络与移动程序员，经验丰富的销售专家，并能流利使用日英双语。他毕业于伊利诺伊大学，曾合著多门大学级别的移动编程课程，并在东京的智能手机技术圈中非常活跃。

罗斯是一位对金融领域有浓厚兴趣的信息系统专家。他曾设计、实施并管理价值数百万美元的 IT 系统，并领导关键任务的开发工作。他是一名全职的 iOS 和 Android 应用开发者，常驻纽约和东京。
