# 2. Unity 速成课程

在本章中，我们将深入 Unity 的世界。Unity 是开发混合现实应用的首选软件平台。如果你想掌握混合现实开发，就意味着你首先需要掌握 Unity！我们将全面了解 Unity，并理解其用途。我还将引导你构建第一个 Unity 应用。

## 什么是 Unity？

在我们开始之前，你可能想知道 Unity 是什么以及人们如何使用它。*Unity 是一个构建 2D 和 3D 游戏与应用的强大程序*。它在游戏开发者中非常受欢迎，尤其是移动游戏开发者。Unity 还支持广泛的平台。例如：iOS、Android、PlayStation、Nintendo、Xbox One、Windows、Mac、HoloLens、Oculus 等等！

Unity 中非常基础的混合现实工作流程如下：

1.  将你的 3D 对象和其他项目（称为*资产*）导入 Unity。
2.  编程设定你将如何与对象交互，以及对象将如何与你、其他对象及世界交互。
3.  测试你的应用。
4.  导出你的应用，以便将其安装到设备上。

Unity 是一个非常庞大且精密的平台。在本书中，我们将只覆盖混合现实开发所需的 Unity 核心部分。随着你作为一名 Windows 混合现实开发者不断成长，你将花费大量时间掌握 Unity 并为 Unity 进行编程。

### 第 7 步：缩放地面

接下来，我们将学习如何通过缩放地面来*缩放*游戏对象。缩放对象意味着调整其大小。有多种不同的方式可以缩放游戏对象。这里列出了一些常见方法：

-   点击缩放图标，然后在游戏对象上点击并拖动其中一个彩色轴。拖动红色轴可在`X`方向缩放，绿色轴对应`Y`方向，蓝色轴对应`Z`方向。
-   按下键盘上的`R`键，作为点击缩放图标的快捷方式。
-   点击并拖动缩放字段的`X`、`Y`或`Z`标题。
-   直接在缩放字段中输入您想要的缩放值。

### 步骤 15：启用键盘控制

我们希望可以使用键盘作为游戏控制器来移动小球。为此，需要为小球编写一些代码！在 Unity 中，代码文档被称为*脚本*。让我们保持井井有条，创建一个新文件夹来存放脚本，就像之前在步骤 13 中创建存放材质的文件夹一样。

5.  滚动到列表底部，选择“新建脚本”。
6.  在弹出的下一个窗口中，确保脚本语言设置为 C sharp（有时也称为 C#）。
7.  将脚本命名为“PlayerController”。
8.  点击“创建并添加”按钮。

在 Visual Studio 中，请删除所有现有代码，并将其替换为代码清单 2-1 中提供的代码。

```
using UnityEngine;
using System.Collections;
public class PlayerController : MonoBehaviour {
public float speed;
private Rigidbody rb;
void Start ()
{
rb = GetComponent();
}
void FixedUpdate ()
{
float moveHorizontal = Input.GetAxis ("Horizontal");
float moveVertical = Input.GetAxis ("Vertical");
Vector3 movement = new Vector3 (moveHorizontal, 0.0f, moveVertical);
rb.AddForce (movement * speed);
}
}
```
