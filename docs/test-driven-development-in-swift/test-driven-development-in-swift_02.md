# 致谢

若非站在巨人的肩膀上，本书不可能问世。其中最高大的是肯特·贝克，他对测试驱动开发的“重新发现”开启了这一过程，最终将我们带到了这里。紧随其后的是迈克尔·费瑟斯、马丁·福勒和加里·伯恩哈特；他们的工作教导并激励了我。

站在巨人的肩膀上是一种危险的处境。幸运的是，我有守护天使防止我跌落：我的孩子们激励并帮助我脱离工作；我的妻子耐心地等待着我一次次“再给我五分钟”。

非常感谢 Apress 团队给我这个首次作者的机会。特别感谢杰西卡·瓦基利：你一直耐心且乐于助人。我深深感激卢卡·费拉里和马蒂亚·托索。十多年前，在费拉拉大学就读期间，他们找我合作一个初创想法。我们最终使用了 Ruby on Rails，也正是因此我偶然发现了测试驱动开发。还要感谢朱利奥·格里兰达、马尔科·贝尔萨尼和马泰奥·博诺拉，他们加入我们并实现了梦想。多年来，你们一直是我难以置信的朋友和支持者——抱歉我飞走了，让你们少了一双手。

自 2014 年起，我称澳大利亚墨尔本为家。在这里，我幸运地找到了一个友好热情的志同道合社群，与他们分享并打磨想法。感谢阿隆·伯里、奥黛丽·谭、马特·德尔维斯、皮特·戈德史密斯、斯图·格莱多以及 Itty Bitty Apps 团队的各位，感谢你们的对话、支持和陪伴，以及作为好朋友的陪伴。特别感谢亚当·约翰逊、查理·希尔、马丁·赫鲁和理查德·莫尔特对我早期草稿的反馈，以及萨缪埃莱·菲奥里尼最初的鼓励。我还要感谢克里斯·图米和斯蒂芬·维卡里，他们在《The Bike Shed》播客上周复一周分享关于测试的想法，我也是在那里听到了美妙的“有益压力”类比，以及本·奥伦斯坦，我从他那里学到了“神秘嘉宾”模式。感谢所有理应在这里，但因我的疏忽而忘记提及的人。

借用琳恩·特鲁斯在《吃，射与叶子》中的话，我要感谢学识渊博的文字编辑和技术编辑，他们试图理清我的写作并使我免于尴尬。凡固执地保留不足之处，皆为我一人之过。感谢每一位曾访问我博客 [`mokacoding.com`](https://mokacoding.com) 的人。你们的反馈和支持不断为我的工作注入动力。

最后，感谢您，亲爱的读者。感谢您决定拿起这本书，并分享您宝贵的时间与我相伴。
