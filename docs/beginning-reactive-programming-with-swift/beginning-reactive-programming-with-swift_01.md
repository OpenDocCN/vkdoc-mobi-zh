# 引言

随着技术不断变迁，我们观察到基本模式会反复再现。从很多方面来看，真正需要学习的新东西并不多——不过是现有技术与概念的新变体与组合而已。（关于反复出现的模式，可参阅我的著作《用 Swift 学习计算机科学》。）

一如既往，有许多人要感谢，他们为本书提供了帮助。最重要的是那些为这些技术做出贡献的人。谈到众多开源技术（包括 ReactiveX 及其项目），有越来越多的人投入其中，这让所有人都能更轻松地使用它们。

就近而言，Aaron Crabtree 对手稿提供了非常有益且细致的审阅意见。同样一如既往，Waterside Productions 的 Carole Jelen 为本书的出版提供了重要支持。
