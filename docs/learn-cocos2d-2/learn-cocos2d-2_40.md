# 排版后的文本

与其花钱雇人代劳，不如直接在互联网上寻找现成的资源（最好是免费的）。不过，对任何“免费”的东西都要多加小心。我并不是说其中暗藏陷阱；我要提醒的是那种普遍误解，即认为免费就意味着“可以随便用”。获取免费图像或音频文件可能无需任何花费，但这并不能告诉你，在法律上你拥有哪些使用权限。通常这就需要许可证来发挥作用，但实际情况往往并非如此。很多人会在互联网上免费发布自己的源代码、艺术作品、音频和文章，但如果这些作品是打算给别人用的，他们却常常忘记附上合适的许可文件。问题在于，默认情况下，作者拥有版权，并且保留对作品使用方式的所有权利。如果没有明确的豁免声明（最好是许可证形式），你就不应该使用这些作品（尤其不能用于商业产品，包括通过 iTunes App Store 售价为 `$0.99` 的应用）。

作为参考，我想向你推荐 Funplosion Labs（Funplosion 实验室），该网站有篇文章列出了你可以免费获取游戏图形和音频的网站。Funplosion 还附带了免责声明，提醒注意版权问题，并提供了每个网站的许可协议链接（见 `http://funplosion.com/free-assets.html`）。

别忘了还有绝妙的 OpenGameArt 网站（`http://opengameart.org`），在那里你不仅能找到免费的艺术作品，还能找到声音和音乐。

**注意** 要警惕通用公共许可证（GPL），尤其是当你想要使用或集成某个采用 GPL 许可证的源代码库到商业应用中时。在自己的项目中使用 GPL 许可的代码，要求你将自己项目的源代码也进行开源。不仅如此，任何其他人随后都有权使用你的源代码，并对其进行复制、修改和再分发。请注意，较宽松通用公共许可证（LGPL）就没有那么严格了。同样，还有两种知识共享（Creative Commons）许可证。`CC-BY 3.0` 是非限制性的，而相同方式共享版本的 `CC-BY-SA 3.0` 则要求你以相同许可证共享你的项目。关于常见源代码许可证的比较，请参考此链接：`http://developer.kde.org/documentation/licensing/licenses_summary.html`。

### 寻找趁手的工具

有时候，你可能希望拥有一款工具，它恰好能做某件事，无论那个“某事”是什么。作为一名游戏开发者，你时常需要处理数据、修改图像或构建整个世界——这些事情用代码来实现既繁琐又容易出错，或者完全自己动手做太耗时，因为你更愿意把精力集中在编写游戏上。

我的建议是使用 Indie Game Tools（独立游戏工具）网站，该网站负责收集和分类各类工具，并允许其他用户对它们进行评分，这些工具都是为独立游戏开发者设计的。这里的重点并非专业工作室使用的昂贵软件，而是针对几乎所有需求提供的低成本或免费解决方案，从游戏引擎到格式转换器，从脚本语言到服务器技术，从素材包到游戏编辑器。该网站由 Robert “Robc” Charney 维护，是我寻找合适工具的首选去处。请访问该网站：`http://indiegametools.com`。

有一点小小的提醒：不要过于看重单个工具的评分。评分数目非常少，因此相似工具之间可能存在巨大差异，而且评分可能会受到不满意用户和活跃社区的双重影响而产生偏差。你应该留下自己的印记，添加一些自己的评分，这样随着时间的推移，评分会变得更为准确。此外，有些工具根本无法通过评分来比较；它们可能用途完全不同，因此基于在 Indie Game Tools 网站上的排名来比较它们是不公平的。

## 市场营销

好了，你制作了一款游戏并提交到了 App Store。然后呢？人们最初是如何找到你的游戏的？

故事要回到起点。你开始着手制作游戏的那一刻，也应该是营销工作的起点。搭建并运行一个网站，发布你的开发经历，或许再放一些开发中的截图。这应该是你与其他游戏开发者建立联系的第一步。

在市场营销和商业方面，我想帮你省点时间。下面这个链接指向的是“独立游戏市场营销和商业小贴士大列表”，我得说这描述还谦虚了。你会发现，关于这个主题最深刻、最明智的文章几乎都集中在这一页上，所以即使你只是对独立游戏开发的市场营销和商业方面略感兴趣，也一定要去看看。请到 `www.pixelprospector.com/indev/2010/08/the-big-list-of-indie-marketingand-business-tips` 查看。

还对市场营销感兴趣？很好，那就去看看 Scott Steinberg 撰写的免费电子书《电子游戏市场营销与公关》。你可以在该书的网站上获取： [www.sellmorevideogames.com](http://www.sellmorevideogames.com)。

### 营销你的游戏和你自己

如果你真不确定你的游戏仅凭趣味性能否成功，那么一个提高成功几率的有效方法就是制作一款外观出众、色彩丰富、并且仅仅在截图中就看起来乐趣无穷的游戏。由于移动设备上应用的低定价降低了准入门槛，一款画面精美但游戏深度有限的游戏，往往比那些拥有多种游戏模式和数小时潜在游戏时长的复杂游戏表现更好、销量更高。这可能会让一些开发者感到痛苦，因为它违背了他们的信念，但这是生活中无法回避的事实。当然，如果你能将两者结合起来，创作出一款既看起来很棒又富有创新、充满乐趣的游戏，那么机遇将是无限的。

如果你浏览游戏评测，有一件事需要特别关注。那些最显眼的截图总是游戏中的动作场景，通常同一时间有很多事情在发生。这几乎已经成为一种艺术形式；专业开发者甚至会开发专门的工具来布置场景，以便能截取最佳的动作截图，而无需亲自玩游戏，靠运气来抓取完美瞬间。或许你也应该考虑一下这一点。因此，你能创作或付费购买的最佳艺术作品绝对应该是优先事项，制作一个令人信服的游戏宣传片也同样重要。出色的呈现效果也是吸引出版商与你洽谈的非常重要的敲门砖。

这条建议同样适用于你的博客。你确实有博客，对吧？如果没有，现在就开一个！你可以从 `http://wordpress.com` 免费获取博客，并在过程中学会如何使用这个最流行的博客软件。随着博客的成熟，你可能会考虑在自己的服务器上托管博客；你同样可以使用 WordPress 来实现，但通过插件和主题会有更多定制选项。

你的博客是营销自己的重要途径。这里最重要的一点是，自我营销会间接地让你从今往后所做的任何事情都受益。写博客能让你接触到其他志同道合的人，他们迟早会愿意帮助你，有时甚至是免费的。如果你真的在你的开发者社区中闯出了名堂，那是个好消息，因为这能吸引那些关注你所在渠道的出版商的注意。请注意，这与你的大多数游戏玩家不同，你需要以另一种层次去接触他们。

因为博客应该是你的公开形象，所以发表言论的也应该是你本人。不要戴上伪装，试图听起来像个大公司——例如，说你正在做的事情将彻底改变我们玩游戏的方式。我称之为……这个词我不会写在这本书里，但我相信你明白我的意思。
