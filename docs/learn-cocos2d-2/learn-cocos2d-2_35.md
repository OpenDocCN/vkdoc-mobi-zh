# 第 16 章 Kobold2D 简介

在与 cocos2d 合作的两年多时间里，我发现了大量改进 cocos2d 开发工作流程的机会和需求，包括添加有用的代码片段、扩展 cocos2d 类、实现更简单的跨平台开发、自动化重复性任务、简化 cocos2d 升级过程，以及提供完整准确的文档。最终，这些都汇聚成了 Kobold2D，一个仍然基于 `cocos2d-iphone` 但改进了 cocos2d 开发者工作流程的游戏引擎。

Kobold2D 的目标是让新开发者更容易使用 cocos2d，同时满足我自身对专业工作环境的需求（也希望满足你的需求！）。

Kobold2D 将 cocos2d 引擎与许多开发者过去一直在使用的流行库合并在一起。其中一些库几乎已经成为必备的附加组件，包括 Lua、新的 `cocos2d-iphone-extensions` 项目、SneakyInput、Chipmunk SpaceManager、ObjectAL 和 iSimulate。Kobold2D 使所有开发者都能轻松访问和使用这些库——你只需要导入库的头文件并调用库的方法。
