# 我在撰写本书的过程中学到了很多，而在花费数月时间将各章节更新至第二版乃至第三版时，我学到的甚至更多。我最大的愿望莫过于你能通过这本书深入了解 cocos2d 和游戏开发。

在编写关于 `cocos2d` 的内容时，我领悟到，我可以为 `cocos2d` 提供一种解决方案，消除 `cocos2d` 开发者最初面临的大部分障碍。其成果便是 `Kobold2D`，你将在第 16 章以及 `www.kobold2d.com` 网站上找到关于它的介绍。它与 `cocos2d` 的不同之处在于，它自带安装程序、完备的文档、集成了重要的库、已启用 ARC、提供了十几个示例项目以及更多额外功能。显然，我编写第三版的目标之一就是使其与 `Kobold2D 2.0` 100% 兼容。

**Steffen Itterheim**
