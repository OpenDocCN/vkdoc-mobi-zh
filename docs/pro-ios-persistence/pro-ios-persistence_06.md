# 引言

世界已经进入移动时代，而 iOS 应用似乎引领着这一潮流。人们使用 iPhone、iPad，很快还将包括 Apple Watch 来消费、生产和存储多样化的信息。当你在这些苹果设备上开发应用时，你会欣赏到苹果在其持久化框架 Core Data 上所展现的精湛工艺。没有其他移动平台能提供在强大性、易用性或灵活性上比肩 Core Data 的持久化层。随着 iOS 软件开发工具包的每次发布，苹果都在发展和改进 Core Data 框架。

随着 Core Data 的持续发展，我们已将本书撰写并重写了多次。去年夏天，就在本书即将完成之际，苹果宣布了其新的 Swift 编程语言，让我们所有人都感到惊讶。经过我们之间的讨论以及与 Apress 同仁的商议，我们得出结论，应对 Swift 问世的恰当做法是重新编写，并将本书中的每个示例都用 Objective-C 和 Swift 两种语言呈现。当然，这次经历给我们留下了一些“伤痕”，所有 Swift 的早期采用者都可以作证。随着每个新版本 Xcode 的发布，我们不得不重新审视那些我们认为已经完成的章节，因为有些代码片段在新版 Xcode 下不再能编译通过。我们相信已经捕获了所有错误的 `?` 和 `!`，但如果有遗漏，我们在此预先致歉。

## 你需要准备什么

要跟随本书学习，你需要一台运行 OS X Mavericks 或 OS X Yosemite 的 Mac 电脑，并且需要 Xcode 6。如果你运行的是 Xcode 6 的最新点版本（尤其当你使用 Swift 代码示例时），你的体验会最好。Xcode 的每个新版本都在不断改进对 Swift 语言的支持。你可以从 Mac App Store 下载最新版本的 Xcode 6。

本书不涵盖 iOS 开发的入门知识（无论是 Objective-C 还是 Swift），因此你应至少具备 Objective-C 或 Swift 的基础知识，同时了解 Cocoa Touch 和 iOS 开发。

## 你将发现什么

本书从理论和实践两个角度探讨 iOS 持久化与 Core Data，因此你将了解 Core Data 的一般工作原理及其各个部分的组合方式，同时还将获得分步指导，了解如何将这个强大的持久化框架整合到你的应用中。你将看到我们如何用 Objective-C 和 Swift 实现所讨论的每个主题。我们涵盖了一系列主题，包括如何构建 Core Data 数据模型、如何查询数据、如何在模型版本间迁移数据、如何加密数据、如何在不同线程上使用 Core Data、如何在云端存储数据，以及如何调整性能。

你可以从头到尾通读本书，但为了获得最佳效果，我们建议你在电脑前阅读，亲手输入代码示例，以理解它们的工作原理和功能。我们还建议你将本书作为应对持续遇到的 Core Data 需求和挑战的参考书。

## 本书的组织方式

我们将本书组织为从基本原理逐步深入到高级主题，因此从头到尾通读本书，你可能会收获最大。不过，各章的标题清晰地标明了其所包含的主题，因此如果你正在寻找特定主题的信息，可以直接跳到该章。
