# 排版后内容

最后，我想用一段话献给本书的合著者，他至少在公开场合拒绝承认自己很出色。罗伯·华纳热爱创作高质量的作品。他痴迷于软件设计和语言之美，甚至能将代码转化为散文。我感谢他让我有机会再次与他合作。

——迈克尔·普里瓦

我的女儿莱拉多次提醒我，她应该在这本书中得到比过去作品中更多的“主角”待遇，所以这是我向她和我其余家人展示他们在我的生命中是多么闪耀的星星的机会。我爱我美丽的妻子雪莉，以及我出色的孩子们：泰森、雅各布、马洛里、卡米和莱拉。我感谢他们的鼓励和支持——即使这种鼓励的表现形式是叫我“书呆子”。我无比幸运，你们都在我的生命里。

我也感谢我的父母，他们为我们买了第一台电脑（一台雅达利 400），鼓励我们学习，并且会购买我写的每一本书，尽管他们从未读过。我的兄弟姐妹和姻亲们总是很擅长询问书的进展，并礼貌地对书中的主题表现出兴趣。我有一个很棒的家庭。

与 Apress 的出色同仁们一起，这又是一次极好的写作经历，我感谢他们的耐心和洞察力。将一本书付诸实践所需要的庞大团队从未如此显而易见，因此我现在比以往任何时候都更清楚，有许多人为这本书的出版投入了默默无闻的时间和精力。特别感谢史蒂夫·安格林、吉尔·巴尔扎诺、安娜米卡·潘丘、马修·穆迪、道格拉斯·庞迪克、罗恩·纳塔莉，以及所有其他参与本书出版工作的人。

同样感谢我们所有的读者提出的问题、反馈、赞扬和批评。我们倾注心血，尽力提供最好的信息，并且一直在寻求改进内容或使其更易于理解的方法。

能与迈克尔再次合作另一个项目，是我莫大的荣幸。我始终希望他不要对我感到厌倦，因为每次与他合作我都学到很多。我感谢他的耐心、奉献、坚持和专业精神。

我效力于一家很棒的公司。Availity 给了我每天创造价值、做出改变的机会。与如此多的聪明人共事，让我时刻保持警醒！当然，篇幅所限，我无法一一列出他们的名字，但我要特别感谢领导团队（特伦特·加瓦齐、杰克·亨特、玛丽·安妮·奥伦丘克、塔琳·特雷斯卡和史蒂夫·沃恩）的支持。也感谢杰夫·帕克伍德定期检查，确保我们朝着目标稳步推进。我还每天与一个出色的团队——洛里·克里尔、玛丽·霍恩、罗伯特·麦吉尼斯、罗伯特·文特罗内和安伯·威尔逊——一起工作，努力创造卓越。

——罗伯·华纳
