# 关于技术审校者

![9781430260288_unFigFM-03.jpg](img/9781430260288_unFigFM-03.jpg)

**罗恩·纳塔莉**拥有 35 年在 Unix、OS X 和 Windows 上使用 C、Objective-C 和 C++ 开发大型应用的经验。他拥有约翰·霍普金斯大学的电气工程学位，并教授过编程、网络设计和计算机安全方面的专业课程。他大部分时间与妻子玛吉在弗吉尼亚州和北卡罗来纳州之间度过。

![9781430260288_unFigFM-04.jpg](img/9781430260288_unFigFM-04.jpg)

**伊桑·马特哈**是一名软件工程师，目前担任 Ubiquiti Networks 公司的 iOS 移动架构师，并为广泛的客户提供私人咨询支持。作为一名拥有 12 年经验的全栈工程师，伊桑的经验涵盖了为美国国防部编写服务器端软件，以及为 Actian Inc、Paypal、Salesforce 和湾区其他初创公司承担重要的移动项目。作为丈夫和两个孩子的父亲，伊桑喜欢住在圣马特奥县红杉林的中心地带，在那里，他对自然的好奇和惊叹是日常生活的一部分。他认为自然环境激发了他对解决问题的热爱，他经常在树林中长途徒步，以思考最具挑战性的工程难题。
