# 关于作者

![9781430260288_unFigFM-01.jpg](img/9781430260288_unFigFM-01.jpg)

**迈克尔·普里瓦特**是位于佛罗里达州杰克逊维尔的 Availity, LLC 公司的高级开发人员。他负责监督构成交易处理平台各个组件的设计与实现。他曾合著《Pro Core Data for iOS》（Apress, 2011）和《Beginning OS X Lion Apps Development》（Apress, 2011）。他发布了多个 iOS 应用，包括 Ghostwriter Notes、Budget Tracker 和 Fluid Notes。他在法国尼斯大学获得计算机科学工程学位，之后加入麻省理工学院从事应用人工智能研究。他目前与妻子凯莉以及孩子马蒂厄和克洛伊居住在法国。

![9781430260288_unFigFM-02.jpg](img/9781430260288_unFigFM-02.jpg)

**罗布·华纳**是位于佛罗里达州杰克逊维尔的 Availity, LLC 公司的高级技术职员，他与各种技术打交道，为医疗保健领域提供解决方案。他曾合著《Pro Core Data for iOS》（Apress, 2011）、《Beginning OS X Lion Apps Development》（Apress, 2011）以及《The Definitive Guide to SWT and JFace》（Apress, 2004）；他在 `www.grailbox.com` 上写博客。他在杨百翰大学获得英语学士学位，并正在佐治亚理工学院攻读计算机科学硕士学位。他目前与妻子雪莉以及五个孩子泰森、雅各布、马洛里、卡米和莱拉居住在佛罗里达州杰克逊维尔。
