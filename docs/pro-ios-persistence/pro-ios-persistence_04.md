# 致谢

写书是一项耗时费力的工作，会占用大量本应陪伴家人的时间。这是我的妻子凯莉和我的两个孩子马蒂厄和克洛伊允许我做出的牺牲，并承诺等项目完成后我会弥补他们。我非常感谢他们的耐心，以及允许我完成这个项目。我现在意识到，弥补的时间到了，对此我感到非常兴奋。每一版我们都会遇到新读者，他们直言不讳地告诉我们喜欢什么、不喜欢什么，并希望在下个版本中看到什么。我感谢那些在谈论这本书时表现出极大热情的人，我也感谢所有带来想法帮助我们每版改进的人。像这样的项目起初往往显得令人望而生畏。我们越是深入思考各个章节，决定如何组织它们，并推测所有需要阐述的内容，就越感觉到自己分身乏术。但出色的 Apress 团队全程给予我们支持。他们通过提供指导、帮助我们组织工作以及提供所需的协作工具，真正地减轻了我们很多负担。他们真是一家新一代的出版公司。没有我所工作的公司 Availity 的支持，这本书就不可能存在。从领导层的每个人，包括 Trent Gavazzi、Mary Anne Orenchuk、Jack Hunt 和 Steve Vaughn，他们经常来询问这本书的进展。我们的一些工程师也乐于提出建议并表达支持：Rob McGuinness、Ed McNeil、Herve Devos、Kelly Burton。与他们共事非常愉快，我感谢他们所有人的支持。
