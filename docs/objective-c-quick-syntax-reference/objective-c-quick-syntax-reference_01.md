# 关于作者

![A978-1-4302-6488-0_BookFrontmatter_Fig1_HTML.jpg](img/A978-1-4302-6488-0_BookFrontmatter_Fig1_HTML.jpg)  
马修·坎贝尔是一名专业软件开发人员、企业家、作者和培训师。他就职于 Mobile App Mastery，这是他于 2008 年创立的一家基于网络的软件开发培训公司。在创建 Mobile App Mastery 之前，马修学习了心理学，曾担任心理健康顾问，并在普林斯顿的教育考试服务中心担任数据分析师，支持心理测量研究。他创作的书籍和培训课程旨在消除阻碍开发者精通技艺的障碍。
