# 引言

Objective-C 是一种工具，可用于为 Mac、iPhone 和 iPad 创建出色的应用程序。这种独特的编程语言源于 C 编程语言。Objective-C 是增加了面向对象编程特性的 C 语言。

如今，学习编程就是学习如何塑造我们的世界。Objective-C 程序员处于一个独特的位置，可以创建全球人们在日常生活中都能使用的移动应用程序。

使用 Objective-C 是一种愉悦的体验。当其他编程语言有时显得笨拙时，Objective-C 会以优雅的方式展示其力量和影响范围。在其他编程语言中看似棘手的问题，在 Objective-C 中会迎刃而解。

本书的核心在于，毫不赘述地阐明 Objective-C 能够做什么。当您知道想要做什么，但只需要了解 Objective-C 的实现方式时，请使用本书获取帮助。
