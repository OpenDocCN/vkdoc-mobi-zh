# 21. 键值编码

概要

通常，当你想获取或设置对象中的属性值时，你使用点符号来获取对属性的引用以更改值。然而，通过键值编码，你可以使用字符串键来间接存储和检索属性值。需要归档的应用程序需要这种功能，以便应用能够从归档文件中检索对象数据。

## 键值编码定义

通常，当你想获取或设置对象中的属性值时，你使用点符号来获取对属性的引用以更改值。然而，通过键值编码，你可以使用字符串键来间接存储和检索属性值。需要归档的应用程序需要这种功能，以便应用能够从归档文件中检索对象数据。

### 设置属性值

你可以使用键值编码来设置属性值。要设置属性值，你必须使用`setValue:forKey:`消息，并提供新的属性值以及该属性的`NSString`名称。

```
[p setValue:@"New Project" forKey:@"name"];
```

### 检索属性值

要使用键值编码检索属性值，你可以直接向对象发送`valueForKey`消息。该消息需要以`NSString`格式的属性名称作为参数。下面演示了如何从`Project`对象中检索`name`属性值：

```
NSString *retrievedName = [p valueForKey:@"name"];
```

这适用于任何类型的对象，包括数据集合对象和自定义对象。现在，你可以使用`NSLog`函数将这个值打印到日志中。

```
NSLog(@"retrievedName = %@", retrievedName);
```

如果 Project 的名称是 New Project，执行上述代码你将得到以下输出：

```
retrievedName = New Project
```
