# 引言

大家好，欢迎来到 Android 游戏开发的世界。你们来到这里是为了学习 Android 上的游戏开发，我们希望自己能够成为帮助你们实现想法的人。

我们将一起涵盖相当广泛的内容和主题：Android 基础知识、音频和图形编程、一点数学和物理知识、OpenGL ES、Android 原生开发套件（NDK）入门，最后是游戏的发布、营销和盈利。基于所有这些知识，我们将开发三款不同的游戏，其中一款甚至是 3D 的。

如果你知道自己在做什么，游戏编程可以很轻松。因此，我们力图以这样一种方式呈现材料：不仅为你提供有用的可复用代码片段，而且还能让你看到游戏开发的整体图景。理解底层原理是应对日益复杂的游戏创意的关键。你不仅将能够编写出与本书中开发的游戏相似的作品，还将具备足够的知识，可以自行上网或去书店探索游戏开发的新领域。

## 本书面向的读者

本书首先面向的是游戏编程的完全初学者。你不需要任何相关领域的先验知识；我们将带你逐步了解所有基础知识。然而，我们需要假设你具备一点 Java 知识。如果你觉得自己在这方面有些生疏，我们建议通过阅读布鲁斯·埃克尔（Prentice Hall，2006 年出版）的《Thinking in Java》来巩固记忆，这是一本优秀的编程语言入门教材。除此之外，没有其他要求。无需预先接触 Android 或 Eclipse！

本书也面向希望涉足 Android 的中级游戏程序员。虽然其中一些内容对你来说可能是老生常谈，但书中仍包含许多提示和建议，值得一读。Android 有时是一个奇怪的家伙，而这本书应该被视为你的战斗指南。

## 本书的结构

本书采用循序渐进的方法，我们将稳扎稳打，从最基础的知识逐步深入到硬件加速游戏编程的精妙境界。在本书的进程中，我们将构建一个可复用的代码库，你可以将其作为大多数类型游戏的基础。

如果你纯粹是为了学习而阅读本书，我们建议你按顺序阅读各章，从第 1 章开始。每一章都建立在上一章的基础上，这会带来良好的学习体验。

如果你阅读本书的目的是在最后发布一款新游戏，我们强烈建议你直接跳到第 14 章，学习如何设计你的游戏使其具备市场潜力和盈利能力，然后再回到开头开始开发。

当然，经验更丰富的读者可以跳过他们已熟练掌握的某些章节。只需确保通读你跳过的章节中的代码列表，以便了解这些类和接口在后续更高级的章节中是如何使用的。
