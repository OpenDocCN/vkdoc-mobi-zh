# 致谢

每位作者写书都是为了讲述故事，无论是为了娱乐、教育，还是为读者提供介于两者之间的体验。经过过去几年的写作，我意识到写作过程本身也构成了一个故事。本书的写作过程比前两本更为曲折，但在此，我想感谢所有参与其中的各位。

首先，我要感谢我的编辑 Jessica Vakili 和 Aaron Black，他们非凡的耐心和支持使我得以完成这部作品。我真诚地为可能给你们带来的任何麻烦道歉，但我为我们共同创作出的成果感到自豪。

接下来，我要感谢本书第一版的原合著者 Gheorghe Chesler 和 Manny de la Torriente。除了共同写作，我们在协作中的经历极大地塑造了我作为开发者的成长，并留下了许多至今仍令我怀念的美好回忆。第一版始终是我引以为傲的作品，尤其是我们在 Swift 和物联网仍充满不确定性的时代完成了它！

最后，我必须感谢我杰出的家人和朋友，他们一直支持我，甚至包容我移居日本的疯狂冒险。他们帮助我意识到自己正在追寻新的挑战，并不断鼓励我坚持克服其中的每一个困难。谢谢大家！

对所有我可能遗漏的人，请别忘了“花之飞鸟组”的传奇，我很快会回来，与你们在博纳旺蒂尔展开下一次冒险！
