# 引言

欢迎回到物联网世界。2016 年，我与最初的合著者 Gheorghe Chesler 和 Manny de la Torriente 共同撰写本书第一版时，我们将职业生涯、咨询项目及梦想中的收获融汇在一起，希望构建一个能帮助未来开发者使用 Swift 为 iOS 编写物联网应用的故事架构。如今浏览 Apress 网站，发现仅通过该网站就有超过三万人阅读了本书，我很高兴至少触及了部分读者！

然而，时间在向前推进。自第一版出版以来，Swift 和物联网都经历了巨大演变。我们最初想写的一些技术终于成为现实，而另一些则已被淘汰。最重要的是，Swift 作为一种语言终于开始趋于稳定，开发者社区也逐步建立起一套更清晰、适合该语言的设计模式和编码标准。除了最初版本中的 Swift 版本（2.0）与现代语言版本不兼容之外，彻底修订本书的时机也已成熟。

本版在回顾第一版中备受喜爱的内容与呈现上一版未涉及的新概念之间取得了平衡。我删减或精简了那些逐渐消沉的内容，并大幅扩展了那些被证明比以往更重要的部分。特别是，本书增加了更多关于构建硬件项目的内容，因为高质量、价格合理的零部件已迅速普及，且使用难度也大幅降低。此外，我注意到，总体上有更多人希望自行构建硬件及其配套的应用。

同时，我努力让本版内容更便于更多读者理解。虽然我写的每一本书都面向具备 iOS 开发基础知识的开发者，但本卷我调整了格式，以反映在教学和与读者交流中收到的反馈。也就是说，本版允许你在不同主题间更灵活地跳跃，并提供了更详细的逐步指导。我不再仅仅提供可复制的代码，而是尝试解释完成项目中每个任务所需整合的内容。希望这能让你更轻松地调试问题，并使本书在最初编写时所针对的 Swift 版本（Swift 4.2）之后仍能持续使用。

虽然无法用一段话全面描述物联网，但本书聚焦于如何构建、交互及联网硬件传感器，从而让用户的生活更便捷、信息更丰富。在家中，我可以用语音关闭灯光，判断是否在电价过高时使用电器，以及确认泡茶水温是否完美（温度确实完美，但我需要记得在定时器响起时起身）。在本书中，你将实现的项目包括更精准地记录锻炼情况、确认是否锁门，以及在 Apple TV 上展示家庭仪表盘。对于 DIY 项目来说，这相当酷！

最后，请记住，这本书为你而存在。写作的最终目标是帮助你及开发者社区更深入地理解与 iOS 相关的物联网开发，并为你未来的项目提供丰富的灵感。如果你希望确保代码保持更新，请向本书的 GitHub 仓库提交拉取请求（[`https://github.com/Apress/program-internet-of-things-w-swift-for-ios`](https://github.com/Apress/program-internet-of-things-w-swift-for-ios)）。如有任何反馈，我期待通过 Apress 或我的网站（[`www.devatelier.com`](http://www.devatelier.com)）收到。祝编程愉快！
