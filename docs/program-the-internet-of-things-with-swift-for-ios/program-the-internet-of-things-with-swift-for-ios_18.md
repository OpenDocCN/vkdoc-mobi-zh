# 总结

在本章中，你学习了如何运用 iOS 编程知识，为 `IOTHome` 硬件创建一个 tvOS 仪表板应用。尽管不得不为 tvOS 做出一些特殊处理，但其开发流程和可用的 API 与 iOS 中的对应部分非常相似。通过将 tvOS 应用作为目标添加到 `IOTHome` 项目中，你甚至能够与 iOS 应用共享代码，包括 `Network Manager`。为了让应用锦上添花，你还学习了如何为视图应用视觉效果，使用 `Font Awesome` 实现基于字体的图标，以及使用 `OpenWeatherMap.org` 获取公共天气数据。

构建这款应用的过程充满乐趣，同时也让你有机会让 `IOTHome` 传感器的数据对用户更有用、更易于访问。在为部署物联网传感器和大数据的公司提供咨询时，我反复遇到的最大挑战是如何利用所有已生成的数据。通过向用户展示其数据的可视化效果，并补充其他有用的关联信息，你得以减少使用硬件的障碍，并提升其对用户的价值。通过打造改善用户生活质量的系统，你可以帮助物联网设备从“玩具”转变为“工具”。
