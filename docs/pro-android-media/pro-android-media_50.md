**手指绘画**

在 `Bitmap Canvas` 上创建静态绘图固然很好，但让我们更进一步，探索如何制作一个允许用户创建绘图的应用程序。

**触摸事件**

要开始这个应用程序，我们需要了解 Android 如何告诉我们用户已触摸触摸屏。能够处理此事件后，我们就可以允许用户使用手指在触摸屏上创建绘图。

众所周知，我们在 Android 中使用的许多 UI 元素都派生自 `View` 类。由于我们一直在使用显示在 `ImageView` 中的 `Bitmap` 中的 `Canvas`，因此看看这是否有助于我们检测用户触摸的位置是合理的。

幸运的是，`View` 类支持触摸。它有一个方法，我们可以指定哪个类应接收其获得的任何触摸事件。此方法是 `setOnTouchListener`，它接收一个实现了 `OnTouchListener` 接口的对象。

我们可以通过声明我们的 Activity 实现 `OnTouchListener` 并创建一个名为 `onTouch` 的方法来实现这一点。
