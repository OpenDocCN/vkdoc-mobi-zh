# 致谢

**关于作者**

**肖恩·范·埃弗里** 经营着一家移动和流媒体咨询公司，帮助公司更好地利用与音频和视频相关的新兴技术，重点关注移动和流媒体应用。他的客户包括 19 Entertainment、MoMA、迪士尼，以及摩根士丹利、雷曼兄弟、纽约大学医学院，以及无数初创公司和其他小型客户。

此外，肖恩还是纽约大学互动电信项目中的传播学兼职助理教授。他的教学内容丰富多样，涵盖参与式与社交媒体、编程、移动技术以及交互式电话。2008 年，他因教学卓越而荣获大卫·佩恩·卡特奖。

他曾在众多会议和技术演示中展示、展出并发表演讲，包括 O'Reilly 的 Emerging Telephony、O'Reilly 的 Emerging Technology、ACM Multimedia、Vloggercon 和 Strong Angel II。他曾是开放媒体开发者峰会、Beyond Broadcast（2006 年）及 iPhoneDevCamp NYC 的联合组织者。

肖恩拥有纽约大学互动电信硕士学位，以及布法罗大学媒体研究学士学位。

**关于技术审阅者**

**史蒂夫·布尔** 自他在帕洛阿尔托的保罗·艾伦区间研究公司工作以来，一直从事移动设备的编码与操控。作为一名混合媒体技术艺术家和企业家，布尔在过去九年里创建了基于特定位置的叙事和游戏，探索手机在社会、技术和创意方面的可能性。可通过[www.stevebull.org](http://www.stevebull.org)与他联系。


  
**华莱士·杰克逊**是一位经验丰富的多媒体制作人和 i3D 程序员，专注于 Acrobat3D PDF、Android 移动应用、iTV 设计、JavaFX 和 JavaTV 领域。自 Atari ST1040 和 AMIGA 3000 时代起，他就开始设计富媒体内容，并且从近二十年前*《多媒体制片人》*杂志创刊以来，他一直为领先的多媒体出版物撰写关于新媒体内容开发的文章。您可以通过[www.wallacejackson.com](http://www.wallacejackson.com)与他联系。
