# 垃圾回收

`垃圾回收`（`Garbage collection`）是另一种用于自动化内存管理的内存管理系统。该方法引入了一个垃圾回收器的概念，它定期查找不再被使用的对象并将其回收。

`垃圾回收`仅适用于自版本 `10.5` 开始的 `Mac OSX`。注意，`iOS` 应用程序不能使用垃圾回收。

在垃圾回收中，对象可以与根对象建立强引用或弱引用。根对象通常是一个顶层的应用程序对象。例如，Mac 应用程序有一个类型为 `NSApplication` 的对象，它负责整个应用程序。其他所有对象都包含在 `NSApplication` 对象中，使得 `NSApplication` 对象成为根对象。对根对象持有强引用的对象可以被根对象访问到，而持有弱引用的对象则无法被根对象访问到。

垃圾回收机制会定期激活，任何持有弱引用的对象都会被销毁。
