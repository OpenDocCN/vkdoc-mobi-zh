# 致谢

在我生命这段最具挑战性又收获满满的时期，有太多我想要感谢的人，多到或许我能为此写一本书（哈哈！）。但为了不失去读者的兴趣，我将仅简要提及几位。如果遗忘了任何人，请原谅我；我编写应用就是为了弥补我的健忘。

首先，我要感谢我的家人——穆罕默德、莱拉和罗巴·巴克尔，是他们一直推动我去探索自己的全部潜能。

接下来，我要感谢 Apress 出版社的米歇尔·洛曼，她在 Macworld 展会上找到了我，并亲切地询问我是否有兴趣写一本书。我永远不会忘记我们关于睡眠不足的谈话。

凯文·沃尔特，我的协调编辑，他肩负着不断提醒我完成章节这一艰巨任务，但每次交流他总是带来积极的正能量。

罗素·道格拉斯，我的技术编辑，以及莎伦·威尔基，我的文字编辑，她们帮助我找到了写作的风格，并纠正了我所有多余的逗号。

格奥尔基·切斯勒，我的技术导师，他教会了我整洁代码（及 Perl 高尔夫）之美。他作为本书技术审校者的投入，帮助我将思路梳理成他人可用的作品。

迈克·麦肯齐，我的商业导师，也是我的首批客户之一，他总是在奥兰治县为我提供宝贵的建议和美好的时光。

拉斐尔·罗德里格斯和维拉福·森苏里亚，devAtelier LLC 的早期两位成员，他们持续的勤奋、尽职和积极态度极大地塑造了公司的形象。一位高雅的设师（拉斐尔）和一位卓越的工程师（维拉福），他们是我不可思议的伙伴和知己。

卡罗尔·米勒，我在北卡罗来纳州立大学时首批计算机科学教授之一，她引人入胜的讲座和关于“危机鸡”的故事让每一堂课都充满影响力和启发性。

最后，我要感谢所有一直支持我、鼓励我，并有耐心（大多数时候）倾听我疯狂故事的朋友们。再见啦，太空牛仔们。
