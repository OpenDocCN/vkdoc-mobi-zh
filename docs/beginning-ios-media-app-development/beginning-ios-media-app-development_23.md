# 关于作者

**艾哈迈德·巴克尔**是 devAtelier LLC（`www.devatelier.com`）的创始人兼首席开发人员，这是一家位于圣地亚哥的移动开发公司。在花费数年时间为嵌入式系统编写软件之后，他开始出于兴趣在咖啡店里开发应用程序。消息传开后，他开始承接客户，并辞去日常工作，全职投入应用程序开发。自那时起，他参与了超过 20 个移动项目的开发，并见证了多个项目进入应用商店前 25 名，其中一个甚至在其类别中达到了第一名（视频调度器）。他的客户范围从充满活力的初创公司到大型企业，例如思杰。在业余时间，艾哈迈德会在旅途中探索新地方，谈论移动开发，并且仍然在咖啡店里工作。
