# 引言

你是否还记得第一次在电脑上体验多媒体的情景？是在 Apple II 电脑上《俄勒冈之旅》那精心编排的小镇音乐？还是 YouTube 上那只才华横溢的猫在键盘上弹奏动人曲调的视频？

那你开始学习编程时呢？是为了制作游戏？还是为了模拟学校铃声的确切音调以便提前下课？

对许多人来说，多媒体一直是计算中最令人兴奋和个性化的方面之一。对于我们中的一些人（包括我自己），它是学习编程的灵感源泉。随着计算向移动设备发展，多媒体的魅力并未改变，但幸运的是，现在开始制作你自己的多媒体应用比以往任何时候都更容易。

iOS 提供了一组极其深入的 API（应用程序编程接口），允许你在应用中显示和捕捉照片、视频和音频。这些 API 直接内置于 Cocoa Touch 中，这意味着你无需包含任何外部库即可使用它们。对于许多 API，包括相机在内，在你的应用中暴露的捕捉界面与系统全局使用的界面相同。相比“美好的旧时光”，这是一个巨大的进步，那时集成相机可能需要与元件制造商交涉数周，并移植示例代码。

本书的目标是帮助你利用已有的 iOS 应用开发知识，将其应用于媒体应用开发。Cocoa Touch 的妙处之一在于，Apple 为你抽象了大量底层功能。现在，你可以构建能播放视频的应用，而无需成为视频编解码器的专家。

通过构建专注于特定 API 的应用，并在此过程中学习新的编程概念，你将循序渐进地取得进展，并避免被 iOS 中看似无穷无尽的 API 所压倒。本书的各个单元（照片、视频、音频）会先展示如何开始使用相关 API，然后逐层深入，让你能够定制应用，超越开箱即用的基本功能。

在 iOS 开发之旅中，你已经取得了不少成就，我很高兴你决定开始探索多媒体功能。让我们一起迈出下一步，构建一些出色的媒体应用吧！
