### PCI 总线

`PCI`是`Peripheral Component Interconnect`（外设组件互连）的缩写，这是一种由英特尔开发的局部总线*标准*。`PCI`是一个 64 位总线，尽管通常以 32 位总线实现。它可以在 33 或 66 MHz 的时钟速度下运行。在 32 位和 33 MHz 下，其吞吐率可达 133 MBps。`PCI`是一种同步总线架构，所有数据传输都相对于系统时钟（`CLK`）执行。虽然由英特尔开发，但`PCI`不特定于任何微处理器系列。

`PCI`实现了一个 32 位的多路复用`地址`和`数据`总线。多路复用的`地址`和`数据`总线减少了`PCI`连接器上的引脚数量，从而降低了`PCI`组件的成本和封装尺寸。`PCI`总线周期通过在一个称为地址阶段的第一个时钟边沿将地址驱动到`AD[31:0]`信号上来启动。地址阶段由`FRAME#`信号的激活来指示。下一个时钟边沿开始第一个或多个数据阶段，在这些阶段中数据通过`AD[31:0]`信号进行传输。

在`PCI`术语中，数据在作为总线主控的发起方和作为总线从属的目标之间传输。`PCI`的较高速度将单条总线上的扩展插槽数量限制为最多四个。为了允许具有超过 3 或 4 个插槽的扩展总线，`PCI SIG`定义了一种`PCI 到 PCI 桥接`机制。`PCI 到 PCI 桥接器`是`ASIC`，它们在电气上隔离两条`PCI`总线，同时允许总线传输从一条总线转发到另一条总线。每个桥接设备都有一个“主”`PCI`总线和一条“从”`PCI`总线。多个桥接设备可以级联，以创建具有多条`PCI`总线的系统。
