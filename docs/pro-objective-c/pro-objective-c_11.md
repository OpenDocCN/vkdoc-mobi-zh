# 关于技术审阅者
![image](img/9781430250500_unFM-02.jpg)

**费利佩·拉索·马塞蒂** 是一名 iOS 程序员，就职于 Lextech Global Services 公司。他热爱与苹果相关的一切，同时也喜欢电子游戏、烹饪以及演奏小提琴、钢琴或吉他。在业余时间，费利佩喜欢阅读和学习新的编程语言和技术。

费利佩喜欢在自己的博客 `http://iFe.li` 上撰写文章，作为 Ray Wenderlich 博客团队的一员在 `www.raywenderlich.com` 上创作 iOS 教程和文章，并且担任 Objective-C 和 iOS 相关书籍的技术审阅者。你可以在 Twitter 上通过 `@Airjordan12345` 找到他，在 Facebook 上搜索他的名字，或者在 App.net 上通过 `@iFeli` 找到他。
