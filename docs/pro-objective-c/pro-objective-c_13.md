# 简介
Objective-C 编程语言的受欢迎程度和使用率持续增长。这得益于语言本身的强大和易用性，以及不断添加到平台上的众多功能。许多程序员已经掌握了该语言的基础知识，现在希望进一步提升他们的专业技能。《Pro Objective-C》将带你进入下一个层次。

## 本书内容
*《Pro Objective-C》* 提供了一本关于该语言、其运行时和关键 API 的深入、全面的指南。它以清晰、易于理解的方式解释了 Objective-C 的关键概念，并详细介绍了其更复杂的特性。其主要主题包括：

*   Objective-C 基础知识和关键语言元素
*   Objective-C 运行时系统
*   Foundation 框架 API
*   Objective-C 高级语言特性

本书还包含大量实用示例——代码片段和完整的应用程序——展示了如何将你所学的内容应用到实际代码中。

每个主题都清晰、简洁地进行了讲解，并包含了你有效开发 Objective-C 代码所需的细节。最重要的特性都得到了深入的探讨，并且每章都包含大量示例，清晰地展示了 Objective-C 的用法。

## 本书读者对象
本书面向已有一定 Objective-C 经验的中高级开发者。对于没有使用过 Objective-C，但具备一些 C 语言编程经验并理解面向对象编程的开发者来说，本书也非常适合。

## 你需要准备什么
在开始为 Apple OS X 和 iOS 平台编写 Objective-C 代码之前，你需要一台运行 OS X Mountain Lion（OS X 10.6.8）或更高版本的基于 Intel 的 Mac 电脑（MacBook、iMac、Mac Pro 等）。你还需要 Xcode，即苹果公司用于 iOS 和 Mac 软件开发的工具集。第 1 章 提供了获取和安装 Xcode 的说明。

## 本书内容概览
以下是本书各章节的简要概述。

#### 第 1 章
本章介绍 Objective-C 编程语言和开发环境。你还将下载并安装 Xcode，并编写你的第一个 Objective-C 程序。

#### 第 2 章
第 2 章全部关于类，这是面向对象编程的主要构建块。它涵盖了 Objective-C 为开发类提供的关键元素和独特特性。

#### 第 3 章
在第 2 章中，你学会了如何创建类；在本章中，你将学习如何使用它们。具体来说，你将学习关于 Objective-C 对象创建、初始化和消息传递的概念和细节。

#### 第 4 章
正确的内存管理是开发既正确又高效的程序的关键。在本章中，你将了解计算机内存是如何为 Objective-C 程序分配和释放的，Objective-C 的内存模型，以及如何编写正确进行内存管理的程序。

#### 第 5 章
Objective-C 包含一个预处理器，用于在编译之前对源文件进行转换。在本章中，你将学习预处理器的工作原理，以及如何在 Objective-C 源文件中使用预处理器语言。

#### 第 6 章
正如你在本书前面所了解到的，ARC 是 Objective-C 内存管理的推荐方法。在本专家章节中，你将学习围绕 ARC 内存管理的一些更细致的细节，以及如何将 ARC 用于免桥接对象。

#### 第 7 章
第 7 章开始我们对 Objective-C 运行时系统的研究，深入探索 Objective-C 语言的动态特性。这包括运行时类型确定、方法解析、对象自省、动态代码加载以及其他特性。

#### 第 8 章
在第 8 章中，你将了解 Objective-C 运行时系统的架构和设计，以及其动态特性是如何实现的。本章还展示了你的代码在编译时和程序执行期间是如何与运行时交互的。

#### 第 9 章
在本专家章节中，你将通过开发几个运用其一些关键特性和 API 的示例程序，来增强你对运行时系统的了解。
