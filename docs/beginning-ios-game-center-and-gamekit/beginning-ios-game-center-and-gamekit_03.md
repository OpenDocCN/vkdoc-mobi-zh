# 先决条件

本书假定您已具备创建 iOS、Mac 或 Apple TV 应用所需的基本技能和理解能力。同时，本书也假定您已掌握使用 `Xcode 13` 或更新版本所需的背景知识。本书不会提供关于如何定义方法和变量、安装和启动 `Xcode`，或创建及操作新项目或类的基础教程。关于这些主题，已有许多优秀的书籍，例如 Apress 出版的 *Beginning iPhone Development with Swift*。当您确信自己已准备好开始学习诸如 `Game Center` 和 `GameKit` 等更高级的框架技术时，我们假定您已基本掌握这些基础知识，能够在不查阅其他参考书的情况下顺利阅读本书。

## 本书的组织结构

当您开始阅读本书时，您会发现它由独立的章节组成。我们已尽力确保每一章都可以独立于其他章节阅读。如果您对 `Game Center` 或 `GameKit` 尚无经验，强烈建议您先阅读前两章，然后再跳读其他章节，因为这两章将为您提供在开发环境中启动并运行 `Game Center` 和 `GameKit` 的基本信息。

每一章都将配合第 1 章中引入的一个简单 iOS 游戏示例进行讲解。从头到尾跟随本书，您将亲历创建一个功能完备、集成了 Game Center 和 `GameKit` 的 iOS 游戏的全过程。此外，每一章都将为一个可重用于您所有项目的 `Game Center Manager` 类添砖加瓦。

如果您已具备 `Game Center` 和 `GameKit` 背景知识，并正在寻求特定技术的帮助，每章都将引导您学习其涵盖的技术，并提供如何将该技术应用于您软件中的示例。

## 所需的软件、材料和设备

要开发 iOS、macOS 或 Apple TV 软件——更具体地说，是基于 Game Center 和 `GameKit` 的软件——您首先需要一台运行 macOS 11 (Big Sur) 或更新版本的 Intel 或 ARM 架构 Mac 电脑。虽然您可以在较旧版本的 macOS 上进行开发，但这将无法支持最新版本的 `Xcode`。您还需要一份 `Xcode` 的副本，可以从 Mac App Store 或 `http://developer.apple.com` 免费下载。本书的目标环境是 iOS 15。

除了软件和硬件要求外，您还需要一个由 Apple 提供的开发者账户。该账户允许您在设备上构建和测试软件，并将最终产品发布到 App Store。软件开发者账户每年收费 99 美元，您可以在 `https://developer.apple.com/programs/` 购买。
