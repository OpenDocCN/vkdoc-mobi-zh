# 引言

借助 Windows 10，微软将“通用”应用程序架构的概念引入 Windows，使得使用 UWP 设计的应用能够跨多种 Windows 设备运行，例如 PC、平板电脑、智能手机、Xbox One、物联网设备、Surface Hub 和 HoloLens。

你可以使用多种技术开发通用 Windows 平台（UWP）应用，例如 C# 和 XAML、Web 技术（HTML、CSS、JavaScript 和 WinJS）、C++ 等。

本书中的示例涵盖使用 Web 开发技术（HTML、JavaScript 和 CSS）开发通用 Windows 平台（UWP）应用。一旦你掌握了 Web 开发技术，并熟练运用本书中的 UWP 应用开发示例，你将抢占先机，构建出能在几乎所有 Windows 设备上运行的精良 Windows 应用。

## 本书读者对象

本书面向具备 HTML、CSS 和 JavaScript 等 Web 开发技术基础知识，并对开发 Windows 应用感兴趣的开发者。书中采用的循序渐进方法将帮助开发者更轻松地理解相关概念。


## 你将学到什么

第 1 章：了解通用 Windows 平台（UWP）及其功能，以及如何开始开发 UWP 应用。你将学习 UWP 应用开发所需的工具，并掌握如何创建应用并在 Windows 机器上运行它。

第 2 章：学习 `WinJS` 库的基础知识。在本章中，你将探索如何在 `WinJS` 中使用命名空间、类和 promise。

第 3 章：在本章中，你将探索 `WinJS` 库中可用于构建 Windows 应用的各种控件。你将学习如何在 HTML 页面中添加 `WinJS` 控件并设置其属性。此外，你还可以了解一些常用的 `WinJS` 控件。

第 4 章：了解`列表控件`，例如 `Repeater` 控件、`FlipView`、`ListView`、`AppBar` 和 `ToolBar`。探索如何在 `ListView` 控件中筛选和分组项目。

第 5 章：在本章中，你将学习 `WinJS` 中的数据绑定以及通用 Windows 平台应用使用的导航结构。

第 6 章：通用 Windows 平台运行在多种不同屏幕尺寸的设备上，这给开发者和设计师带来了使应用适应这些场景的挑战。在本章中，你将学习如何让用户界面适配不同的屏幕。

第 7 章：了解应用程序状态和生命周期事件。同时学习如何处理应用的终止与恢复，以及应用中的状态管理。

第 8 章：你的 UWP 应用是否面向多个区域和语言？本章提供了必要的指南，帮助你为使用 `WinJS` 构建的 UWP 应用添加全球化和本地化支持。

第 9 章：本章概述了在 Windows 10 应用中存储应用数据的技术。你将学习应用设置，如何将应用设置作为应用数据进行存储和检索，以及如何处理应用数据文件夹。

第 10 章：本章涵盖如何在通用 Windows 平台应用之间共享数据。数据共享使开发者能够构建功能，允许用户从一个应用向另一个应用共享信息、复制/粘贴以及拖放内容。

第 11 章：了解 UWP 中的后台任务。探索系统事件触发器、创建和运行后台任务等主题。

第 12 章：你想在 Windows 应用中集成位置 API 吗？本章涵盖了地图和位置等主题，开发者可以利用这些功能将其集成到 UWP JavaScript 应用中。

第 13 章：在本章中，你将学习如何在通用 Windows 平台应用中集成 Microsoft Azure 移动服务。这包括介绍如何在 Azure 中创建移动服务后端，以及如何从应用连接以执行 CRUD 操作的指南。

第 14 章：本章提供了关于 Windows 10 磁贴和通知功能的深入见解。你将学习如何创建默认磁贴、自适应磁贴以及 Toast 通知。

第 15 章：在本章中，你将探索 Windows 设备中的各种设备功能和传感器，以及如何在 UWP 应用中使用它们。

第 16 章：在本章中，你将学习其他工具，例如 JavaScript 控制台窗口、DOM 资源管理器、诊断工具、Windows 10 Mobile 模拟器等。这些工具使开发者能够轻松调试和测试其 Windows 应用。

第 17 章：在本章中，你将探索如何在设备上旁加载和安装你的 Windows 应用。你将学习如何使用 Windows 应用认证工具包来测试你的应用，并生成报告以检查其是否已准备好上架 Windows 应用商店。

第 18 章：应用开发完成后，你需要将其提交到 Windows 应用商店，以便用户可以下载你的应用。在本章中，你将学习如何创建应用程序包并提交到商店。你还将学习如何通过 Windows 应用盈利。
