# 第十章：共享数据

本章涵盖如何在通用 Windows 平台应用之间共享数据。共享数据使开发人员能够构建功能，允许用户在应用之间分享信息、复制/粘贴以及拖放。例如，用户可能希望从一个应用复制一些文本或图像，然后粘贴到另一个应用中。用户也寻求在社交网站之间分享信息/链接。

Windows 10 应用以两种不同的方式支持数据共享。

*   源应用允许用户将数据从该应用共享到其他应用。
*   目标应用允许用户将一个应用声明为目标应用，该应用可以检索来自其他应用的共享数据。被声明为目标应用的应用允许用户在共享数据时将其选择为目标应用。

通过使用共享数据选项，用户可以与其他应用共享纯文本、网页链接、照片和视频。在本章中，您将学习如何实现从源应用共享数据以及声明目标应用。
