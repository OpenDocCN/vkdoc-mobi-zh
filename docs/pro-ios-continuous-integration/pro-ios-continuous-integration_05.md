## 示例应用：Github Jobs

本书的目标不是教你如何使用 `Objective-C` 和 `Xcode` 创建 iPhone 或 iPad 应用。要充分利用本书，请确保你已经熟悉 `Objective-C` 和 `Xcode`。

作为示例，我们将创建一个与 `Github jobs` API 通信的 iOS 应用。这个应用将是一个非常简单的主从（master/detail）应用。编写应用的代码不会花费太长时间，但请做好在项目配置上花费大量时间的准备。

## 先决条件
