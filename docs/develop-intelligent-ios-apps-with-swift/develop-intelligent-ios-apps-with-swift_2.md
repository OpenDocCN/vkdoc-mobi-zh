# 排版后的 Markdown 文档

2014 年，伊利亚·苏茨克维（Ilya Sutskever）提出了一种序列到序列（`Seq2Seq`）学习框架，该框架允许通过神经网络将一种文本映射到另一种文本。这一框架被证明在机器翻译中非常实用。谷歌翻译于 2016 年开始使用该框架，并用深度 LSTM 网络取代了其基于短语的翻译方式。据谷歌的杰夫·迪恩（Jeff Dean）称，此举将 `500,000 行`基于短语的机器翻译代码替换成了一个仅含 `500 行`的神经网络模型。

2018 年，预训练语言模型通过相比现有技术展现出显著改进，实现了重大飞跃。这些模型在海量无标签数据（例如维基百科文章）上进行训练。这使得模型能够学习各种词汇的用法以及语言的一般运作方式。通过使用更小的特定任务数据集，模型可以将这些知识迁移到任何具体任务中。这些模型就像“博览群书”的人一样，知识渊博，相比一无所知者能够更轻松地学习新知识。

2018 年是自然语言处理领域取得重大进步的一年，涌现了诸如 `ULMFit`、`ELMo` 和 `OpenAI transformer` 等新型预训练语言模型。

在这些模型出现之前，我们需要大量的特定任务数据来训练自然语言模型。而现在，借助这些知识渊博的模型，我们可以轻松地针对任何特定语言任务训练模型。

在最后一章中，我们将开发一个智能 iOS 应用程序，该程序能够利用 `BERT` 模型在给定的文本中找出问题的答案。

## 总结

本章介绍了机器学习、深度学习以及自然语言处理的基本概念。我们通过回顾深度学习与自然语言处理在过去几十年间的改进历程，试图理解其背后的原理。接下来的章节将更加注重实践，因为我们将在 iOS 开发中使用自然语言处理技术，并构建智能应用程序。
