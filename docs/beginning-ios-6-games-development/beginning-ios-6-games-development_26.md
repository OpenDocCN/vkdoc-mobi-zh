# 关于技术审校

![image](img/9781430244226_unFM-02.jpg)

**托尼·希勒森** 是一位移动开发者，也是 Tack Mobile 公司的联合创始人。他毕业于大使大学，获得管理信息系统学士学位。他日常工作中可能会用到 `Objective-C`、`Java`、`Ruby`、`Coffeescript`、`Javascript`、`HTML` 或 shell 脚本。托尼曾在 RailsConf、AnDevCon、360|Flex 上发表演讲，并制作了广受欢迎的 O'Reilly Android 视频教程。

在空闲时间，托尼喜欢弹奏贝斯和 Warr Guitar，并制作电子音乐。托尼与妻子洛瑞以及两个儿子泰特斯和林肯居住在科罗拉多州丹佛市郊。
