# 栈式导航

第二种导航风格是栈式或树形导航，由`UINavigationController`对象管理。你在 iOS 系统中随处可见这种风格。设置应用就是一个非常明显的例子。导航控制器的标志是其出现在屏幕顶部的导航栏（参见第 5 章中的示例）。它向用户显示他们当前所在的位置，并提供一个按钮让他们可以返回之前的位置。当内容视图控制器以模态方式呈现新的视图控制器时，导航控制器会将其添加到用户可以逐步返回的视图控制器栈中。这被称为推入视图控制器。在导航控制器的上下文中使用时，视图无需提供返回呈现视图控制器的方法，因为导航栏中的返回按钮提供了该操作。

你可以在严格限制内自定义导航栏，添加自己的标题、按钮甚至控件。导航控制器还可以在显示区域底部添加一个工具栏，你可以在其中填充按钮和指示器。这两个元素都由导航控制器拥有和管理。

*   当存在多层模态视图时，使用导航控制器可以让用户了解他们当前的位置，告知他们来自何处，并提供一致的返回方法。
