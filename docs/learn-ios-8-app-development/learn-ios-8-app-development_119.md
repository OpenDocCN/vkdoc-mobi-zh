# 总结

采用 `UIDocument` 为你的应用增加了一层现代数据存储功能，用户既欣赏这种功能，也已经习以为常。你学会了如何以及在哪里存储应用的文档。更重要的是，你理解了对象和方法各自扮演的不同角色，它们协同工作，将模型对象转换为原始数据，再转换回来。你学会了如何构建可以增量保存和延迟加载的多文件文档。在此过程中，你还学会了如何归档对象，以及如何创建可归档的对象。

你已经走了很长的路，应该对自己的 iOS 能力很有信心了。为应用添加持久化存储，实际上是你需要完成的最后一项主要的 iOS 技能。下一章将深入探讨 Swift，以磨练你的语言知识和熟练度。
