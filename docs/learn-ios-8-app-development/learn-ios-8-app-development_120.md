# 第 20 章 看 Swift，看 Swift 奔跑

正如承诺的那样，我在本书开始时并没有先讲枯燥的 Swift 课程。你直接上手开始创建应用——我认为这非常棒。Xcode 甚至能让新手程序员设计和制作出高质量的 iOS 应用，这打开了一个充满可能性的世界。但你可能不希望永远停留在新手阶段；新手在修道院的食堂里可坐不到好位置，也从来不会有粉丝来信。我并不是说读完这一章就能让你成为 Swift 大师，但这绝对能提升你的水平。

如果你对 Swift 感到吃力，或者只是相对新手，请认真阅读本章。这基本上是一个速成课程，能为你打下坚实的基础。本章并非旨在成为 Swift 的权威指南，毕竟它只是一章。苹果有两本优秀的 Swift 语言指南，我强烈建议你下载并随时查阅。这两本书都是免费的。你可以通过 Mac 或 iOS 设备上的 Apple iBooks 应用下载它们。

*   *The Swift Programming Guide*
*   *Using Swift with Cocoa and Objective-C*
