# 模态导航

模态导航是最简单的一种导航方式，也是你在本书中最常使用的。当`DrumDub`展示`MPMediaPicker`控制器以及`MyStuff`展示`UIImagePickerController`时，这些视图控制器都是以模态方式呈现的。新的视图控制器接管了设备的界面，直到其任务完成。任务完成后，它会通过代理消息将结果传达给你的控制器，然后由控制器关闭模态控制器并重新控制界面。被呈现的视图控制器负责实现一个界面，该界面在任务完成时会发出信号。

*   当你需要“跳出”当前界面以呈现相关细节或控件时，或者当你希望执行某个任务后立即将用户返回到他们之前所在的位置时，请使用模态导航。
