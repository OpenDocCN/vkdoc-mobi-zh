# 总结

给自己鼓掌吧；你在本章中涉及了大量的内容。你学习了 Xcode 的使用方法，向项目添加了资源，并使用 Interface Builder 来创建、配置和连接界面中的对象；你还学到了很多关于自动布局约束的知识。令人惊讶的是，你做所有这些事都没有编写一行计算机代码。

本章的重点并非避免编写 Swift 代码。毕竟，我们是计算机程序员。如果我们不写代码，那我们的报酬从何而来？重点是向你展示，使用 Interface Builder 和 iOS 对象，你可以添加多少功能，以及可以避免多少繁琐的细节。
