# 高级事件处理

按住 `Option` 键并再次点击。屏幕上会出现两个位置，如图 4-23 右侧所示。当你按住 `Option` 键时，模拟器可以让你测试简单的双指手势。仅使用 `Option` 键，就能测试捏合与缩放手势。同时按住 `Option` 键和 `Shift` 键则可测试双指滑动。

不过，模拟器的功能仅限于此。要测试其他任何触摸事件的组合，你必须在真实的 iOS 设备上运行你的应用。回到 Xcode，停止你的应用，并将方案更改为 iOS 设备（iPhone、iPad 或 iPod Touch——即你已连接的设备）。再次运行你的应用。

现在在 iOS 设备上尝试 Touchy 应用。尝试用两根、三根、四根甚至五根手指触摸。移动它们，抬起一根手指，再放下。这出奇地有趣。
