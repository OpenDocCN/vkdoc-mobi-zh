# 总结

在掌握 iOS 应用开发的旅程中，你已经取得了长足的进步。视图控制器是你用户界面的基石，而掌握导航则是开发应用用户体验的关键一步。

在本章中，你使用了所有主要的视图控制器类：`UIViewController`、`UITableViewController`、`UINavigationController`、`UITabBarController` 和 `UIPageViewController`。更重要的是，你学习了内容视图控制器和容器视图控制器之间的区别，以及如何使用故事板来组装和连接它们。你以编程方式创建了存储在故事板文件中的视图控制器，并以此构建了页面视图控制器的数据源。你还学习了呈现模态视图控制器的基本原理、涉及的对象以及呈现决策是如何做出的。最后，你探索了使视图适应不断变化的显示环境的所有主要技术。

这是一个巨大的成就。它非常令人兴奋，你应该与朋友们分享。下一章将向你展示如何做到这一点。
