# 致谢

> 撰写一本关于技术的书籍是一项相当耗时的工作，因为它不仅关乎将你的想法付诸笔端，还涉及编写代码并验证其完整性。我要感谢我的妻子 Kapila Sood 和我的女儿 Nitara Baruah，感谢她们在我投入无数小时来促成这本书问世的过程中给予的坚定不移的支持和耐心。她们对作为本书核心部分的应用程序美学以及内容大纲提出的反馈意见，是无比宝贵的。

> 我还要感谢我的母亲 Meenu Baruah，她在我生命的早期就发现了我的写作冲动，并一直鼓励我坚持写作。尽管我时常感到不确定，但她从未怀疑过有一天我会成为一名出版作者。她也是我们家族中第一位出版作品的作者，我确信我的写作基因遗传自她。

> 我还要感谢我已故的父亲 Sailender Nath Baruah，他在计算机尚未普及的年代引领我进入了软件世界。

> 我对我的 Apress 团队的感激之情无以言表，感谢他们为这本书的成型所提供的所有帮助：感谢 Miriam Haidara，我向她推介了这本书的第二版；感谢 Sowmya Thodur 在协调工作上的帮助；还要感谢 Massimo，他审阅了我的代码并确保其按预期工作，并在重新组织章节和内容方面提供了宝贵的建议。
> 
> — Shantanu Baruah
