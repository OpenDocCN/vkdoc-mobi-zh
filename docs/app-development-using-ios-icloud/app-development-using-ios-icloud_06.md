# 将记录保存到多个表

iOS 17 允许在私有数据库和公共数据库之间保存多条记录。在本节中，我们将学习使用 `CKModifyRecordsOperation` 和 `modifyRecordsResultBlock` 来实现这一点。

## 推送通知

发送推送通知是 iOS 开发的一个重要功能。它是应用与用户通信的方式。本书中有一节将介绍推送通知、订阅提醒以及在特定日期和时间显示提醒。在发送任何通知之前，应用必须请求允许推送通知的权限；本节将提供实现此目的的详细信息。

## 用户之间的对话

为了解释上述功能，我们将创建一个聊天应用。该聊天应用将允许用户相互通信。我们将使用 iCloud 公共数据库来存储所有聊天对话数据。
