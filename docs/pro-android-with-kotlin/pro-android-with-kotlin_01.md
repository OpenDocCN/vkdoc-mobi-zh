# 前言

《专业 Android 与 Kotlin 开发》是著名的 Apress 系列中针对 Java 平台 Android 开发书籍的继承者。随着 Kotlin 成为 Android 环境中的官方语言，与 Java 标准相比，它能编写出更优雅的程序，本书将深入探讨现代 Android 应用的高级方面。通过对 Android 系统内部重要部分和专业级 API 的详尽描述、高级用户界面主题、高级开发主题、深度通信调查、专业级硬件主题（包括手机以外的设备）、包含解决内存和性能问题指导的故障排除部分，以及应用盈利介绍，本书旨在为希望为现代 Android 设备构建最先进专业应用的开发者提供宝贵的资源。

本书并非 Kotlin 语言的入门教程。为此目的，请查阅 Kotlin 网站或任何 Kotlin 入门级书籍。然而，您将在本书中看到的是，尝试尽可能多地使用 Kotlin 的特性，以编写比 Java 更少代码的优雅稳定应用。

2021 年和 2022 年，Android 12 和 12.1 版本相继推出。在专业环境中，编写要求 API 级别达到 31 的应用并非好主意，因为截至本书撰写时，全球运行 12.x 版本的设备占比低于 10%。但您可以通过在代码中引入分支或使用特殊的兼容性类，编写同时兼容 12.x 及 6.0 至 12.1 版本的应用，从而覆盖约 95%的 Android 设备，这也是我们将在本书中采取的做法。我们仍将专注于现代 12.x 开发，但若使用低版本无法支持的新特性，我们会特别说明。

请注意，本书不重点关注 Android 6.0（对应 API 级别 23）以下的版本。如果您查阅在线教程，会发现大量针对低于 23 的 API 级别的构建。尤其是在涉及为改善向后兼容性而引入的支持库时，如果您着眼于低于 23 的 API 版本，开发会变得不必要地复杂，因为此类设备的分布如今已可忽略不计。本书将假设您对这些旧版本不感兴趣，从而在许多情况下无需研究此类支持库，并大大简化开发过程。
