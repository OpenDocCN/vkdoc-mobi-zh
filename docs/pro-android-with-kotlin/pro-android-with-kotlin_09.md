# 开发系统

在手持设备上运行的操作系统只是故事的一部分——作为开发者，您还需要一个用于创建 Android 应用的系统。后者发生在个人电脑或笔记本电脑上，您使用的软件套件是 *Android Studio*。

Android Studio 本身是您用于开发的 IDE，但当您安装和操作它时，SDK 也会随之安装，我们将在以下章节中讨论两者。我们还将讨论虚拟设备，它为在各种目标设备上测试您的应用提供了宝贵的帮助。
