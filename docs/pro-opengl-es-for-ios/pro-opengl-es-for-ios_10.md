# 第 2 章：那些数学玩意儿

任何一本关于 3D 编程的书，如果没有至少一个章节介绍 3D 变换背后的数学原理，那它就是不完整的。如果你对此毫不关心，那就跳过吧——这里没什么好看的。毕竟，OpenGL 不是会自动处理这些吗？当然会。但熟悉内部原理仍然很有帮助，哪怕只是为了理解 3D 领域的术语。

我们先定义一些术语：

- **平移**：将物体从其初始位置移动（见图 2-1 左）
- **旋转**：围绕一个中心原点旋转物体（见图 2-1 右）
- **缩放**：改变物体的大小
- **变换**：上述所有操作的总称
