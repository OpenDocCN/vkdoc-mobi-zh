## 关于作者

**Mike Smithwick** 的编程之路始于 1963 年他得到了第一台 3 位塑料 DigiComp 1 计算机 (http://en.wikipedia.org/wiki/Digi-Comp_I) 之时。在此之前不久，他就对天文馆产生了兴趣。最终，在 1980 年代，他逐渐转而编程 NASA 飞行模拟器图形。但他真正想做的是成为一名连环漫画作者（真的！）。在未能获得任何联合供稿合同后，他编写并销售了广受欢迎的 Distant Suns 天文馆程序，该程序运行于 Commodore Amiga、老式 Mac 和 Microsoft Windows 平台，同时他还兼职担任合同程序员，为 Apple、3DO、Sense-8 和 Epyx 工作。最终，他在 Live365 找到了一份“正式”工作，负责客户端的 Windows、Windows Mobile 6、TiVo、Symbian（啊……Symbian……）和 iPhone 软件。在经历了 13 个短暂的年头后，他决定重返合同编程、写作和开发的“黑暗面”，并在 Distant Suns 在 App Store 取得一定成功后，专注于 iPhone 版本的开发。有时在深夜，他仿佛能听到他那一台带有 Woz 亲笔签名的 Apple II 正在车库里呜咽着寻求关注。可以通过 `www.distantsuns.com`、AIM 上的 `lazyastronomer` 以及 Twitter 上的 `@distantsuns` 或 `@lazyastronomer` 联系他。

## 关于技术审校

**Leila Muhtasib** 自她在 MS-DOS 上编写第一个程序起，就一直对编程充满热情。此后，她毕业于马里兰大学帕克分校，获得计算机科学学位。她着迷于移动技术及其日益普及的趋势，自第一个 SDK 发布以来就一直在编写 iPhone 应用程序。她现在是思科系统公司移动开发团队的资深软件工程师和技术主管。
