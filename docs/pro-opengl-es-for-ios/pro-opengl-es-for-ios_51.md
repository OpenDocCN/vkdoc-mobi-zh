## API 统计
显示其他位置未涵盖的累计使用指标。每个 OpenGL API 调用记录的字段包括：调用次数、每次调用的总用时（微秒）、以及平均用时（微秒）。我的应用显示，在五分钟内，`glDrawArrays()` 被调用了 3,369 次，平均用时为 67,174 微秒。`EAGLContext` 中的 `PresentFrameBuffer()` 仅有 306 次调用，但总耗时惊人，平均用时高达 83,561,776 微秒。

## 总结
在本章中，你学习了使 OpenGL 应用真正具备高性能的基本技巧和最佳实践。顶点缓冲对象通过将常用几何体保留在 GPU 上，减少了总线带宽的饱和。将状态变更和 `glGet*` 调用的次数降至最低，也能显著提升渲染速度。

之后，你初步了解了在 Xcode 4 的 Instruments 中新增的 OpenGL ES 分析器；它有点像你代码的“核磁共振成像”机器，能让曾经隐藏的问题变得一目了然。

在最后一章中，你将了解一些关于 OpenGL ES 2.0 以及当下风靡一时的神秘着色器的知识。
