# 1. 引言

2016 年，全球有超过 500 万 Android 开发者。三年后的 2019 年，全球流通的 Android 设备已达 25 亿部。毫无疑问，Android 是一个被广泛采用的操作系统，并且短期内不会消失。

本书专为 Android 软件工程师（无论是新手还是老手）设计，重点不放在典型的 UI 设计上。相反，本书聚焦于 Java 和 Android 系统中那些在学习构建应用时常被遗忘的部分。它将作为一本参考指南，涵盖从设备和用户唯一标识符到 ProGuard 混淆，再到长时间运行的服务等技术——进而深入探讨一些并非一开始就显而易见的原理。

正如书名所示，本书不会涉及 Android UI 开发，也不会涵盖底层 C 语言或内核技术。相反，本书将专注于 Java 和 Android 系统中那些易于理解、实用且有趣的技术。

## 本书内容

*   一本针对非 UI 元素的 Android Java 编程参考指南。
*   本书涵盖从 Android 4.4 到 Android 11.0 的操作系统。

## 本书不涵盖的内容

*   本书不包含 Java 入门教程；如果您需要这方面的内容，市面上有许多其他优秀的资源。
*   本书不涉及任何底层 C 语言编程、内核交互或漏洞研究。
*   虽然本书可能会涉及 Android UI 的一两个方面，但这不会是重点。
